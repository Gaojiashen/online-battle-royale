## 关键约束

玩家与NPC的交互是**选择支驱动**，不是自由对话/角色扮演。

每次遭遇 = 法官给出场景描述 + 列出2-4个选项 → 玩家选择 → 法官告知结果。不开放自由输入对话内容。

**为什么：**
- 保持法官的工作量可控（不需要即兴扮演NPC对话）
- 保持所有玩家体验一致（不因法官状态不同导致NPC性格飘忽）
- 选择有明确的分支和后果，玩家能感受到自己的决策分量

## 选项差异化原则

每个选项必须导向**不同的物质结果**——不同物品、不同路径、不同风险。因为游戏没有信任度/好感度系统，纯对话风味的差异对玩家是虚假选择。

**判断标准**：「选A的玩家和选B的玩家，一小时后手里拿的东西、能去的地方、面临的风险，有任何不同吗？」没有 → 不需要选项。

**仅传递信息的节点不设假选项**：合并为单一叙事事件，标注时间消耗（如"花费0.5h"），玩家投入时间 → 获得全部信息 → 自然推进。

❌ 「选项A对话氛围不同但和选项B获得同样情报、走到同样出口」
✅ 「选项A→获得衣扣；选项B→获得草图；选项C→不搜索直接走」
✅ 无差异则：单一事件→花费时间→获得信息→一个出口

**为什么**：选项存在的唯一理由是让玩家的决策产生物质性后果。否则只是让法官多写了三份文本。

## 其他设计原则

- 玩家有身份决定开局包裹和出生区域
- 玩家间无沟通渠道
- 世界自行运转——即使无人介入，NPC也有自己的命运（可能死亡）
- 情感弧成立：普通NPC（莉莉、邓肯）展示世界残酷；伊索贝尔是"如果伸出手可以改变一些事"的机会
