## 整体结构

四阶段：日常(Day1-7) → 试探(Day8-14) → 暴露(Day15-21) → 终局(Day22-28)

## 选择支设计

每个交互节点 = 场景描述 + 2-4个选项 + 每个选项的结果。玩家选择影响后续节点的触发条件和NPC信任度。

### 关键节点
- 馅饼摊 → 建立第一印象
- 图书馆闭架区门前 → 初次信息交换
- 牛门后巷 → 信任窗口
- **地下拱顶入口（全剧情最关键分支）**：选同行→最大信任；选劝阻→她独自行动
- 海伦娜工坊相遇 → 信任确认点
- 警告信 → 决定是否继续深入
- 死鸽子事件 → 建立可求助印象
- **终局：隐秘水窖** → 看到光弦（七年前灯之飞升者的碎片形态），触碰或等待

### 三终局结果
- **存活**：玩家赴约，触碰光弦→获得核心真相，她成为后续盟友
- **死亡**：玩家未介入→讣告，可追查伤痕
- **反目**：玩家背叛过她→她不认你的声音
