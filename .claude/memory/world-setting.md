## 基本信息

- 架空城邦世界，借用爱丁堡地图
- 纪年：唐纳王朝第73年（自创凯尔特风王朝名）
- 氛围：阴冷潮湿的哥特式 + 神秘诡谲
- 地图：28区核心老城 + 城外乡村/利斯港/庄园（逐步开放）

## 权力格局

- **灯之教会** / **蛾之教会**（两个主流教会，分离独立，非敌非友）
- **断刃修会(刃)** / **熔心会(铸)** / **银棺修会(冬)** / **赤脉结社(心)**（四个地下独立教团，表面敌对暗中共存，各有公开伪装——格斗俱乐部/工匠行会/殡仪馆/娱乐社团）
- 市政厅与贵族勾结，地下势力是高层的狗
- 教会通过影响人→影响权力，不直接掌握权力
- 当局知道地下教团存在：睁一只眼闭一只眼，必要时打压
- 大学受多方势力渗透，鱼龙混杂

## 飞升体系

- 六性相：灯/蛾/刃/铸/冬/心
- 等级：0→2→4→6→8→10→12→14→15（仅偶数和15）
- Lv15需通过关键仪式
- 密传是书，等级2/4/6/8/10/12/14，2合1合成，学习后销毁
- 性相等级 * 2 = 可读密传等级上限
- 技能解锁：Lv2/Lv6/Lv10/Lv15各解锁一个技能（可被动可主动）
- 裁定依赖规则，不是叙事理由

## 玩家

- 扮演爱丁堡本地人，奉身神秘的求道者
- 开局有身份（如报社老大/酒馆服务生），决定起始包裹和出生区域
- 知道有其他玩家存在（竞速压力），但无互相沟通渠道
- 经济：金币，无生存压力
- 昼夜循环：8-20白天/20-22夜晚/22-8深夜日切

## 第一届（彩蛋）

- 第一届灯蛾成功飞升
- 灯被掩盖为书店火灾，蛾被掩盖为飞蛾骚乱
- 伊索贝尔·格雷是灯飞升的幸存者
