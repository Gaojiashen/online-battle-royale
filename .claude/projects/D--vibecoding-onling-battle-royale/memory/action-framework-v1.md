行动框架核心设计为「动词 + 目标 + 可选方式 = 裁定」的三段式结构。

**V1.1 精简（2026-06-24）：**

当前活动动词 5 个：
- 探索(L1) — 扫描区域获取线索，物品发现自动获知信息
- 移动(L1) — 需先通过探索发现路径
- 社交(L2) — 与NPC互动，结构化方式（交谈/说服/威胁/交易/招募）
- 攻击(L3) — 设计中
- 仪式(L3) — 可提升性相等级/获得物品/召唤

移除：调查（合并入探索）、传讯（玩家间不开放）
搁置：潜伏、研究

新增性相系统：灯/蛾/刃/铸/冬/心，等级2/6/10/15为技能解锁硬门槛。性相影响探索判定。仪式是提升性相的主要途径。

每轮每名玩家最多3个行动，其中最多1个L3行动。

裁定流程三阶段不变：
1. 收集与分类——按区域分组
2. 处理与裁定——四步法：合法性检查 → 难度评估 → 判定(1d10+性相修正) → 结果记录
3. 反馈与世界推进——私密反馈 + 公共通告 + 世界状态更新 + 势力推演

信息差来源：探索收获 / NPC交互 / 世界剧情 / 技能（后续展开）

详细文档见 `docs/行动框架_动词表.md` 和 `docs/行动框架_裁定流程.md`。

**Why:** 精简动词表减少了法官工作量和玩家决策噪音，性相系统将成长曲线从离散的技能树整合为一个更直觉的维度。

**How to apply:** 玩家提交行动必须遵循此格式。法官按四步法裁定。性相等级作为修正值纳入1d10判定。
