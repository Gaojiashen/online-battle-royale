这是一款受《密教模拟器（Cultist Simulator）》启发的多人在线文字沙盒游戏。

玩家以「求道者」身份进入一个共享的动态世界，通过探索、社交、势力斗争、获取秘密知识、参与仪式和争夺稀缺资源，最终尝试完成自己的飞升。

核心循环：探索世界 → 获取情报与秘密 → 发现飞升机会 → 采取行动争夺机会 → 获得知识/资源/影响力 → 推进飞升进程 → 暴露自身并引发新的风险与竞争 → 接触更深层秘密 → 重复。

胜利条件：完成飞升（而非击杀所有玩家或活到最后）。生存、战斗、资源获取、势力经营等，都只是实现飞升的手段。

玩家选择社会身份开局（如记者、医生、古董商、黑帮、流浪汉等），身份决定出生点、初始资源、NPC关系、信息渠道和专属行动——而非传统职业（战士/法师/盗贼）。

**Why:** 这是所有后续设计的根基，任何设计决策都应回扣"飞升竞赛"和"玩家间竞争"这两个核心。

**How to apply:** 在设计新系统（身份、行动、势力、飞升路线）时，先问：这个设计是否服务于飞升竞赛？是否促进玩家间竞争而非合作？
