- 追求非对称平衡而非数值公平——每个身份有独特玩法路径，社会身份代替传统职业
- 玩家天然会抱团 → 所以主动在设计层面压制过度交流
- 战斗不是目的，是解决利益冲突的一种方式
- 世界应当动态运行（势力自动推演、NPC欲望驱动），而非等待玩家触发
- 相信玩家的智慧——提供工具而非剧本，让玩家创造故事
- 规则先行，为自动化铺路——数据结构化、行动格式标准化
- 不希望玩家为了战斗而战斗，而是为了利益和飞升自然产生冲突
- 独狼玩家应有生存空间和隐性优势

**Why:** 这些偏好来自项目定位（飞升竞赛而非传统TRPG）和上一把游戏的经验教训。

**How to apply:** 做设计决策时优先考虑：是否让世界更动态？是否给独狼空间？是否朝结构化/可自动化方向前进？
