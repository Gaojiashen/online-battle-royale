# 剧情线：伊索贝尔·格雷

> **本文档为法官操作手册。**
>
> 设计原则：**链式推进，不依赖日历。**
>
> 每个交互节点自然衔接到下一个——伊索贝尔在对话中暗示她接下来会去哪里。
> 玩家决定是否跟过去。跟过去 → 推进。不跟 → 卡住，但NPC不会等你太久。
>
> 不追踪信任度、危险等级、知识进度。节点走到哪就是哪。

---

## 一、剧情链全览

```
① 图书馆偶遇
     ↓ 「修道院街的馅饼摊，下午5:00出摊，味道还不错。」
② 修道院街馅饼摊
     ↓ 「牛门那边有条干货铺后巷，明天下午2:00左右阳光照得到。」
③ 牛门后巷
     ↓ 「牛门地窖入口——我需要有人帮我记住拐了几个弯。」
④ 地下拱顶入口 ★ 分歧点
     ├─ 陪她下去 → 解锁⑤
     ├─ 不陪她下去（灯判定通过） → 解锁⑤
     └─ 【放弃】/不陪她下去（判定未通过） → 💀终结
         ↓
⑤ 海伦娜工坊
     ↓
⑥ 警告事件（自动触发）
     └─ 【放弃】不再联系 → 🔓 锁释放
         ↓
⑦ 终局准备
     ├─ 做准备赴约 → 终局（降低渡水难度）
     ├─ 不做准备赴约 → 终局（标准难度）
     └─ 未赴约 → 💀终结
         ↓
终局｜隐秘水窖
     ├─ 渡水判定 → 触碰选择 → 多终局
```

### 全局追踪

```
锁状态：[🔓无主 / 🔒锁定(玩家名) / ✅完结 / 💀终结]
NPC当前节点：[①→②→③→④→⑤→⑥→⑦→终局]
已消耗节点：[ ]
锁定玩家：[姓名]
```

### 关键原则：NPC的行动只发生一次

伊索贝尔的剧情是一条**全局共享的线性时间线**。无论有没有玩家参与、无论是哪个玩家参与的——节点一旦完成，不会重来，不会回退。

- 节点①（图书馆偶遇）完成后 → 她不会再坐在图书馆窗边等新的人来搭话。她已经迈出第一步了
- 节点④（地下拱顶）探索完后 → 她不会再回去。那条路她走过了
- 玩家A在节点④放弃 → 🔓锁释放 → 玩家B如果想接触她，她此时已经在节点⑤了。玩家B从节点⑤开始，不是从①重新开始

> **上一个玩家放弃的不是"进度重置"——是接力棒的交接。**

## 二、锁系统

### 状态定义

| 状态 | 含义 |
|:----:|:------|
| 🔓 **无主** | 无玩家在推进。任意玩家首次交互即可锁定 |
| 🔒 **锁定** | 某玩家在推进。只有该玩家可触发剧情节点 |
| ✅ **完结** | 剧情正常结束 |
| 💀 **终结** | NPC被击杀/绑架。剧情强制结束 |

### 转换规则

**首次锁定：**
```
🔓 无主 ──[在节点①与NPC交互]──→ 🔒 锁定(玩家A)
```

**后续锁定（有玩家放弃后）：**
```
🔓 无主 ──[在NPC当前节点与其交互]──→ 🔒 锁定(新玩家)
  ↑                                          │
  │                               ┌──────────┴──────────┐
  │                               │                      │
  │                         [完成终局]              [主动放弃/超时]
  │                               │                      │
  │                               ↓                      ↓
  │                             ✅ 完结                🔓 无主
  │                               │
  │                         [被击杀/绑架]
  │                               │
  └───────────────────────────────┴── 💀 终结
```

> ⚠️ 新玩家不是从节点①重新开始。NPC在哪，就从哪开始。

**放弃触发条件：**
- 节点④选「放弃」或判定未通过 → 💀终结
- 节点⑥选「不再联系」 → 🔓锁释放
- 节点⑦未赴约 → 💀终结
- 连续超过合理时间未跟进 → 法官判断后自动释放

**放弃后：**
- 什么也不会发生。她不会恨你、不会追杀你、不会写信骂你
- 世界通告不会提到你。在她看来你只是"没有再来的人"
- 其他玩家可以从NPC**当前所在节点**接手——而不是从头开始
- 新玩家接手时，与该玩家的关系从零开始（NPC不记得他/她），但知识进度已在当前节点（她不再是图书馆里那个安静的旁听生了）

### 锁定状态下其他玩家的权限

| 行为 | 锁定玩家 | 其他玩家 |
|:-----|:---------|:---------|
| 推进剧情 | ✅ 专属 | ❌ 她礼貌拒绝：「我在等一个人」 |
| 观察位置 | ✅ | ✅ 可在公共区域看到她 |
| 跟踪行踪 | ✅ | ✅ 可调查她的活动规律、会面对象 |
| 攻击/绑架 | — | ✅ 成功则💀终结，所有玩家线关闭 |

## 四、链式节点

---

### 节点①｜圣吉尔斯图书馆

| 字段 | 内容 |
|:-----|:------|
| **前置** | 无（入口节点） |
| **位置** | ③圣吉尔斯图书馆 主厅 |
| **错过** | 如果其他玩家已触发此节点——她不再在图书馆窗边。你只能从NPC当前节点（见全局追踪）接手 |
| **锁变化** | 🔓无主 → 🔒锁定(该玩家) |

**场景描述：**

> 图书馆靠窗的位置坐着一个少女。她面前摊着一本书，但没有在读——她的手指安静地搁在书页上，像在感受纸面的温度。
>
> 她的眼睛覆着一层灰白色的翳膜。她微微侧了一下头，朝向你的方向。
>
> 「你拿的那本书——第三卷左边那本——是《灯》。」她轻声说，「印刷质量不太好，但第八章关于视觉的论述很有意思。」
>
> 她说完没有等你回答，继续翻自己的书了。你甚至不能确定她是在跟你说话——还是在测试自己的判断。

**选项：**

| # | 选项 | 结果 | 玩家获得 |
|:--|:-----|:------|---------:|
| A | 在她对面坐下，翻开那本书 | 她嘴角动了一下——一个几乎看不出的微笑。「我猜对了。」然后她不再说话。你安静地看了半小时书后离开。走出门口时，听到她在身后轻声说：「你右脚的鞋跟磨损得比左脚厉害。」——她在记你的脚步声 | — |
| B | 问她：「你怎么知道是哪本书？」 | 她完整解释了判断过程。然后顿了顿——「你翻书的速度比普通人快。你不是在读书，你是在找东西。」没有追问，但你意识到她看到了比你以为更多的信息 | — |
| C | 不做反应，去别的书架 | 她没再说话。你离开时感觉到她的视线方向跟了你几秒——但她没有主动搭话 | — |

**→ 出口：**

她离开图书馆时，经过你旁边，轻声说了一句——像自言自语：

> **「修道院街的馅饼摊。下午5:00出摊，5:30就收摊了——排队的人很多，但味道确实值得等。」**

### 节点②｜修道院街馅饼摊

| 字段 | 内容 |
|:-----|:------|
| **前置** | 完成节点① |
| **位置** | ⑤修道院街 转角馅饼摊 |
| **错过** | 节点③仍可触发，但跳过一个信息获取的机会 |
| **锁变化** | 🔒 持续锁定 |

**场景描述：**

> 修道院街转角处，馅饼摊前排着几个人。一个少女站在队伍中。
>
> 她的前面是两个中年男人。这条街窄，他们的对话你听得很清楚——
>
> 「……图书馆闭架区那扇门昨晚又开了。门缝里有光。」
> 「你少管闲事。那扇门——不是给普通人进的。」
> 「我知道。但我认识一个下去过的人——他回来以后再也不提了。什么都不记得了。」
>
> 两个男人买完馅饼走了。少女站在原地，手指微微收紧——但她没有追上去。她买了自己的馅饼，转身准备离开。

**选项：**

| # | 选项 | 结果 | 玩家获得 |
|:-:|:-----|:------|:---------|
| A | 排到她身后让她感知到你在 | 她咬馅饼的动作停了一下。没回头——「你也听到了？」这句话低声但明确是对你说的。她买完后没有立刻走，在摊位旁侧了一步，等你。你们并肩离开了馅饼街。路上她主动说：「那两个男人说的闭架区——我知道是哪扇门。图书馆东翼最深处那扇。」 | — |
| B | 直接上前：「刚才那些话——」 | 她后退半步，语气变冷：「我不知道。」但脚步没动——她在等你说下去。你追问，她重复了一遍「我真的不知道」，然后转身走了。她对你没有恶意，但认定你"不够谨慎" | — |
| C | 坐在街对面远远观察 | 她买完馅饼往回走，走了几步停下来——转向你的方向，静止了大约四秒。然后继续走。她知道你在看她 | — |

**→ 出口：**

她没有再多说。走出几步，你听到她的声音落下来——很轻，像自言自语，但你确定是说给你听的：

> **「那扇门里的『光』——我见过。七年前在牛门。」**
>
> 她顿了顿，微微侧过头——
>
> **「牛门那条干货铺后巷。明天下午2:00。那堵墙后面——有东西。」**

### 节点③｜牛门后巷

| 字段 | 内容 |
|:-----|:------|
| **前置** | 已完成节点② |
| **位置** | ⑩牛门 干货铺后巷 |
| **错过** | 她独自完成探索。节点④仍可触发但跳过了一个互相了解的机会 |
| **锁变化** | 🔒 持续锁定 |

**场景描述：**

> 干货铺后面的巷子里，少女面朝一堵砖墙站着。她把手掌平贴在墙面上，闭着眼睛。
>
> 你踩到一片碎瓦——她听到了。
>
> 「我说过这堵墙后面有东西。」她没有回头。「你来确认了。」
>
> 她终于转过身来。
>
> 「你不是碰巧来这里的，对吧。」

**选项：**

| # | 选项 | 结果 | 玩家获得 |
|:-:|:-----|:------|:---------|
| A | 「你在找什么？」 | 她沉默了一会儿——「七年前这里有一家书店。后来着火了。」她顿了顿——「我能感觉到地下有什么东西。不是热的——是亮着的。像一根线，从地底穿过来了。」 | — |
| B | 「我是跟着你来的。你在查什么？」 | 「你承认得很直接。」她想了想——「我在追查一家七年前烧毁的书店。细节查清楚之前我不会告诉任何人。」但她的语气没有关上对话的门——她至少知道你是"同样在追什么的人" | — |
| C | 「这家干货铺——和书店有关系吗？」 | 「你做了功课。」她的语气温和了一点。「老板是书店店主的远亲。他关了店，但没有卖掉这间铺子。」 |  |

**→ 出口：**

> **「牛门地窖入口——你知道在哪里吗？我需要一个人帮我记住拐了几个弯。」**

### 节点④｜地下拱顶入口 ★ 分歧点

| 字段 | 内容 |
|:-----|:------|
| **前置** | 已完成节点③ |
| **位置** | ⑭牛门地窖入口（地下拱顶入口） |
| **错过** | 不去则她独自探索地底，可能在地下迷失 |
| **锁变化** | 同行→🔒锁定强化 / 望风→🔒持续锁定 / 放弃→🔓无主 |

**场景描述：**

> 你找到那个地窖入口时，她已经在台阶上了。一只手扶着潮湿的石墙，指节发白。
>
> 她听到了你的脚步声——松了一口气。
>
> 「你来了。我在想你会不会来。」
>
> 「下面就是地下拱顶。我自己下去的话——我数不清拐了几个弯。我需要一个人帮我记住。」

**选项：**

| # | 选项 | 结果 | 玩家获得 |
|:-:|:-----|:------|:---------|
| A | **「我陪你下去。」** | 你们一起进入拱顶。她靠感知引路，避开两处塌陷区，你负责记路：3右转、2左转、1下坡。但在深处通道开始分岔——迷失感笼罩下来。她停下来，手指抵着石壁：「等一下。我分不清方向了。」→ 你尝试辨明方位。**灯判定 DC 6**。**通过**：你注意到石壁上的旧凿痕在重复出现——你们在绕圈。你修正了方向。穿过通道后，在碎石封堵的墙前停下，她在碎石间找到一片深色织物碎片（边缘有焦痕），收了起来。**未通过**：在她犹豫的瞬间，头顶传来碎石松动的声音——你拉了她一把，一块砖石砸在她刚才站的位置附近。她的手臂被划伤，流了点血，但还能走。你们换了一条路，绕了远，仍然找到了那堵墙和织物碎片。**「不要告诉别人我下来过。」** | 拱顶路线图 + 织物碎片 |
| B | **「下面不安全。我在上面帮你望风。」** | 她顿了一下——「那你在上面也是一样的。」她走下去。半小时后，地下深处传来一声闷响——然后安静得不正常。她可能迷失了。你蹲在地窖口，尝试用声音为她指路。→ **灯判定 DC 11**。**通过**：你回忆她走过的路径，在关键的拐弯处大声喊出方向。几分钟后，脚步声在靠近——她爬了上来，脸色苍白，但活着。她从口袋里掏出那片深色织物碎片——和A路径找到的是同一面墙后的东西。**未通过**：你的声音在地下扩散、反射，失去了方位。地下的寂静持续。你喊了几声，没有回应。你在窖口等到天黑。她没有上来。 | 通过→织物碎片；未通过→— |
| C | **「你为什么要下去？」** | 她停下来，语速比平时快——「七年前我父母死在那家书店里。那个店主在地下做了一件事——『飞升』，有人这么叫它。我想看看下面还有什么。」——如果你说**"那我陪你下去"** → 同步A路径。如果你仍然说**"我在上面等你"** → 她走下去，同步B路径 | 同步A或B |
| **放弃** | **「我不想卷进这件事。」** | 她沉默了几秒。点了一下头——「我明白了。」她转身走下台阶。你站在那里，看着她消失在黑暗中。 | — |

**→ 出口：**

**A路径（含C→A）：**
> 你们在地面分手。她走了几步，停下来。
>
> 「如果你还想知道后续——格拉斯市场有一家叫『熔炉与月光』的工坊。我明天下午会去那里鉴定一样东西。」

**B路径·通过灯判定（含C→B·通过）：**
> 第二天你在格拉斯市场遇到她。她脸色不太好，但看到你时点了下头。她口袋里的织物碎片还在。
>
> 你们一起走向了那家工坊。

> **均→ 解锁节点⑤**

**B路径·未通过灯判定 / 放弃：**
> 世界通告：牛门地窖入口发现一名年轻女性的遗体。据调查，该名女性为爱丁堡大学旁听生伊索贝尔·格雷，死因初步判断为地下结构坍塌。官方未公布更多信息。
>
> **→ 💀终结**

### 节点⑤｜海伦娜工坊

| 字段 | 内容 |
|:-----|:------|
| **前置** | 已完成节点④ |
| **位置** | ⑧格拉斯市场「熔炉与月光」工坊 |
| **错过** | 一周内不去——她完成了鉴定，你仍可从其他渠道获取信息 |
| **锁变化** | 🔒 持续锁定 |

**场景描述：**

> 你走进工坊时，她正在和海伦娜说话。她听到你的脚步声，停了一下，转向你——
>
> 她从口袋里掏出叠好的深色织物碎片。
>
> 「这就是那堵墙后面找到的东西。你也该知道。」

**选项：**

| # | 选项 | 结果 | 玩家获得 |
|:-:|:-----|:------|:---------|
| A | 接过织物，仔细看 | 织物质地厚重，焦痕集中在边缘——**不是火灾残留，是仪式参与者袍子的碎片**。海伦娜在旁边补了一句：「这种织法——不是本地的。是熔心会的东西。」 | — |
| B | 「不担心被拿走？」 | 「我分成了两半。」她拍了拍内袋——「这一半是给她的。另一半我自己留着。」 | — |

**→ 出口（无论哪个版本）：**

你离开时，她跟了出来。

> 「接下来可能会变得危险。我收到了一条消息——有人不希望我继续查。」
>
> 她等着你做出反应。

> **「如果你家门口出现一只死鸽子——脖子系着红绳——来我家找我。」**

### 节点⑥｜警告事件

> 自动触发。玩家不需要去寻找，事件会找到你。

| 字段 | 内容 |
|:-----|:------|
| **前置** | 已完成节点⑤ |
| **锁变化** | 🔒 持续锁定。选「不再联系」→🔓无主 |

**场景描述（私信发送）：**

> 你回到住处时，发现门缝下塞着一张纸条。
>
> 没有署名。字迹歪斜，像刻意用左手写的。
>
> **「别再查那家书店了。」**

### 终局场景｜隐秘水窖

> 只有赴约的玩家能进入。
> 本场景分为两个阶段：**渡水**（到达光弦）→ **面对光弦**（决定做什么）。

---

#### Phase 1 · 渡水

**场景描述：**

> 你们穿过碎石封堵的通道，空气越来越潮湿。然后豁然开朗——一个天然的地下空间，中央汇成深不见底的水潭。
>
> 水面上方——悬着一道细长的、淡金色的光。
>
> 光弦离最近的岸约三步远，但中间的旧石阶已被水淹没，看不清落脚点。水面上漂浮着几块不稳定的石板残骸。
>
> 伊索贝尔蹲下来，伸手探了探水——「水很冷。我能感觉到水下的石阶——但不完整。」

**如果玩家准备了绳索+火把（⑦→A→②）：**
> 你把绳索固定在入口石柱上，找到对岸的固定点。有了安全绳，涉水的容错空间大了很多。
> **所有渡水选项判定 +2。**

**如果玩家准备了防水大衣（⑦→A→③）：**
> 你穿上油布大衣，把粉末塞进口袋。
> **渡水失败时，落水惩罚减半。**

**选项：**

| # | 选项 | 判定 | 通过 | 未通过 |
|:-:|:-----|:----|:-----|:-------|
| A | 踩着露出水面的石阶过去 | **灯 或 心 DC 7** | 顺利到达光弦下方 | 踩滑落水，半边身子湿透，冷得发抖。**面对光弦阶段的判定 -1** |
| B | 先扔碎石探路再走 | 无判定（耗时稳妥） | 你到达时光弦比刚才暗淡了一些。**面对光弦阶段的收益降低一个等级**（密传 Lv6→Lv4，或Lv4→情报碎片） | — |
| C | 你背她，让她感知水下给你指路 | **灯 DC 5**（协作优势） | 她精准指示每一步，快速到达。光弦未衰减，保持完整 | 你踩偏了，两人落水。她冷得发抖但没有抱怨。**面对光弦时两人都判定 -1** |

---

#### Phase 2 · 面对光弦

> 你们站在光弦下方。它悬在黑暗中，安静得像一根被遗忘的丝线。微弱的光在你们脸上晃动。
>
> 伊索贝尔伸出手——指尖距离光弦不到一寸——但她停下了。
>
> 「我等了七年。但到了眼前——我不知道碰上去会发生什么。」
>
> 她转向你。

**选项：**

| # | 选项 | 结果 | 玩家获得 |
|:-:|:-----|:------|:---------|
| A | **「碰吧。我在你旁边。」** | 她触碰光弦。指尖发光，瞬间扩散至全身——她看到了七年前的画面。光弦震颤、减弱、消散。她站在原地，泪流满面，但站着。「我看到了。那家书店——那个店主——他确实做到了。变成光的那个瞬间——他看到了什么。」 | 伊索贝尔存活，成为灯之道情报源（长期收益：后续可通过她获取灯相关密传线索） |
| B | **「别碰。让我来。」** | 你伸手触碰光弦。炽热的信息流涌入意识——那不是记忆，而是**知识**：飞升仪式的结构、灯的本质、光弦的构成方法。它们像书页一样在脑海中展开。你收回手时，指尖残留淡金色光，持续数日。掌心留下一道细长的淡色疤痕。如果渡水阶段你**落水了**：信息流因你失温而受阻，获得的知识不完整。 | **未落水→灯之密传 Lv6**（可直接学习） + 掌心疤痕（永久标记）；**落水→灯之密传 Lv4** + 掌心疤痕 |
| C | **「我们一起碰。」** | 你们同时伸出手。双倍冲击——她看到画面，你获得知识。但力量共鸣的瞬间，一道光脉从水面冲天而起，在夜空中短暂闪烁——**任何感知到灯之力的人都会注意到这个信号。** 你们必须尽快离开。 | 伊索贝尔存活 + 你获得**灯之密传 Lv4**。但你们的行动留下可追踪的痕迹（后续可能被追踪/调查） |
| D | **「等等。先观察一下。」** | 你注意到光弦在稳定衰减。你们观察了几分钟。如果决定在消散前触碰→从A/B/C中再选一次。如果等到完全消散→光弦化为碎片落入水中，水潭恢复黑暗。伊索贝尔沉默了很久——「……看来它等的不止七年。」 | 光弦消散→无收益 |

## 五、法官速查表

### 锁定状态下其他玩家可做的行动（裁定指南）

**其他玩家接近已锁定的伊索贝尔：**
> 你走近她。她侧了一下头，朝向你的方向——不是辨认你，而是确认你不是她等的那个人。
>
> 「抱歉，我现在没有时间。如果你要找书，图书馆西侧的光线更好。」
>
> 她礼貌但明确地结束了对话。但你注意到她手中的书不是图书馆的书——是她自己带来的笔记，封面上画着一只舔舐星辰的狐狸。

**其他玩家跟踪她：**
> 她离开图书馆后没有直接回家。她绕了几条路——在确认是否被跟踪。
>
> 最终她停在格拉斯市场后街的一栋公寓前。在门口站了大约十秒，听着身后的声音。
>
> 如果你保持距离：她进去了。二楼左手边的窗户亮起了灯。
> 如果你跟得太近：她停下来，缓缓转身，面朝你的方向——不说话。她等了几秒后后退着进门，锁上了锁。

**其他玩家攻击/绑架她：**
按常规裁定流程执行。如果成功→锁状态变为💀终结，所有玩家收到世界通告。

### 法官操作原则

1. **节点推进不由日历决定，由玩家行动决定**
2. **每个节点的NPC对话都包含下一地点的暗示**——玩家不需要猜"下一步去哪"
3. **如果玩家超过合理时间（3-5次交互回合）未跟进，伊索贝尔独自推进**——错过不重来
4. **锁状态决定谁能推进剧情**——锁定玩家独占，其他玩家只能观察/攻击/拦截
5. **玩家选「放弃」后锁释放**——释放后其他玩家可从NPC当前节点接手
6. **击杀/绑架导致锁终结**——所有玩家线关闭
7. **每个节点的选项都是有效的推进路径**——没有"扣分"选项，只有方向区别
