# NPC："中间人"（The Broker）

---

## 基础信息

| 字段 | 内容 |
|:-----|:------|
| **全名** | 不详。地下势力只称其为"中间人"（The Broker）。极少数与其有过直接接触的人知道——他对自己使用第三人称。从来不说"我"，只说"中间人认为"、"中间人安排了"、"中间人不问这个"。 |
| **年龄** | 不详。其外表特征在十五年的间接记录中高度不一致——身高、体型、步态、偏好的手，在不同目击者的描述中都是不同的。唯一固定不变的是他的站姿——那种将重量压在左脚上、右肩微微下沉、仿佛永远在阅读一份看不见的文件的姿势。 |
| **性相** | 不详。从未有任何已知的密传感知者能确认他的性相归属。格里格·哈尔斯特德在与他有过唯一一次间接交易后说："这个人没有心跳。"——事后解释为那句是一句修辞。他不是在修辞。 |
| **外观** | 每一次被看到的描述都不同。但有一些所有描述中一致的元素：① 他从不背对门——无论是谁的房间、谁的桌子、谁的地盘。② 他只被看到过独自一人。没有副手、没有护卫、没有中间人（这个词用在他身上有一种怪异的贴切）——但他总是能提前到达约定地点，并在所有人离开后从没有人能看到的方向离开。③ 他穿着的风格是"任何人都不会注意到的暗色、合适的质地、绝对朴素"——像一个行走的空白点。 |
| **嗓音** | 也因人而异。有人说他"声音极低，像教堂的忏悔介"，有人说"声音没有特征、听完了你描述不出这个人是老还是年轻"。唯一一致的细节：他说话时没有任何口音。一句也没有。在爱丁堡的所有人——从贵族到地下——都有某种口音或方言残留。他没有。像是一张空白的唱片。 |
| **常在区域** | 玛莉金巷(⑮) —— 十五年前这里还是瘟疫时期被遗忘的封闭街区。现在其地下深处的某个被重新装修过的石砌大厅——人称"中间厅"——是其进行敏感交易的地点。从未被标注在任何城市档案上，从未有官方人员得知其存在。 |
| **偶尔出没** | 没有任何人发现过他"偶尔出没"任何地方。他似乎不存在于"日常生活"这种时间概念之外。那些与他做过交易的人仿佛不是在找他——而是当他决定需要找某个人时，这个人就恰好会在某个完全无关的时候、在某个完全无意去的地点——遇见他。 |
| **最近行踪** | 七日之后，一场拍卖将在玛莉金巷举办。拍卖物品是E·洛克哈特的完整跨性相飞升手稿。迄今为止，所有已知的大势力——灯之教会、蛾之教会、断刃修会、熔心会、银棺修会、赤脉结社——都收到了无署名的拍卖邀请。邀请的方式是：他们各自的某个高层成员，在某天早晨发现自己口袋里的手帕、桌角的镇纸、或者睡觉前插在门后的封门木条上——多了一张折得整整齐齐的羊皮纸片。纸片上一行用手写体刻出的字（不是墨水，是某种烫压进纸面的手法）：「七日后，子夜。钱或命。」没有指明是哪个势力。每一个势力都以为是专门发给自己的。中间人没有发错。 |

## 背景：一个没有来历的人

**关于中间人——他自己给出的信息为零。他从来没有说过真名，没有透露背后的组织（如果他背后有组织），没有解释为什么他在十五年的时间里从来没有选过边。**

他从不在同一种买卖中出现两次。各类势力高层都和他交易过，但每个人只说得出他的一个侧面——而且是模糊的侧面。

他主持过最高级别的密传交易——那些不属于任何性相的可合成系统、带有跨性相特性的仪式织件，他是唯一能给出准确估价的人。

他安排过改变势力格局的中介撮合——烛光下，两个永远不可能对面谈事的人突然被安排在同一个房间里，桌上是一份互不亏的交易条。两个人都不知道对面是谁——但他们知道是中间人安排的。中间人安排的交易从不翻牌。

他经手过不可能被找到的物品——某些已经失传的密传碎片、某些被市政厅永久封禁的档案馆内秘籍、某些残留在十年前飞升现场的物品。他从哪里拿到这些物品，又是如何在不被发现的条件下运输和交付的——所有这些，无人知晓。

**十五年来，他没有被任何一个势力渗透过。没有任何一个卧底能抵达他的真实身份。所有的怀疑都建立在否定的基础上——"他不是银棺修会的人"——"他不可能只是地下势力的棋子"——"他太有秩序了，不像是断刃修会的"。**

他是爱丁堡神秘世界地表下面最深的暗流。每一个人都听说过他，极少人与他直接见过面，见过的人——都闭紧了嘴。不是被迫。是一种自发的沉默。因为第一次见他的时候，你会获得一个无法抹掉的念头：你正在隔着某种不可确认的距离观察一个"不应该属于人类范畴的东西"。

**但有一种理论——在极少数看过他站姿的人之间，从未被说出过。**

他的站姿和 E·洛克哈特完全一致。那个在灰衣修士的资料室里花了十年整理冬之飞升图纸的老学者——在他最后几年的正式立册照中（极少有人能看到，因为修会要求保密），站立的姿势和这个被称为"中间人"的人一模一样。不是相似——是完全的一回事。膝盖的角度、脊椎的微妙前倾、那双隐藏在不合季节的长袖下面的手指微微握起的样子。

洛克哈特死了十年。死于银棺修会殡仪馆登记的日期是唐纳63年。尸体被处理过了——乔纳斯·惠特菲尔德是最后一批看到他尸体的人。但尸体是完整的吗？一份没有死因的死亡记录，加上一个与死者在静态站立时完全重合的拍卖中介。

**可能性的列表不长：**

1. **洛克哈特假死。** 那个老人预判了莫蒂默·阿什的清洗，伪造了自己的死亡，在阴影里重新建立了一个比银棺修会更高阶的操作体系——中间人就是洛克哈特。这种解释最大的漏洞是：洛克哈特死前已年逾古稀，而中间人在各种间接能量痕迹中被描述为"有精力的"、"没有老龄特征"。但如果跨性相理论确实包含某种允许转接的手段——

2. **63年双飞升的参与者。** 唐纳63年那次飞升事件中，不仅灯之飞升成功——同时还有蛾之飞升。如果中间人是当时的蛾之飞升者——他不是在"服务"神秘市场，他是在用他的存在本身调节整个市场的节律。蛾之飞升的本质是"变得不可追踪"，这与中间人的特征几乎完美吻合。但有一个问题：中间人的行事风格不是蛾的。蛾的变化是侧面的、是反向的、是诱人的——中间人的风格是：绝对方正。他的交易从不隐藏条款，他的布置精密如钟表。那是灯的特征。

3. **唐纳王室密使。** 唐纳家族从来没有在六性相之间公开选边。但七十三年的统治，如果他们还在——还没有被任何一个性相渗透到内部——那他们要么什么都不知道，要么知道一切。有一个推断：王室掌握着所有性相交织点的通行权——通过一个永恒的"中间人"角色保持各方在自己轨道上的均衡。如果是这样，中间人不属于六个性相中的任何一方——他是第七个位置：平衡本身。

4. **完全陌生的第三方。** 一个爱丁堡之外的存在。如果地图并不是全部——如果利斯港之外的海洋确实通往某个地方——洛克哈特的跨性相论著，会不会同时是某种"跨边界"理论的副产品？中间人是来找回这份论著的——在它把一个本不应被打开的边界打开之前。

**无论哪种可能是真的——中间人已经启动了他十五年里最大的一笔生意：将洛克哈特的手稿暴露在全城所有势力面前。七日之后，爱丁堡的地下将会有一场所有势力同时在场的拍卖会。**

拍卖的标的不是金币。是"代价"。每一个势力都必须拿出某种等同于洛克哈特知识的价值的牺牲品——他知道他们会拿什么。他始终知道。

## 表层身份

**无。**

中间人没有表层身份。他不隐藏——他就是隐藏本身。他不扮成普通商人、普通市民、普通匿名卖家。他的身份是"这个人不存在的"。如果你问他，他会回答："中间人不认识那个人。"

---

## 真实欲望

**未知。**

以下四个推测——每一个都可能是真的。也许不止一个。也许全部不是。

- **如果是洛克哈特本人：** 他十年沉默后举办这场拍卖——不是在卖研究成果。是在测试每一个势力：谁是十年前杀他的人？谁会在拍卖中出价的方式暴露自己的过去？拍卖是一场审判。
- **如果是蛾之飞升者：** 他十五年没有暴露过自己。为什么选择在这个特定的节点亮出底牌？也许蛾的飞升有一个时间限制——他已经没有下一个十五年。也许这场拍卖的"代价"中有一件是他需要的：飞升的最后一步材料。
- **如果是王室代理：** 这场拍卖可能不是手稿——而是**诱饵**。将各个势力拉到同一个房间里，让他们以手稿竞价的方式互相暴露手中资源。王室需要一个理由来重新调控平衡。手稿是一个完美的借口。
- **如果是外部方：** 爱丁堡处于被封锁状态——但如果这个封锁不是为了防止人离开，而是为了防止某个东西进来？手稿中可能包含了打开边界的指令。中间人拿走它——是取回某种不应该被留在封锁侧的钥匙。

**他问过的问题——如果他会这样问——可能是这三个：**

1. **这些势力准备好面对他们竞价的东西了吗？** — 洛克哈特的手稿不是收藏品。它是改变游戏规则的武器。谁买了它——谁就会在某个非常近的未来面对选择。中间人不需要问他们是否准备好了——他会在拍卖会上看着他们签下代价。
2. **哪支力量会第一个后悔？** — 他的拍卖会从未有任何人能够撤回出价。不是因为他们不敢，是因为他自己从来不让人在交出代价后还留在同一个层级上。他不知道"后悔"是被允许的。
3. **站在中间的人，还能站多久？** — 如果他是洛克哈特——十年已经很长了。如果他是蛾之飞升者——十五年。如果他什么都不是——十五年的平衡快要自我崩塌了。有人在等他的失误。他没有给他们机会——所以这整个拍卖，可能不是为了卖出手稿。而是为了让那些等他失误的人——暴露在失误发生的同一时刻。

## 可提供的资源

| 类型 | 具体 | 获取条件 |
|:-----|:-----|:---------|
| **洛克哈特手稿（拍卖品）** | 完整的跨性相飞升论著——打通六性相过渡屏障的原始密钥。如果被所学——持有者可同时走在多条性相路径上 | 在拍卖中出价。出价的形式不是金币。"代价"必须由中间人认定"等同于手稿的价值"。什么样的代价能够得到这个认定——未知。 |
| **跨性相仪式中介** | 他可以安排不同性相的求道者在绝对匿名的条件下交换仪式材料、密传副本、或合作完成需要多种性相的共同仪式 | 找到他 + 他有兴趣。他不对所有要求响应——他主动挑人。 |
| **不可获取的密传碎片** | 某些已经失传的密传片段、被官方禁封的档案、或飞升现场的残余记录——他知道从哪里获得并交付给你 | 这笔交易本身要求等价的代价。有些代价是物质，有些是情报，有些是——你的未来。 |
| **势力间桥梁** | 如果你需要一个方法让两个永远不会碰面的势力在匿名和不暴露弱点的前提下完成交易——他可以用自己的信誉为桥梁 | 你是两个潜在交易方的其中之一。并且交易的性质符合他最近在调节的宏大局面。 |
| **身份保护** | 一旦和他达成正式的交易——该交易的存在性本身在爱丁堡的地下世界中将是被保护的信息。有他的章印的交易，没有第三方能追到源点。 | 交易已经完成。他是先做成再保护——不是先签合同。 |
| **答案——关于洛克哈特的死** | 如果他是洛克哈特——他就在你面前。如果他不是——他有答案。但目前——没有任何人能在他的交易条件之外从他嘴里获得任何信息 | 参加拍卖会。然后让他对追问他身份的人说一个字：「请。」——然后他消失。 |

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 不存在"对外姿态"。因为他自己不作为一个普通的社交实体存在。他不是人——他是一个函数。你输入问题，他输出交易条件。你没有输入——他不出现。 |
| **真实一面** | 如果这十五年的记录中有任何一致性可以读出他的"性格"——那就是一种近乎极端的、不偏不倚的精确。他从不迟到。从不改价。从不食言。从不解释。那些与他打交道的人说：在和他做交易的过程中你不会感觉你在和一个人谈判——你感觉你看着某种节律，某种比判断更古老、比隐藏更深的东西，在一层一层的揭开必要的环节。他不是冷。他是——精密的。 |
| **弱点** | 如果有——没有人发现过。可能的线索：① 他的站姿——如果他不是洛克哈特，为什么模仿他的姿势？如果是——为什么要让这个姿势这么容易被识别？故意的？还是无意暴露？② 他从来没有在同一时间出现在两个地方——推论：他只是一个人，不是全知的幻影。③ 如果他需要拍卖——那说明他在找某样他自己无法直接拿到的东西。在他的绝对形式之中，存在一个"需要出价"的裂缝。 |
| **道德立场** | 无法按人类逻辑判断。他不对任何人负责。他不保护"好人的利益"或"坏人的安全"——他维持棋盘。不是某一边胜出，是棋盘自己不被掀翻。如果有人威胁到了棋盘的稳定性——他会出现。如果有人威胁到了人类——那是人类自己的事。 |
| **可被利用的方式** | 没有任何已知的方式可以"利用"中间人。他不是被动的。你永远在被动位——你永远要等他来敲门。但有一件事是可以做的：如果你能判断出他拍卖的最终目的——不是在拍手稿，而是在用拍卖过程引出一个特定的回应——那么，向他报价的人中，有一个他正在等待的出价。如果你就是那个人——或你知道那个人是谁。 |

## 对应剧情线的可能方向

```
主线线索：
  七日后的拍卖会——洛克哈特手稿落入谁手？以及，中间人到底是谁？

核心谜题线（身份的四种可能）：
  洛克哈特假死 / 蛾之飞升者 / 唐纳王室密使 / 外来的第三方。
  玩家在不同NPC和区域中获得的碎片会逐渐收窄这四种可能性。
  最终的揭示取决于哪条主线被玩家推进最远。

支线线索A（拍卖准备）：
  七个势力收到了同样的拍卖邀请。他们各自在准备出价。
  他们的"代价"是什么？
  - 灯之教会：将图书馆封存的"被禁借阅记录"原版——也就是埃德蒙暗格的上级原物。
  - 蛾之教会：向中间人交出索恩的代码本——他们最高级情报的加密方式。
  - 断刃修会：将前成员——罗文·刀——作为"活的代价"交付。
  - 熔心会：艾拉拉的"神的容器"原型图纸——她从开始到现在所有的记录。
  - 银棺修会：洛克哈特死因的真相——莫蒂默·阿什将被迫在拍卖中当众承认。
  - 赤脉结社：格里格·哈尔斯特德——他们知道他是独立激活心之道的人；他们准备"让他入会"。
  - 唐纳王室（如果他们就是拍卖组织者）：无出价。他们只是在看。

支线线索B（中间人的痕迹链）：
  格里格手里有一次间接交易记录——十八年前——和一个"没有心跳"的人。
  乔纳斯在暗格交叉比对中发现了四个死于非命的求道者——在死前都见过同一个无法确认身份的人。
  埃德蒙在借阅记录中发现一本被借出、被焚毁、被重新放回架上的书的借卡上——
  签名的人已经死了十年。笔迹不是死去的人的。
  海伦娜在地下工坊收到过一块"被从内部熔穿"的石砖——是她父亲最后那件作品的残留物。
  石砖的背面有一张贴在剥离前的封蜡——蜡印是一朵没有开放的花苞。
  没有人认得那个符号。但所有人在不同时间、从不同方向、在不同交易中被同一个人交过手。

支线线索C（拍卖会夜）：
  七日后——玛莉金巷地下。
  所有势力在同一时间出现在同一个暗厅。
  中间人站在所有人面前，洛克哈特的姿势，一样的话术。
  然后他说：
  「中间人收到的一切出价——如果通过——将成为不可撤回的代价。」
  
  然后第一个出价的人可能是任何势力。也可能是玩家。
  
  拍卖会的结尾不是一个胜利者——而是一个选择：
  谁在出价的时候暴露了自己真正在保护的东西？

支线线索D（如果他是洛克哈特）：
  西尔维娅十年逃亡的终站。
  如果她站在中间人面前——发现洛克哈特没有死。
  如果他就是在等她出现——整个拍卖会的目的就是一个给她的信物。
  她不一定要原谅。但那一刻——她会知道：这个男人给自己留下了全城所有势力都没能打破的隐匿——
  然后用自己的隐匿换了一场所有人的暴露。
  这是不是飞升？——不是六性相中的任何一种。是所有路径的交汇点。
```

| **西尔维娅·韦恩**（冬） | 如果他是洛克哈特——她是他的学徒、他的遗稿抄写员、他死前最后托付的人。如果他是蛾之飞升者——他在借用洛克哈特的手稿作为交易筹码。如果是王室——他知道西尔维娅逃亡的每一站。她是他暗下的一把棋。无论哪一种，他与西尔维娅之间的联系不是信息的，是实质性的——她用了十年在阅读那个老人的思想，而他就是那个思想的现实对应物。如果相遇——她会回答他留在封蜡信封里的问题。或者她会问一个新的。 | 拍卖会。或——西尔维娅某一天发现自己不需要追查了。因为他已经在她面前。 |
| **乔纳斯·惠特菲尔德**（冬） | 乔纳斯是最后一批在洛克哈特死前接触他尸体的人。如果中间人是洛克哈特——那乔纳斯的暗格就是整个故事中唯一一个可以被实证的核心证据。那个空白的死因栏——不是掩盖。是在等待填上新的真相。乔纳斯如果得知中间人站立的姿态和洛克哈特一致——他的三十年殓师直觉不会让他问"是不是他"。他会问："那个我填的空白——他看到了吗？" | 当乔纳斯的暗格被打开——里面的交叉比对数据中有一次殡仪馆夜间保安记录：唐纳63年某月夜，一个"自称家属"的人被允许进入了遗体室。那个"家属"的描述和中间人高度一致。 |
| **格里格·哈尔斯特德**（心） | 格里格是极少数通过"间接心跳"感知过中间人的人。他的那句"这个人没有心跳"是所有中间人描述中最关键的一条。如果他能在拍卖会上当面见到中间人——他三十年来问过的三万个问题中最重要一个会在他站到中间人面前时说出口。不是用嘴。是用心跳。"听"一个人的完整节奏只需要几秒钟。而他对中间人的那几秒钟——将会决定他这辈子最后一笔大交易的方向。 | 拍卖会。或——中间人某一天会来当铺。因为他知道格里格能听。他也许想让格里格听——某个在格里格的收集中一直在沉默的人。 |
| **所有其他 NPC 高层联系人** | 莫蒂默·阿什、索恩、伦威克、熔心会长老阿格尼丝、赤脉结社的社主任——每一个势力的高层都有某种程度上的"以中间人为中介"的经历。他们是棋盘上的棋。中间人是棋手的某个版本。 | 都在拍卖会上同场出现——有些人在此前可能从未在同一间房里见过彼此。 |
| **E·洛克哈特**（死者——或不是） | 洛克哈特不是这个游戏的 NPC。他要在游戏开始时已经死了十年。但如果中间人是洛克哈特——那么全城所有和他有过接触的人——西尔维娅、乔纳斯、莫蒂默、埃德蒙——过去十年的所有行为，都在这一个人物的阴影下运行。 | 最终的揭示——不在这些角色卡的任何一个描述中。在玩家发现那个站姿之谜的时刻。 |
