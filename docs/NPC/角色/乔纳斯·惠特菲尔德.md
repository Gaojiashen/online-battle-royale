## 背景：三十年的守门人

**乔纳斯·惠特菲尔德是银棺修会最不像成员的核心成员。**

他是银棺修会殡仪馆的主任殓师——这个职位意味着他掌管着全城最完整的死亡记录。三十年的任职。送走了三任主教、两位唐纳王室远亲、不计其数的普通人和不那么普通的人。他把每一具尸体当做文本，每一份死亡记录当做编目。他从不表达立场。

**但十年前——唐纳63年，洛克哈特死的那一天晚上——他做了他的第一次选择。**

莫蒂默·阿什——银棺修会时任首席长老——在深夜敲了他办公室的门。亲手交给他一份填写完整的死亡登记表。表上所有的信息都已经填好了：姓名、日期、送葬流程。**死者是 E·洛克哈特。** 死因栏是空白的。莫蒂默的声音很平静：「登记——不要填死因。如果有人问——说自然死亡。如果有人坚持要看——让他们来办公室找我。」

乔纳斯接过登记表。他看着死因栏的那个空白。一个殓师——一个三十年的殓师——填一份死亡登记表时不填死因，等于不写日期。那是记录者最大的沉默。

**他填了。他把那份不完整的死亡证明归档在修会档案室的深处。** 然后他做了一件修会没有要求的事：他单独开了一个暗格，把洛克哈特死亡当天殡仪馆的全部出入记录、访客名单、和所有能够交叉锁定那晚某人行踪的内部日志——复制了一份藏在那个暗格里。他没有告诉任何人。包括莫蒂默。

他不知道为什么。也许是因为那个空白的死因栏：你可以让一个殓师说谎，但你不能让一个殓师在三十年的记录上留一个空白而不给他一个解释。

**从那以后，他在做一种微小的、缓慢的"纠正"。**

他匿名向西尔维娅·韦恩发出过警告——几周前，通过一个不记名的口袋信。他不能告诉她更多——银棺修会的情报系统比他自己的神经更密集——但他可以让她知道：有人在查她。

他定期和埃德蒙·里德通信——表面上是交换"历史埋葬记录"，实际上是核对那些被官方掩盖过的死亡和飞升之间可能存在的关联。

**马库斯潜入档案室那天晚上——他看到了。** 他从二楼窗户看到马库斯从地下室后门出来。他第二天去检查了地下室的锁——完好无损。但他记住了马库斯的脸。后来的某个晚上，当马库斯在节点⑤再次接近档案室时——他发现地下室的门没锁。不是疏忽。是乔纳斯特意在那一夜提前去把门栓抽开了。

他没锁。他已经等了十年。他在等人来找洛克哈特的记录。

## 表层身份

**殡仪馆主任殓师。** 一个老资格的银棺修会成员，从无到有地管理着修会在灰衣修士墓园的全部殡葬业务。职业性的沉默寡言，被同事们视为"老乔纳斯的职业习惯"。没人问他在想什么。三十年了，也没人怀疑过他在想。

他的办公室里有一面墙的档案柜——按照日期排序的、全城所有经银棺修会处理的死亡记录。加上一份只有"他知道的"暗格扩充。

---

## 真实欲望

**补全洛克哈特的死因——以及所有被空白的死因栏上没有被记录的东西。**

他不是为了正义。一个殓师的正义不是给死者报仇——而是给死者正确的记录。洛克哈特的死因栏不是"被漏填"，而是"被要求空着"。那个空白本身就是证据——而他想在上面填上正确的字。

他想知道的三个问题：

1. **莫蒂默·阿什在掩盖什么？** — 这个人在银棺修会的层级里有多高？他的指令是自己的决定，还是上层的？如果是上层的——那死的不是一个退隐研究家。死的是一个被整个冬之教团高层一致决定要让他"死得不太像自然"的人。
2. **西尔维娅会来找他吗？** — 他发出了警告。他没有署名。十年——她曾经是他档案室里的一个新职员，他记得她的字迹（娟秀、左撇子、写完会习惯性地在逗号后面不起笔抬而多停一下）。他不知道她是否还活着。但如果她来找他——他的暗格里有她要的东西。
3. **马库斯在找什么？** — 那个潜入档案室的军人——他在翻什么？乔纳斯后来查过他碰过的文件夹编号。那些编号对应的不是洛克哈特，是另一组死亡记录——日期很分散，但死者都是刃之道相关的人。马库斯在找的不是手记作者，而是……刃之道曾死去的求道者名单？还是某个特定的名字？

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 沉默、职业性、不可接近。他的回答永远是"是的"、"不是"、"这不符合程序"和"请您在表格上签名"。他用程序当盾——程序是最好的借口，尤其是当你不打算执行程序的时候。 |
| **真实一面** | 在专业性的面具后面，是一个用记录的方式背叛了修会、也背叛了自己沉默原则的老人。他不是一个演员——他的沉默是真的，不是表演。但他心底有一个殓师的顽固信仰：每一具尸体都应该有原因。每一段死亡都应该有结论。洛克哈特的死亡没有——而这件事在他内部像一根钉子，已经扎进去十年了。 |
| **弱点** | ① 对"空白"的执念——任何被故意隐瞒的死亡信息都能触发他某种近乎仪式性的需要去补全它。② 他的年龄——六十三岁了，双手在渐渐不那么稳。再过几年他可能再也做不了遗体处理。他想在退下来之前把那个空白的死因栏填掉。③ 莫蒂默的人如果发现他的暗格——他会毫不犹豫地毁掉那些资料来保护自己，但毁掉的代价是在他内心留下一个他永远填不上的洞。 |
| **道德立场** | 记录高于个人、事实高于教团。他认为殓师的职业誓言和求道者的教团忠诚不存在冲突——因为在他看来，冬之道本身是"对终结的忠实记录"。粉饰死亡记录是同时对职业和性相的背叛。 |
| **可被利用的方式** | ① 以任何方式让他相信你在追查洛克哈特的死因——他可能会主动提示比你需要的更多的方向 ② 如果你能告诉他西尔维娅还活着——他会沉默，但眼眶下面会有一点只有细心人才能注意到的收缩 ③ 威胁暴露他的暗格或他与埃德蒙的秘密通信 |

## 对应剧情线的可能方向

```
主线线索：
  洛克哈特的死因——那个空白的死因栏后面是什么？

支线线索A（莫蒂默·阿什的追踪）：
  莫蒂默·阿什是银棺修会的首席长老。乔纳斯已经确认是他亲手交上了那份不完整的死亡登记表。
  莫蒂默是个人行动，还是在执行修会上层的指令？
  如果是上层——那银棺修会的飞升计划可能远比外界所知的更进取。
  他们可能在十年前就已经掌握了某种冬之飞升的核心条件，
  而洛克哈特的存在——一个独立研究冬之飞升、不属于任何教团的老人——
  威胁到了他们对"谁第一个飞升"的控制权。

支线线索B（暗格中的交叉记录）：
  乔纳斯的暗格里不只是洛克哈特一个人的资料。
  他将三十年里所有"死因存疑"的案例交叉汇总——
  发现一批死者在死前几个月内都借阅过同一批书，
  或者在同一个区域频繁出现。
  这些人中，有四个人在死前最后一周见过同一个人。
  那个人不是莫蒂默。是一个乔纳斯身份不明的访客。
  如果这张脸能和埃德蒙·里德或马库斯的信息关联——

支线线索C（给西尔维娅的第二封信）：
  他发了第一封警告信。如果西尔维娅没有收到——或者收到了但没有相信——
  他会不会发出第二封？第二封不再是匿名的。
  那意味着他将暴露自己。
  他不怕暴露——他怕的是暴露之后，他还没有把暗格里的资料交到一个能继续这件事的人手里。

支线线索D（马库斯在找的名字）：
  马库斯潜入档案室翻的不是洛克哈特——是刃之道的死亡记录。
  他在查的某个名字——可能和一个十一年前被清剿的教团据点有关。
  乔纳斯注意到这一点。如果这两个发现被串接起来——
  马库斯找的人和洛克哈特的死，可能隔着某种间接的线。
```
