# NPC："冷酷"诺伍德（Grim Norwood）

---

## 基础信息

| 字段 | 内容 |
|:-----|:------|
| **全名** | 不详。"诺伍德"是地下势力给他起的姓——来源于他二十年前第一次出现在牛门地下时身上那件带着诺伍德区旧标记的囚服外套。"冷酷"（Grim）是后来加上去的——因为从来没有人见过他笑。他自己接受了这两个名字，用它们在交易和威胁上签名。 |
| **年龄** | 44 岁（估计——他的"入行记录"最早出现在二十年前的牛门，当时的描述是"二十出头的健壮男性"） |
| **性相** | 普通（无性相） |
| **外观** | 矮壮，像一台被压缩过的格斗机器——肩膀极宽，臂长和身高不成比例。头顶剃光——不是秃，是刮的。下颌骨宽大，鼻子断过至少两次，复原得不算好也不算坏——像一个被修补过的旧家具，能用，但能看出历史的痕迹。额头上有一道从发际线划到左眉的旧疤——据说是一把匕首的刀背。他穿暗色厚布衣——不穿外套，不管什么天气都让手臂外露——手臂上布满了格斗留下的陈年疤痕，像一圈一圈的年轮。 |
| **嗓音** | 低沉、慢、每个词之间的停顿大约是普通说话者的两倍。那不是迟钝——是在给你时间考虑。他说"不"的时候，那个字的尾巴会往下降——不是愤怒，是终结。 |
| **常在区域** | ⑩牛门 — "地下铁笼"（Iron Cage Below）——一个在牛门最底层的坑道里的格斗场。没有窗户，没有正规设施，只有一圈铁笼、几盏煤油灯、和一地的血迹——旧的盖新的，层层叠叠。 |
| **偶尔出没** | ⑭地下拱顶（和格里格谈生意——两人关系复杂）、⑮玛莉金巷（去中间人的拍卖区域做"安保评估"——他会在那里站半个小时，不说话，然后离开） |
| **最近行踪** | 蛾之教会的人找了他两次——邀请他为某场"私人活动"提供外围安保。他没有答应，也没有拒绝。他在等第三次数目。 |

## 背景：铁笼里的国王

二十年前，一个穿着囚服外套的年轻人从诺伍德区——爱丁堡城外的某个惩戒殖民点——逃出来，身上只有一只鞋、三根手指的冻伤、和一双"不在乎你是不是疼"的眼神。他在牛门的最深处找到了一个最肮脏的格斗坑——那种连格斗手都不想去的地方，因为太底层、太危险、太没有回报。

他在那个坑里打了七年。没输过一场。不是因为他最强——是因为他"最不在乎被打"。对手打他一下，他会继续往前走。对手怕了。所有人都会怕一个不躲的人。

**现在，那个坑是他的。** 他用了十三年的时间——从格斗手变成老板，从老板变成"牛门地下没有人能在不和他打招呼的情况下进入格斗业"的掌控者。他的格斗场——"地下铁笼"——是整个爱丁堡最硬核的地下搏击场。贵族少爷来寻刺激，地下势力来处理私人恩怨，普通市民来赌、来输、来遗忘自己的房租和老婆和明天的太阳。

**罗文·刀是他的冠军。** 不是"他雇的冠军"——罗文不在任何人的工资单上。但罗文偶尔会来打——当诺伍德有一个"对手不太好处理"的时候。诺伍德不付罗文酬金——他付"东西"：一个安全的藏身处、一个不会被城防军注意到的身份、和一个沉默的约定——"你在我的笼子里是被保护的，出了笼子是你自己的事"。罗文是诺伍德那套字典里唯一一个"不需要谈价格"的人。

**格里格·哈尔斯特德是他的合作伙伴——和不完全信任的势力对手。** 诺伍德的格斗场的门票收入——特别是那些贵族私局的门票——一部分通过格里格的当铺系统洗进合法经济。格里格收取合理的佣金。两个人互相不喜欢——不是敌意，是一种"两个都活了太久的地下生物之间的互相嗅"。诺伍德不信任格里格的不设防和"不介入"原则；格里格不信任诺伍德的"用拳头说话的哲学"。但他们不需要信任——交易本身是干净的。

**马库斯·布莱克伍德和他是同行——但不是朋友。** 马库斯有黑铁俱乐部，诺伍德有地下铁笼。一个在西弓街，一个在牛门。两个格斗场之间有一条不成文的边界线——谁的客人归谁，谁的地盘谁说话。两人偶尔会在格斗手问题上起摩擦——马库斯有一个格斗手去年跳过合约直接来了地下铁笼，诺伍德收了。马库斯没有追究——不是因为懦弱，是因为他需要诺伍德的人在某一笔生意上别碍事。

**蛾之教会的"安保提议"是最近的事。** 他们的条件很诱人——一笔他一年收入总和的钱，只需要在某个晚上带着他的人守在某个门口。诺伍德不是笨蛋——他知道太好的价钱通常意味着"你不需要知道但你会被牵扯进去"。他在拖时间。他在观察蛾之教会找的其他人——如果他的对手们都在签，他会签。如果有人在拒绝——他会想知道是谁。

## 表层身份

**格斗场老板。** 地下铁笼的经营者、"诺伍德的人"的保护者、一个在牛门的每一个酒馆里都被当成"敬酒话题"的人。他经营的是合法的地下产业——不是暴力的施暴者，是暴力的舞台管理者。

---

## 真实欲望

**安静地赚钱，然后安静地离开。如果可能——在离开之前，知道那些"地上面的人"是不是真的比他更值钱。**

他没有哲学上的追求。他没有飞升的野心。他在这座城市的底层活了二十年——他看到的每一个"更高的目标"都建立在某些人的尸体上。他不想加入那堆尸体——也不想踩上去。他想在这座城市的齿轮之间找到属于他的那一条缝——在齿轮转得太紧之前从缝里退出去。

他想知道的三个问题：

1. **蛾之教会到底在拍卖什么——以及为什么需要他的"安保"？** — 如果只是一场地下物品交易——他们有自己的人。如果需要他的格斗手来守门——那说明他们预期会有"不止口头上的冲突"。他需要知道冲突等级再决定要不要接。
2. **格里格能不能帮他洗掉他全部积蓄？** — 他在准备离开了。不是今年——但两年内。他的积蓄目前分散在几个不同的渠道里。如果能一次性全部变成"安全的货币"——他可以在某个阴天的凌晨离开，无人知晓。格里格能不能做到？价格是多少？
3. **罗文·刀还能打多久？** — 不是因为他的格斗场需要她——是因为如果她垮了，他会失去"地下铁笼"里唯一一个让他觉得"这个人不会在背后捅我"的人。不是感情。是稀有。

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 沉默、冷硬、不废话。他不是装酷——是真的认为说话浪费能量。如果有人在他面前吹牛，他不会打断——他会用沉默让对方尴尬到闭嘴。 |
| **真实一面** | 一个花了二十年在地下建造了一套自己规则的逃亡者。他不是残忍——他是务实的。如果有人在他的笼子里犯规——他会用犯规者示范规则。不是愤怒，是秩序。他的秩序是铁笼的秩序——简单、暴烈、不可谈判。 |
| **弱点** | ① 他在准备离开——如果他的积蓄或退出计划受到威胁，他会被迫做出不寻常的激进反应。② 罗文——如果她在他的地盘外遇到麻烦，他不确定自己会不会出手。他讨厌这种不确定。③ 他对"不可预测"的容忍度极低——如果有人打破了他的铁笼逻辑，他会用不符合自己长期利益的方式做出反应。 |
| **道德立场** | 不是善良，不是邪恶——是他的法则：在他的铁笼里，按他的规矩。在铁笼外，各自求生。他不欺负弱者（不值得），不畏惧强者（不划算）。如果一个人掉进他的笼子里——他会给对方一个公平的机会。在格斗场上，公平是他唯一相信的神。 |
| **可被利用的方式** | ① 让他打一场——如果他输了（极少发生），他会尊重那个击败他的人并给出更多信息。② 给他一条"离开爱丁堡的通道"——如果这条路是真的，他会付任何价格。③ 威胁罗文的安全——但做好心理准备，这是将他变成敌人的最快方式。 |

## 触发事件

**事件一：笼中之择**

玩家因为需要钱、需要情报、或需要"地下势力的尊重"进入地下铁笼。诺伍德在入场口看了你一眼——那个眼神的意思是"你不是这里的常客，但你是自己走来的，所以给你一个机会"。

格斗对手的选择是一个节点：诺伍德给你两个名字——一个是随机的地下格斗手（实力中等，奖励标准），另一个是"一个你认识的人"。那个"认识的人"可能是马库斯的某个手下、罗文（如果她愿意）、或者某个与你在其他剧情中有过冲突的角色。

如果你选择和认识的人打——输赢的后果都会影响你与该角色、以及与诺伍德之间的关系。如果你赢了——诺伍德会给你更多信息（关于拍卖的安保、关于罗文、关于蛾之教会最近的招募）。如果你输了——你仍然是"进过铁笼的人"，在地下势力中获得基本的通行权。

**事件二：拍卖会的安保工作**

如果玩家在地下势力的可信度达到一定水平——或在诺伍德面前证明了自己的战斗能力——诺伍德会在蛾之教会第三次来谈条件时，带你作为"外围人员"一起参加安保部署会议。

会议在牛门一个中立酒馆的后屋——索恩的一个代表、诺伍德、几个其他地下势力的头目都参加。玩家作为诺伍德的人列席。会上会透露拍卖会的具体时间、大致地点、以及"预期的干扰方"——蛾之教会预期灯之教会或银棺修会可能会派人闯入。

玩家可以：① 向诺伍德建议退出（如果他退出，蛾之教会外围安保会出现缺口——这可能会改变拍卖会夜的局势）；② 保持沉默参加——获得第一手拍卖会情报；③ 向第三方泄露此次会议的内容（引发势力冲突）。
