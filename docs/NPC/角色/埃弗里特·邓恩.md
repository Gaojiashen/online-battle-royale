## 背景：一个不敢好奇的人

埃弗里特·邓恩二十岁进了市政厅档案室。他的第一份工作是给旧档案编号——把过去五十年的治安报告、财产登记、税务记录按年份和类别整理归档。这份工作不需要聪明，只需要细心和不偷懒。他有这两样。

**十五年过去了。他还在档案室。职位从"档案助理"变成了"档案管理员"——头衔变了，工作没变。** 他的世界是四堵墙、三百二十七个档案柜、和一个他已经背熟了的编目系统。他知道每一份文件的位置，但不知道大部分文件的内容——他学会了"不读"。档案管理员的工作是找到文件，不是理解文件。

但他偶尔会不小心读到一页不该读的。一句"现场提取到三根带有异常燃烧痕迹的蜡烛"——在一份关店火灾报告里。一个"死者双手被交叉置于胸前，面朝东方"——在一份意外死亡记录里。他每次读到了，就会把那页翻过去，深呼吸一下，然后放进编目系统里属于它的那个格子。他不问。不查。不告诉任何人。

**他和埃德蒙·里德有一个微妙的默契。** 两个人都是被各自机构收编的"信息管理员"——一个是市政厅的档案室，一个是圣吉尔斯图书馆的闭架区。他们之间的信息交换看起来是学术性的：埃弗里特帮埃德蒙找市政厅的旧地图，埃德蒙帮埃弗里特确定某些旧书中的引文出处。但实际上——两个人都知道对方在找东西。只是都不问对方在找什么。

**伦威克来借档案时，埃弗里特每次都会多看他一眼。** 那个队长的调阅申请里有一种"我知道我在找什么但我不说"的气质——和埃弗里特自己的"我知道我看到了什么但我不说"正好是对应的。伦威克从来不解释他的调阅理由。埃弗里特从来不问。这是他最喜欢的类型的人。

## 表层身份

**市政厅档案室管理员。** 那个在走廊尽头楼梯下面、你交申请表然后等半个小时、他把你需要的文件推出来的人。大多数人在他面前只停留三分钟——足够的时间让他找文件、让你签登记本、然后各自离开。没有人记住他的脸。他记住每个人的调阅记录。

---

## 真实欲望

**在所有人不注意的情况下，把那些"不该在档案室里的东西"完整地保存下来，直到有人需要它们。**

他不是求道者。他不知道密传是什么意思。但他做了十五年档案管理员——他知道什么文件是"被故意放错位置"的、什么文件是"有人在等着被销毁"的、什么文件是"只要有人问起就会被处理掉"的。他有一个私人抽屉——藏在最角落那个标注为"废弃表格"的档案柜底层——里面是他从销毁清单上悄悄抽回来的几页纸。

他想知道的三个问题：

1. **唐纳63年"书店火灾"的档案里到底写了什么？** — 他读了第一页。那不只是火灾报告。里面提到了一个名字——"E·L·"——和几条被涂黑的段落。涂黑的人在涂黑之前忘了把所有内容都涂掉。他看到了一个跨页的标题："跨……过渡论证"。他没看懂。但他没合上。
2. **为什么最近有人在大量调阅唐纳63年的档案？** — 过去三个月，有六个人——不同的名字、不同的机构——调阅了唐纳63年相关的卷宗。这不是巧合。有人在找东西。他在登记本上圈出了这六个名字——万一他们中的某一个回来的时候需要被人注意。
3. **埃德蒙到底在闭架区藏了什么？** — 那个图书馆员的借阅记录里有好几本书是他不应该有的——被标记为"已销毁"的、被教会上锁的、作者名被刮掉过的。埃弗里特不觉得埃德蒙在做坏事。但他好奇——如果一个人读了那些不该存在的书，他会变成什么？

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 内向、寡言、容易被忽略。他说话声音不大，会用"可能"、"也许"、"我不确定"来缓冲每一个判断。 |
| **真实一面** | 比任何人想的都敏锐。十五年的归档不是机械劳动——他看到了这座城市文件系统里的模式：哪些文件"被丢失"了、哪些记录在某个日期之后突然中断了、哪些人的名字在不同年份反复出现但每次写法都略有不同。他不说。但他知道。 |
| **弱点** | ① 一旦他开始读一份不该读的文件——他就会读下去。这个弱点他藏了十五年，只在唐纳63年的盒子上失控过一次。② 他的道德感——他不会为了自保而销毁一份他知道重要的档案。如果有人命令他批量销毁——他会拖延，会藏起其中最关键的几页。③ 他怕冲突——如果面对直接的威胁，他可能会交出来。然后恨自己。 |
| **道德立场** | "档案不应该被销毁。即使是不好的档案——尤其是'不好的档案'——更应该被留着。因为总有一天需要有人证明发生过什么。"这不是政治立场。是档案管理员的本能。 |
| **可被利用的方式** | ① 告诉他某份文件将被销毁——他会在销毁令下达之前"备份"一份。② 给他一个他从未见过的新信息——他会用旧信息交换。③ 威胁要举报他"私藏被封档案"——但做好心理准备，他交出来之后会再也不和你说话。 |

## 触发事件

**事件一：被封的盒子**

如果玩家因为任何原因需要查阅唐纳63年的官方记录——比如在调查十年前"书店火灾"的真相——埃弗里特会在你填写调阅申请时沉默半分钟。然后他说："那年的档案有一份不能给你看的。是被密封的。"

他会观察你的反应。如果你表现出知道里面是什么——或者提到火灾中有一个死者叫洛克哈特——他会压低声音："我去拿给你。但你不能让人知道是我拿的。还有——我需要你知道一件事：有人在那份档案上涂改过。涂得不干净。"

给玩家看档案需要付出代价——具体的代价取决于玩家已有的立场。可能的出价形式：① 帮他查一份他不敢自己查的调阅记录背后的申请人信息；② 从埃德蒙那里帮他借一本被闭架的书；③ 一笔钱（他不是勒索，他是需要给自己一个"我收了钱所以我是被利用的"的心理借口）。

**事件二：销毁令**

丹比——埃弗里特的上级——签发了一份档案清理令：将一批标注为"无效财产记录"的旧文件销毁。埃弗里特在整理时发现了其中的异样——这批文件实际上是唐纳55-60年间高街沿线几家店铺的原始地契，其中就包括那间被火灾摧毁的书店。

埃弗里特在新调令执行前开始"生病"——不是真的病，是拖延。他在等一个理由不销毁这些文件。如果玩家在这期间找到他——他会偷偷把其中最关键的几页塞给你："帮我保管几天。如果丹比问——就说没看到这份文件。"等到风头过了再取回——或者由你来决定它们的最终去向。
