## 背景：书脊之间的裂缝

**埃德蒙·里德做了四十一年的档案管理员和图书管理员。**

他不是学者——他没有写出过任何惊世骇俗的论文，没有担任过教职。他的工作是：分类、编号、登记、上架。他知道每一本书的位置，知道哪些书被借出、被谁借出、逾期了多久。

正是这个"无聊"的工作，让他成为了这座城里**最了解谁在读什么的人**。

大约二十年前——他还没意识到事情的严重性的时候——他开始注意到一些模式：某些书被同一批人反复借阅。那些书表面上毫无关联（一本炼金术手册、一本苏格兰地方志、一本关于钟表机械的拉丁文小册子），但借阅名单高度重叠。他出于习惯记下了这些模式。

然后那些借阅者中，有人死了。有人失踪了。有人换了名字继续借书。

埃德蒙花了十年时间才意识到自己在追踪什么。那些书——单独看都是无害的学术著作——但放在一起，它们拼凑出了一幅关于「灯」之道路的模糊地图。每一个借阅者都在沿着某条线索前进，而那条线索……指向飞升。

他没有告诉任何人。他开始用自己的方式追踪——在编目卡片上留下只有自己能看懂的暗记，在闭架区偷偷复制某些被撤出流通的书页，在深夜的图书馆里比任何人都待得更久。

**他不是灯之教会的成员。但他是一个比大多数教会成员更接近灯之核心秘密的人。**

十年前——唐纳63年——那场被称作"书店火灾"的事件发生时，埃德蒙是第一批被叫去整理遗物文献的人之一。他在废墟中拣出了一本半烧毁的皮革面手账。封面烫金的字已经熔化了，但他认出了那些被烧得卷边的书页上的笔迹。

他见过这个笔迹。在过去的借阅记录中。在一份已经注销的读者登记卡上。

**那名飞升者，曾经在他的图书馆借过书。**

从那以后，他不再只是"好奇"。他在追查那条线索。他想要知道：一个在图书馆借过书的人，是如何一步一步走到飞升的。

## 表层身份

**圣吉尔斯图书馆的高级档案员。** 一个无害的老馆员，在这座城市里几乎和书架一样古老。教授们对他客气但不会多聊，学生们找他找书，他从不出错。他的办公室里有一把谁都可以坐的旧椅子，但很少有人坐下来和他聊天——他身上那股旧纸味让人不自觉地想保持距离。

他的办公桌右数第二个抽屉里有一个暗格，里面放着那本烧焦的手账、一份他自己绘制的"关联图"、以及三把钥匙——分别对应三间他无权进入但知道怎么打开的闭架书库。

---

## 真实欲望

**还原那场飞升的全貌——不是为了飞升本身，而是为了理解"一个人如何通过知识变成光"。**

他一生都在和书打交道。如果阅读的终极结果是变成另一种形态的存在，那他四十一年的职业生涯突然有了全新的意义。

他想知道的三个问题：

1. **那个人在飞升前的最后三个月读了什么？** — 他已经拼凑出了那条借阅链路的80%，但那最后三个月的记录是空的——要么是那个人停止了借书，要么是有人在事后清除了记录。
2. **还有多少人在重复那条路？** — 他注意到近两年新的借阅模式又在形成。新的面孔，相似的书目组合。他的暗记告诉他：有人在沿着同样的轨迹前进。
3. **飞升之后的"痕迹"能不能被读？** — 如果灯是揭示的本质，那飞升者留下的残余信息能不能像书一样被"阅读"？

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 温和、稍显迂腐、话不多的老馆员。借书、还书、逾期罚款，一切照规章。 |
| **真实一面** | 极度敏锐的观察者。他很少暴露自己知道多少——往往是在你已经说了很多之后，他才轻描淡写地抛出一句让你意识到他什么都知道的话。 |
| **弱点** | 他的心脏不好（近两年开始出现心悸），长时间熬夜后会发作。另外——他对"即将被销毁的文献"有一种无法克制的拯救欲，这可能被用来设陷阱。 |
| **道德立场** | 他相信知识不应该被销毁或隐藏。但他也相信——有些知识需要**时机合适**才能被阅读。他不是一个无差别的信息散播者。 |
| **可被利用的方式** | ① 一本他知道存在但以为已经失传的书 ② 一条他尚未拼凑完整的借阅链路中的关键信息 ③ 有人威胁到图书馆的藏书（火灾威胁比人身威胁对他更有效） |

  「……灯不是找到的。灯是——」

## 附注：里德的暗记系统

他在编目卡片上留的暗记很简单——左下角用铅笔点一个点：
- **一个点** = 这本书的借阅模式可疑
- **两个点** = 借阅者已死亡或失踪
- **三个点** = 核心线索，他自己也不完全理解

没有人注意到这些点。或者至少，没有人问过他。

如果你注意到这些点并问他——他会沉默很久，然后说："那是书签标记，方便我找到常被借阅的书。" 然后他会观察你是否相信这个回答。
