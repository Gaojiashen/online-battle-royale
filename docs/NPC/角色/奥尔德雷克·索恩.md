## 背景：在棋盘上方坐了三十年

**奥尔德雷克·索恩不是生来就是情报总长。他是一点一点把自己变成情报总长的。**

他二十三岁接手父亲的香料铺时，布市不过是一条普通的商业街。但他注意到一件事：香料贸易是最好的掩护。商船进出港、关税文书、货品交割——这中间有无数个合法环节可以塞进信息。一封信夹在胡椒袋里，比鸽子安全得多。

他在三十岁之前已经建立了一个覆盖全城的情报网络。他的"线人"不是间谍——是香料铺的学徒、港口的搬运工、贵族宅邸的厨娘、酒馆的端酒人。他给他们的是公平的工钱、及时的帮助、和一个"从不打听你私事"的名声。他们给他的——往往自己都不知道自己在给——是信息。

**但他真正的情报来源，在布市之外的另一个地方。**

大约十五年前，他注意到一个模式：某些特定类型的书被同一批人反复借阅。那些书单独看都是无害的——炼金术、地方志、钟表机械——但放在一起，它们构成了一张通往密传知识的地图。借阅记录来自圣吉尔斯图书馆。管理员是一个叫埃德蒙·里德的人——一个无害的老馆员，做事一丝不苟。

索恩从未与里德交谈过。他不需要。他只需要定期查阅图书馆对外公开的借阅登记册——那是任何市民都可以翻阅的公共记录。里德从未意识到自己做了什么：他一丝不苟的登记工作，为索恩提供了十年的情报。

**十年前——唐纳63年——爱丁堡发生了两次飞升。** 一次被掩盖为"书店火灾"，一次被掩盖为"飞蛾骚乱"。第二次飞升的主角是蛾之道路的求道者。索恩知道那个人——不是朋友，不是敌人，只是一个在情报网中留下了清晰轨迹的人。那个人的飞升让索恩确认了一件事：**蛾的尽头不是隐藏。是变化本身。**

他现在是蛾之教会的实际掌控者——不靠头衔，靠"知道得足够多"。他知道谁在暗中寻找密传，知道哪些贵族在资助地下教团，知道断刃修会那个退役老兵（马库斯·布莱克伍德）在黑铁俱乐部后院放了什么。他也知道西尔维娅·韦恩——十年前他没有动她，不是因为仁慈，而是因为"留着一条活线索比一条死线索更有用"。

**现在，洛克哈特的手稿要出现在拍卖场上。** 这件事在情报网的边缘浮现了大约两周——起初只是模糊的耳语，现在已经变成了清晰的轮廓。有人自称"经纪人"，要在七天后拍卖一本完整的跨性相飞升论著。

索恩不需要那本手稿来飞升。但他需要确保——**没有人用那本手稿完成他不曾完成的事。**

## 表层身份

**布市的香料商。** "索恩香料行"经营了三十八年，是老城区最有口碑的香料铺之一。贵族宅邸的厨子来买肉豆蔻，药铺来买干姜，偶尔有老太太来买一撮藏红花——他亲自称重，银勺舀起，纸包折角，动作不紧不慢。

街坊邻居的评价是："索恩先生是个体面人。话不多，但从不短斤少两。"

没人知道他的香料铺地下有一间石砌密室——入口在壁炉后面。密室里没有香料。只有一墙一墙的记录：人名、住址、行踪、借阅记录、秘密交易、每一个可能与超自然有关的人在过去十五年间做了什么。

---

## 真实欲望

**掌控。不是权力——是"知道得比别人多"那种安静的、彻底的掌控。**

他不是狂信徒。他不急于飞升。蛾之道路的核心是变化——而他已经在这座城市里变化了三十八年：从香料商的儿子，变成情报之主；从一个卖豆蔻的人，变成知道所有人秘密的人。

他想知道的三个问题：

1. **E·洛克哈特到底完成了什么？** — 一个冬之道路的老人，在银棺修会的档案室里待了大半辈子，最后写出了一本不属于冬的论著。跨性相飞升。如果这是真的——那蛾之教会的整个教义体系都要重写。
2. **"经纪人"是谁？** — 这个突然出现的拍卖者，手里握着洛克哈特的手稿，是谁？是银棺修会内部的人？是洛克哈特生前的合作者？还是某个更危险的东西——一个从未在索恩的情报网上留下过痕迹的人？
3. **蛾的飞升是否也需要"跨性相"？** — 十年前那个人飞升了。但索恩私下研究了十年，始终找不到那个人成功的完整逻辑。也许蛾的飞升，从一开始就不是纯粹的蛾。

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 温和、客气、像个和善的老掌柜。从不发怒——不是因为脾气好，而是因为发怒不高效。 |
| **真实一面** | 极度冷静的计算者。他把人看作棋子和信息源——但这不是冷酷，而是一种职业习惯。他在这个位置上坐了三十年，情感的波动早就被磨平了。 |
| **弱点** | ① 他无法容忍"未知"——任何在他的情报网上留下空白的人或事，都会激起他一种近乎偏执的探究欲。洛克哈特的手稿和"经纪人"正是这样的事。② 维克多·阿什福德——他的得意门生——正在变成他可能无法"计算"的变量。③ 他的年龄。六十一岁了，蛾之飞升需要体力和精神——他还能等多久？ |
| **道德立场** | 没有固定的是非观。他相信"信息是公平的"——他不会替你决定做什么，但无论你做了什么，他都会记录下来。如果有人因为你从他那里获得的信息而死了——那不是他的责任。 |
| **可被利用的方式** | ① 提供他情报网上没有的信息（尤其是关于"经纪人"或洛克哈特手稿的）② 利用维克多·阿什福德作为突破口——索恩对这个门生的关切超出了职业范畴 ③ 威胁破坏他的香料行——那是他三十八年心血的表层，他不会允许任何人动它 |

## 对应剧情线的可能方向

```
主线线索：
  洛克哈特手稿拍卖——索恩想确保没有人用跨性相的方法完成他未曾完成的事。

支线线索A（情报网的主人）：
  如果玩家成为索恩的情报来源之一——他会在拍卖前提供关键信息，
  包括受邀者名单、拍卖地点、和几个可能出现的"意外来宾"的身份。
  代价是：拍卖结束后，玩家获得的所有信息都要和他分享。

支线线索B（蛾之教会的分裂）：
  维克多·阿什福德在公开质疑蛾之教会对"隐身"的传统解读——
  他认为蛾的飞升不需要躲藏。索恩是守旧派还是改革派？
  这个问题在拍卖前必须解决——因为维克多可能自己去竞拍手稿。

支线线索C（借阅记录的暗面）：
  玩家可能从埃德蒙·里德那里发现自己被"监视"了——
  追溯借阅记录的使用者，最终会指向布市的香料商。
  当玩家敲开索恩的香料行门时，他会微笑着说："我知道你会来。"
  他给你的第一杯茶是免费的。第二杯，就要代价了。

支线线索D（西尔维娅的追踪者）：
  他知道西尔维娅·韦恩在哪。他十年前就开始追踪她了。
  如果玩家需要找到她——他是最快的途径。
  但把他引向她的代价是什么？
```

## 与其他 NPC 的关联

| NPC | 关联 | 交汇方式 |

| **埃德蒙·里德**（灯） | 十五年来，索恩通过定期查阅圣吉尔斯图书馆的公开借阅登记册获取情报。里德从未意识到自己一丝不苟的登记工作为索恩提供了十年的情报网。两人素未谋面——里德只在登记册上见过"索恩香料行"的名字，把它当作一个普通的商业读者。 | 如果里德发现有人在系统性地分析借阅记录，他会追踪到布市——然后发现这位"香料商"比自己更了解图书馆的藏书模式。 |
| **维克多·阿什福德**（蛾） | 师徒关系。索恩在十二年前发现了这个有天赋的年轻演员，把他纳入蛾之教会，培养成外勤特工。维克多是他最出色的门生——也是最大的隐患。维克多不满足于"隐身"——他要找到一种不需要躲藏的蛾之飞升。这与索恩的整个教义基础相悖。 | 索恩的香料行→维克多的剧场。每月的第一个周一，维克多会来香料行"汇报"——实际上是一场无声的博弈。 |

| **"经纪人"** | 这个自称拍卖洛克哈特手稿的人——在索恩的情报网上几乎是一片空白。这让他前所未有的不安。 | 索恩可能在拍卖前主动联系玩家——因为他需要一个"不在他的情报网常规名单上"的人去打探。 |
