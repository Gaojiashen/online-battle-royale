# NPC：扒手皮特·科斯特洛（Pete Costello）

---

## 基础信息

| 字段 | 内容 |
|:-----|:------|
| **全名** | 皮特·科斯特洛（Pete Costello）——"扒手皮特"这个绰号他自己接受，但不喜欢。他说："扒手是一种职业。不是一种身份。" |
| **年龄** | 26 岁 |
| **性相** | 普通（无性相） |
| **外观** | 瘦小灵活，骨架子比实际年龄看起来更年轻——二十六岁的人、十九岁的身体轮廓。淡褐色的短卷发，脸窄，眼睛转得很快——不是在逃，是在持续扫描周围所有人的口袋和腰带和手腕上的东西。穿着不起眼的暗灰色短外套——太不起眼了，不值得被注意——但如果你仔细看，外套内衬上有十几个他自己缝的暗袋。手指细长而灵活，中指上套着一枚镀铜戒指——不是装饰品，是他给自己"毕业"的标记：第一次从格拉斯市场的一个富商口袋里掏出了一只银烟盒。 |
| **嗓音** | 快、跳、时不时冒出一个自嘲的笑声。他会一边说话一边在手里翻一枚硬币——永远在翻，像他的双手需要被持续占用。 |
| **常在区域** | ⑧格拉斯市场（最佳的工作地点——人多、混乱、人群的注意力都在摊位上）、④高街（"高风险高回报"——有钱人的聚集地，但城防军也多） |
| **偶尔出没** | ⑩牛门（找格里格销赃——或找玛格特蹭一杯酒）、⑱圣路德公园（教过基特在这里练习"两指取"——他现在对此有复杂的感受） |
| **最近行踪** | 他在高街从一个衣着考究的中年绅士身上摸出了一个皮夹子。回家打开一看——里面没有钱。只有一张折了四折的羊皮纸片，写满了他不认识的字。有些字在纸面上发着微弱的光。他发现这个东西在他碰过之后——羊皮纸上的字开始动了一下。 |

## 背景：三根手指和一个命运

皮特·科斯特洛十岁被格拉斯市场的老扒手"独眼乔"收为徒弟。乔的规矩很简单：每天回来要么交钱要么交东西，交不出就睡在外面。

十二岁，皮特已经能在不被发现的情况下从一个人身上摸走手表、钱包、和钥匙——三个不同的口袋，同一个人，来回六趟电车。乔说他是"五十年一遇的手指"。

十八岁，乔被城防军抓了，死在拘留室里——不是虐待，是六十岁的身体受不了拘留室冬天的湿冷。皮特从那以后一个人干。他继承了乔的三个原则——① 不偷穷人的钱。② 不在同一天、同一个地点偷两个人以上。③ 如果你发现你偷的东西不对劲——扔掉。不要好奇。

**他违反了第三个规则。**

**基特——那个十五岁的街头流浪小孩——是他两年前教出来。** 不是正式收徒，是基特自己在格拉斯市场跟着他三天，怎么甩都甩不掉。最后他们在圣路德公园的长椅上坐了一个小时——皮特教了他"两指取"和"话术转移"。他不确定自己应不应该教。他看着基特像看着十二岁的自己——同样的瘦、同样的饿、同样的"想靠手指改变命运"的眼神。他不知道那是什么结局——但乔给了他那些手指。他觉得他欠这个世界一个传承。

后来基特消失了。他听玛格特说基特"最近还好"。他不问了——因为他怕听到自己不想听的事。

**他从格里格的当铺学到了一件重要的事：不是所有值钱的东西都应该被销赃。** 格里格有一次收了他一只从新城西区摸来的旧怀表——看了一眼，然后推回来，说："这个不是金子。是记忆。你不应该卖掉它。"皮特后来在当铺后屋听格里格说了四十分钟关于"有些东西的价值不在金属里"。他不全懂——但他在那之后开始把一些"觉得不对劲"的东西留在自己床下的木箱里，而不是拿去卖。

现在，那张发光的羊皮纸也在床下的木箱里——旁边是他攒了三年的现金、一本翻烂了的旧小说、和二张他觉得"特别"但说不清哪里特别的旧纸片。

---

## 表层身份

**扒手——街头底层生存者。** 他没有"职业"——扒窃是他的收入来源，格拉斯市场是他的工作场所，格里格的当铺是他的银行。他不知道怎么做别的事。如果有一个不需要偷的未来——他还没想到那个形状。

---

## 真实欲望

**存够钱，开一间自己的小铺子——不是当铺，是"旧货铺"。买下那些被人丢掉但还有用处的旧东西，修好，卖给需要它们的人。**

这个愿望他在酒喝多了之后才敢说出来——因为一个扒手说"我想开铺子"听起来像一句笑话。但他真的有在存钱。他的积蓄够了开半间铺面的首期。再过两三年——他也许就不是扒手皮特了。他也许就是格拉斯市场上那间"科斯特洛旧货铺"的老板。

他想知道的三个问题：

1. **他偷的那张发光的纸是什么？** — 他认识每一个他偷过的东西在格里格那里的估价。但这一张他不敢带去找格里格——因为那个中年绅士看到皮夹子被偷后的反应很奇怪：他没有喊。没有追。他只是站在高街的正中央，用一个不像愤怒、更像恐惧的眼神扫了一整条街。皮特觉得那个人怕的不是被偷钱——是那张纸被读。
2. **基特现在在做什么？** — 那个孩子学会了皮特教的手法，但没学会他的克制。基特偷东西的时候不选目标——谁的都拿。皮特听过一些传闻——基特开始和一群"不是普通街头扒手"的人混在一起。玛格特说她"最近还好"——但玛格特对所有不确定的事都说"最近还好"。
3. **"独眼乔"为什么死在拘留室？** — 乔的身体没那么差。乔在被抓的前一天和皮特说了一句话："如果我出事了——不要来找我。把我的东西烧掉。"乔在等什么？或者——在躲什么？皮特当年没有多想。现在他开始在想：如果乔不是因为偷窃被抓的——而是因为偷错了别的东西？

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 油滑、自嘲、永远在开玩笑——用自己的职业作为笑话材料。那种轻松是他处理恐惧的方式——如果他不笑，他就会发现自己在害怕。 |
| **真实一面** | 一个困在了"生存"和"想要更好的生活"之间的年轻人。他的手指是他唯一值钱的资产——但他知道这些手指能给他的日子是有期限的。有一天他会失手。有一天他的手会开始抖。在那之前——他想攒够离开这个职业的钱。 |
| **弱点** | ① 好奇心——独眼乔说"不要好奇"，但他从来没学会不好奇。那张发光的纸就是证据。② 基特——如果那个孩子出了事，皮特会冲进去，不管后果。③ 他的积蓄——全在他床下的木箱里。如果有人找到了它——他的人生会倒退五年。 |
| **道德立场** | 不偷穷人。不偷为别人买东西的人。不偷一个人的最后一点现金。他的道德体系是街头建立的，但它是真实的——比某些地上面的人管用得多。 |
| **可被利用的方式** | ① 告诉他你是基特的朋友——或威胁那个孩子。② 给他一个好奇——任何"不太对劲"的物品都会让他想摸进来看看。③ 那张发光的纸——如果有人来找它，皮特会用纸换保护。 |

## 触发事件

**事件一：偷错的皮夹**

皮特在格拉斯市场或高街附近碰到玩家——他的状态明显不对。平时那个笑嘻嘻的扒手现在是苍白的、紧绷的。他会主动接触你——如果你已经在街头剧情中有过交集——然后直接说："我搞了个东西——我觉得有人在找。帮我几天。求你了。"

他偷到的那张发光羊皮纸是某人的密传片段——可能是蛾之教会或某个独行者放在那个中年绅士身上准备传递的。现在，那个"中年绅士"的人开始在格拉斯市场打听一个"手指特别快的年轻人"。

玩家可以选择：
- **庇护皮特**：把他藏在一个安全的地方（玛格特帮忙提供的地下通道、诺伍德的地下暗道、或在格拉斯市场的某个隐蔽角落）。代价是——那个追杀者可能会来找玩家。
- **帮皮特脱手**：带那张羊皮纸去找一个懂密传的人（阿利斯泰尔、卡珊德拉、或埃德蒙）——搞清楚它到底是什么、谁在找它、以及怎么在不伤到皮特的情况下还回去或销毁。
- **不管**：几天后，皮特消失了。玛格特会说"他可能出去了一段时间"。他的木箱还在他床下——里面还有那张羊皮纸和上面那行已经开始变淡的发光字。

**事件二：教一手**

如果玩家和皮特的关系在中立以上——他会在一个没什么事做的下午说："想学吗？"

他会安排一个小训练——在格拉斯市场，不偷真人，用假人练习。成功学会基础扒窃手法——对玩家后续涉及盗窃、潜入、或混入人群的行动提供小幅度能力增益。代价是一件小"人情"——可能是帮他跑一趟格里格的当铺，可能是帮他打听基特的消息，可能只是在这周内让他用你的身份（不是名字，是"站在你旁边的时候他被城防军盘问的概率降低"）。

这是个轻松的支线事件——不是剧情的核心，但能加深玩家与地下势力之间的联系。

## 与其他 NPC 的关联

| NPC | 关联 | 交汇方式 |
|:----|:-----|:---------|
| **格里格·哈尔斯特德**（心） | 销赃渠道。皮特是格里格的老客户——两人之间的交易方式是：皮特把东西放在当铺后屋的第三个抽屉里，格里格第二天放现金在同一个抽屉。不需要见面。不需要签字。 | 如果玩家通过皮特介绍认识格里格——格里格会说"皮特介绍的人我多给五分钟"。玩家因此获得更长时间的对话窗口。 |
| **玛格特·芬奇**（普通） | 偶尔一起在牛门喝酒。玛格特是皮特在这个地下世界里唯一一个"不是交易对象"的关系——两个人都不用对彼此撒谎，因为都没有利益相关。 | 如果皮特出了事——玛格特会是第一个注意到他三天没出现的人。她会开始找。 |
| **基特**（未设卡） | 学徒——十五岁的街头小孩。皮特教了他扒窃，基特学会了但开始走向一个皮特没走过的方向。皮特为此内疚，但不知道该怎么办。 | 如果基特在其他剧情线中出现——皮特的角色会变成一个"前导师"的视线。基特可能会回来找皮特——求救、求钱、或求庇护。 |
| **伦威克**（普通） | 被注意与不被注意——皮特知道城防队队长伦威克的存在，伦威克知道格拉斯市场有扒手但他不抓这种小案子。两人之间唯一的交汇是：皮特在格拉斯市场偷过一个不该偷的人，而伦威克在档案里注意到这个"不小心丢了东西"的人是一个已经被他标记成"可疑"的名字。 | 如果伦威克和皮特因为同一件密传碎片产生间接交集——那张羊皮纸会成为两人信息库的交汇节点。 |
