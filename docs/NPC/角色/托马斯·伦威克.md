## 背景：在不该有模式的地方找模式

**托马斯·伦威克二十岁加入城防军。二十五岁的最后一天，他看到了第一具"不对"的尸体。**

那是一具从牛门地下拱顶拖出来的浮尸——官方结论是"酒醉失足落水"。但伦威克注意到了两个细节：死者右手握着一根蜡烛（在水里泡了三天之后，手指肌肉僵直还在握着），以及死者的瞳孔——不是溺水的扩散，而是一种极不自然的收缩，像在死前盯着一个极其明亮的光源。

他写了报告。上司看了，放到一边，说："不要给自己找麻烦。"

他没有停止。他开始在巡逻时注意每一个"意外死亡"的现场——车祸、坠落、突发疾病、酒醉冻死在巷子里。他不是侦探，城防军没有侦探这个职位。但他有一双"受过训练的眼睛"——二十年的军旅生涯让他能分辨一道刀伤是打斗造成的还是摆拍出来的。

**二十五年。他收集了二十五个"不对"的死亡案例。**

他把它们归档在自己的办公桌最后一个抽屉里——不是官方卷宗，是手写的笔记。每个案例都有时间、地点、官方结论、他的质疑、和一个他自己起的代号：蜡烛手、钟表匠、白衣女人、无舌歌女……他用代号来防止任何偶然翻到他抽屉的人看懂里面的内容。他自己也说不清这些名字的选择逻辑——只是觉得"那个东西不应该被直接用名字称呼"。

十二年前，他的婚姻破裂了。他妻子说："你回家的时候人在这里，但眼睛不在。"她是对的。她离开后，伦威克把更多时间花在档案和巡逻上。他的下属邓肯·麦克莱恩——一个老实的点灯人——有一次发现他凌晨三点还在办公室，问他在看什么。他说："在看灯。"邓肯没再问。邓肯知道有些问题不该问。

**他的线人玛格特·芬奇——格拉斯市场上一间杂货铺的女主人——是他最稳定的信息来源。** 六年了。她每隔两周的周四午后会用"给兵营送鸡蛋"的名义来内堡门，把一张折好的纸条压在鸡蛋篮子下面，上面是她注意到的格拉斯市场一带的异常动静：生面孔、半夜里的动静、不该在白天关门的铺子。

三年前，他开始注意到卡珊德拉·普莱斯——高街上那个住在装订匠楼上的女医师。她的几份验尸报告被他调阅过——写得极其专业，带着一种他在其他城防军法医报告中从未见过的精确度。他约了她喝了一次茶，一个小时后，他确认了一件事：**她知道的不比他少。甚至更多。**

从那以后，他每隔一段时间会把一份"说不清"的案例送给她——让她从医学角度来判断，死因中是否有什么"不符合自然规律"的东西。她的回复从不让他失望。

**他怀疑马库斯·布莱克伍德——西弓街上那个退役老兵，黑铁俱乐部的老板。** 马库斯的一个前战友三年前"意外坠楼"了——颈骨折断，落在自家院子里。伦威克看过现场。他注意到马库斯是第一个到达现场的人——比城防军还快。当被问及为什么会在那里时，马库斯说："他欠我钱。"

他没有证据。二十五年了，他从来没有过任何一份确凿的证据。有的只是那个抽屉——那些用代号命名的笔记。那些"不对"的东西。

**现在，洛克哈特的手稿即将出现在拍卖场上。** 伦威克不知道洛克哈特是谁。不知道跨性相飞升意味着什么。但他知道一件事：当"那种东西"被交易的时候，就会有"不对"的尸体。他需要在那之前，找到一个能让他介入的合法理由。他是城防军队长——不是侦探，不是求道者，不是一个有超自然能力的人。他只是一个知道该看哪里、并且看了二十五年的人。

## 表层身份

**城防军队长。** 在内堡门的兵营里有一间小办公室。他的日常工作乏善可陈：巡逻、执勤、处理醉酒斗殴、调解邻里纠纷。他的下属尊重他——不是因为他有威严，而是因为他公道。他从不欺负普通人，从不让手下收小摊贩的"保护费"。这种作风在城防军里不常见，但也没有人因此排挤他——他做得够久了，久到所有人都默认"托马斯就是这样"。

---

## 真实欲望

**在有生之年，找到一个"不对"的案例——一个可以被证明的、不容置疑的、可以公之于众的真相。**

他不追求飞升。他不知道密传是什么——至少他不知道那个词。但他知道有一种力量在这座城市的地下运作，每隔一段时间就会留下一个"不对"的尸体。他想证明这种力量**存在**——不是为了对抗它，而是为了"那些死了的人不应该被记成'意外失足'"。

他想知道的三个问题：

1. **"蜡烛手"的蜡烛意味着什么？** — 二十五年前的第一具尸体，手握蜡烛，瞳孔收缩。他至今没有找到任何能解释这一点的医学或物理学依据。他知道卡珊德拉可能知道。他还没有问她——因为他怕答案是"不要问了"。
2. **马库斯·布莱克伍德在藏什么？** — 一个退役老兵，开着一间格斗俱乐部，后院有一个上锁的房间。他的前战友"坠楼"了。他第一个到达现场。这不是巧合。但他没有搜查令、没有证据、没有马库斯所属"那个圈子"的任何线索。
3. **这次拍卖会带来什么？** — 他通过玛格特听到了一些模糊的传闻：有人在找买家，有人在准备护送，有人从外地来了。他不知道拍卖的是什么。但他认识这种节奏——每次"那种事"发生之前，都会有这样一段安静的、紧张的酝酿期。上一次他看到类似的模式，是七年前的"书店火灾"。

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 务实、寡言、不轻易相信任何人。但他不是冷漠——他对普通人有一种不显山不露水的善意：帮老太太提篮子、在雨天让巡逻的手下多休息一刻钟。 |
| **真实一面** | 执念者。他的二十五年不是一种工作习惯——是一种不能停下来的本能。他有时候会想：如果有一天他证明了"那种东西"的存在，然后呢？他没有答案。但他不能停止——因为"如果没有人记录，那些死掉的人就白死了"。 |
| **弱点** | ① 没有证据——二十五年了，他没有任何一份可以公之于众的确凿证据。这让他始终处于一种"我知道但我不能说"的困境中。② 无法求助——城防军内部没有他信任的人，而他不知道该去哪里找"专业的协助"。③ 他对玛格特·芬奇的安全有隐秘的愧疚——他知道一旦有人发现她在给他提供信息，她会成为目标。 |
| **道德立场** | 老派的、沉默的正义。不是英雄——他不会以身犯险去阻止一件他阻止不了的事。但他的档案在。每一个"不对"的死亡都被记录在案。他相信总有一天会有人需要这些档案——在那之前，他的工作就是继续记录。 |
| **可被利用的方式** | ① 给他一份"不对"的新案例——他会放下手上的事去调查 ② 告诉他你知道马库斯·布莱克伍德的秘密——但不说是谁 ③ 威胁玛格特的安全——这会让他立刻行动起来 ④ 给他一份可以被证实的证据——他会成为你最坚定的盟友 |

## 对应剧情线的可能方向

```
主线线索：
  证明"那种力量"的存在——通过洛克哈特手稿拍卖前后发生的事件，
  找到一份不容置疑的铁证。

支线线索A（二十五个案底串联）：
  二十五个"异常死亡"案例中，至少有七个可以追溯到同一条线索——
  十年前两次飞升事件的相关者。
  如果有人能帮他把这些点连起来——
  他会发现他追查了二十五年的东西，有一个名称：密传。

支线线索B（马库斯的对峙）：
  如果拍卖引发的冲突波及到黑铁俱乐部——
  如果又一具"不对"的尸体出现在西弓街——
  伦威克会去找马库斯。不是逮捕——他没有证据。
  但他会站在俱乐部的前台，看着马库斯的眼睛，
  像他二十五年前看着第一具尸体一样。
  马库斯会怎么回答？

支线线索C（邓肯的胸针）：
  邓肯·麦克莱恩——伦威克的下属——在高街上捡到了一枚银胸针。
  如果邓肯把胸针拿给伦威克看——或者如果伦威克在邓肯"出了事"之后
  发现那枚胸针——它会成为他案底中的第二十六个案例。
  代号："点灯人"。

支线线索D（与卡珊德拉的合作升级）：
  如果玩家同时接触伦威克和卡珊德拉——他们可能会被卷入
  两人之间尚未浮出水面的合作：伦威克的现场观察+卡珊德拉的医学鉴定=？
  这两个人如果真正联手，能发现的东西远超任何一方单独的能力。
```
