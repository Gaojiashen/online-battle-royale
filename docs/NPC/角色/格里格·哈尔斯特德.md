## 背景：每一笔交易都有一个心跳

**格里格在格拉斯市场做了三十二年的典当买卖。** 他二十三岁接手父亲的铺子时，货架上只有旧怀表、镀银烛台、二手皮靴。三十二年后的今天，货架上摆的还是旧怀表、镀银烛台、二手皮靴。但后屋经手的物品清单，已经换了天地。

**他独立发现了心之道——不是从密传里读到的，是从交易里悟出来的。**

他二十七岁的时候，一个地下教团的成员来当铺当掉了一把仪式短剑。银柄，剑身上刻着他看不懂的符号。那人要价太高，格里格没接。但那人走后，他在当铺里来回踱了一刻钟。不是因为后悔没做成生意——而是因为在握住那把短剑的瞬间，他感觉到了一样东西：**一种节奏。** 像是短剑在非常缓慢地……振动。不是物理的振动，是某种他后来花了好几年才理解的东西。

他后来把那把短剑的事记在心里。没有去追。没有去找那个教团成员。他做了一件事：开始留意所有"带节奏"的物品。

他发现某些东西——通常是仪式制品、密传残页、或被人长期佩戴的旧饰品——在接触时会传递一种微弱的节律。不是每样都有。不是每个人都能感觉。但他可以。

**他用了二十年自己摸索出了一套心之道的基础理论：每一件"有故事"的物品都保留着它主人的心之节奏。** 交易本身也是一种节奏——供需之间的涨落、讨价还价之间的停顿、钱货两清那一刻的共振。他在这个无人关注的夹缝里，独立打通了一条通往心之密传的道路。

他没有告诉任何人。他的当铺继续开着。后屋的生意——销赃、中间、情报转手——继续做着。

**他是那种"每个人都需要但没有人真正注意"的存在。** 海伦娜从地下市场找仪式材料时，渠道中有一环在他这里。莉莉每天在脏鸭与壶端酒，他不知道她叫什么，只知道她每周四固定点一杯热麦酒、半块黑面包。罗文帮他在敏感交易时把门——他把钱放在桌上，不多不少，加上几枚因为"你让我看着放心"的理由。

格里格不是求道者。他不会飞升。但他的心之性相让他成为了这个城市里最罕见的一种人：**一个能"听到"地下势力脉动的人，而他们不知道他在听。**

## 表层身份

**当铺老板。** 「诚实价值」当铺——名字是他父亲起的，他自己说这名字"既是招牌又是笑话"。街坊们认识他三十年了，觉得他是个和气的、有点抠门的老头，偶尔愿意赊账给有困难的老邻居。

没人知道他后屋的桌子从来没有堆过任何东西。那张桌子唯一的用途是——当一个交易对象坐在对面时，让不出三秒钟格里格就能捕捉到对方的心率节奏。

---

## 真实欲望

**将心之道应用到极致——不是飞升，而是让"交易"本身成为心之力量的载体。**

他不想飞升。飞升太吵。飞升意味着离开——离开这个他花了三十二年编织的、由交易和人心组成的蛛网。他想要的不是离开，而是在这张网里看得更深。

他想知道的三个问题：

1. **每一笔交易的"心跳"能不能被记录下来？** — 他试过用账簿记录，但数字和字迹不能承载那种节奏。他想找到一种方式——一种载体——能够让那些交易瞬间的共振被保存下来。那会比任何密传都更接近心的本质。
2. **谁的"心跳"最不寻常？** — 他能感觉到某些人的节奏异常：太快或太慢、太均匀或太混乱。那些人通常是求道者，或者被某种力量标记过。他想知道这座城市里到底有多少颗"异常的心"在跳动——如果每个求道者都有一个独特的节奏，那他可以在这个游戏开始之前就知道棋盘上有多少子。
3. **那个"中间人"是谁？** — 格里格在地下市场混了三十二年。他从不同的上家和下家嘴里听过同一个称呼——"中间人"或"Broker"——太多次。每次提到这个人，交易的心跳都会变。那种变不是恐惧，不是敬畏，是一种……识货。像一个老行家看另一件好货时的反应。格里格想知道：那个人的心跳是什么节奏？

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 和气、圆滑、永远不会先拒绝。前台的格里格是一个"什么都可以商量的老头"。 |
| **真实一面** | 极度审慎。三条规则刻在脑子里：① 不问货从哪里来，但问货该往哪里去（卖家的安全不关他的事，但买家的安全影响回头客）。② 不碰"会让铺子被烧"的货——他有自己的红线，不是道德的，是风险评估的。③ 永远不要在一天之内做超过两个"大单子"。 |
| **弱点** | ① 他的好奇心——他对"异常心跳"的人有无法克制的兴趣，偶尔会主动接触一个他本不该接触的人，只为"听一下他的节奏"。② 妻子过世后他没有任何亲近的人，偶尔会把这个空缺错误地投射到某个年轻人身上（罗文是个例子）。③ 他对那个"中间人"的兴趣正在变成执念——因为三十二年来，他是唯一一个格里格无法通过交易脉络推算出任何信息的人。 |
| **道德立场** | 没有大是大非。他的道德哲学是：一笔好交易是双方都觉得自己赚了。如果有人亏了——那要么是对方不够懂行，要么是格里格在替他交学费。主动伤害无辜者不在他的商业计划中，但被动地让别人自己走进坑里，他认为那是"市场的公平性"。 |
| **可被利用的方式** | ① 一件他从未接触过的、心之性相极强的物品——他会忍不住想"听"一下，哪怕那东西明显是个陷阱 ② 关于"中间人"的任何信息 ③ 有人威胁要揭露他后屋的生意给市政厅——他会全力以赴保住他的铺子 |

## 对应剧情线的可能方向

```
主线线索：
  心之道的独特诠释——不是通过音乐或舞蹈，而是通过交易的节奏。

支线线索A（心跳地图）：
  格里格在三十二年的交易中，靠直觉记录了几十个"异常心跳"的接触者。
  这些人里——有些是求道者，有些是被力量标记的普通人，有些已经死了。
  如果他能把这份"心跳名单"和埃德蒙·里德的借阅记录交叉比对——
  可能会拼出一张比任何势力都更完整的"谁在走哪条路"的全景图。

支线线索B（中间人的交易痕迹）：
  格里格经手过至少三件物品最终流向了"中间人"。
  他有一笔交易记录——十八年前的冬天——间接关联了某个"没有心跳"的委托人。
  "没有心跳"意味着：要么对方不是活人，要么对方掌握了某种压制心之振动的方式。
  这对追踪"中间人"是谁至关重要。

支线线索C（海伦娜的供应线）：
  海伦娜从他这里拿的仪式材料——其中有一批货的源头是蛾之教会。
  如果蛾之教会发现熔心会的外聘工匠在用他们的材料——
  这条供应线的每一个环节都会变成双方博弈的棋盘。
  格里格夹在中间——而他最擅长的事情就是不让两边的人发现中间有人。

支线线索D（莉莉的日常）：
  格里格知道莉莉——那个每周四固定买热麦酒和黑面包的姑娘。
  他不知道她是"那个记性太好的姑娘"。但如果有人告诉他——
  他会比任何人都更早意识到：有人在暗处观察她。因为她的"心跳"太正常了，
  正常到在一群"异常"的人之中……反而异常。
```
