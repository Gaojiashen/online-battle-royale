## 背景：全城最不被注意的档案馆

**玛格特·芬奇十五岁开始在地下妓院工作。那年她的母亲——一个给有钱人家洗衣服的女人——把她的手交到了牛门一家"地下旅馆"的老鸨手里。理由是："这里比在外面安全。"**

她母亲是错的。但她后来再也没有提过这件事——不是因为原谅，是因为算了。

二十一年。从最底层的、连名字都不需要报的房间开始，一路做到现在——谁都不敢惹的老妓女。不是靠打架（她这辈子没打过任何人），不是靠后台（她没有）。靠的是——**知道的太多，多到有人想动她的那天，至少有七个人会先拦下来。**

在十五年的地下工作里——从牛门肮脏的隔间到地下拱顶更"高端"一点的隐蔽妓馆——她走过这座城市最不被注意的通道。她知道哪条地下室通道通往修道院街的哪口井，知道哪间妓院的后墙推开是通往玛莉金巷的废弃烟道，知道每一个来找乐子的客人真正在找什么——他们脱掉衣服之后说梦话、睡觉之前对着天花板甩出来的那些词：名字、日期、地点、一个教团的称呼、某种"他们非要找到不太然就会被处理掉"的东西。

他们不说。但他们说。她是全城最不被注意的信息源——因为没有人对一个妓女设防。不是因为她是个女人，而是因为她是个三十五岁以上的、不年轻也不够漂亮到让人记住的妓女。她是一面墙。一面灰掉以后就再也没有人愿意看一眼的墙——然后她记下来她听见的一切。

**她不是个殉道者。她卖情报。** 六年前，卫兵队长伦威克在牛门的一次地下扫荡后提审了她——按流程，她应该是最后一个被问到的人，也是最不需要被问的人。但伦威克不是普通人。他用了十分钟——从她回答前三个问题的方式里——意识到这个妓女在同时读他的眼神、猜他的目的、并在不经意间把他刚才试图问她的问题反过来扔向了他。

他没有逮捕她。他提了一个交易。

现在，她是伦威克最稳定、也是最低调的秘密线人。每年固定的薪金——不是贿赂，是"信息采集费"。他不问她的来源，她不问他的用途。她知道他在找什么。她知道他在查谁。她知道他的监视名单上有哪些名字——罗文·刀的名字就在上面，虽然她从来没有告诉罗文。

**但她不只是伦威克的线人。她是她自己。**

她在格拉斯市场的廉租公寓里和莉莉·斯托克一起长大——同一个院子，不同的母亲，共用过同一个水龙头和同一堵发霉的后墙。莉莉比她小十七岁，叫她"芬奇姐姐"。玛格特搬走以后还在定期探视莉莉的母亲——每次去都会带一块她烤得太硬的果仁蛋糕（她只会做这个，还做不好）。

她和罗文偶尔一起喝酒——不是线人的身份，是一个女人和另一个女人的身份。罗文在她面前不说谎。因为玛格特从来不问罗文从哪里来、为什么在躲、刀上的血是谁的。她只是倒酒。

格里格是她的销赃渠道——她捡到的那些东西不需要上正规交易。格里格给她现金价，在账簿上不留她的名字。

**玛格特掌握着足够炸掉这座城市里至少三个势力的信息碎片。但她只卖给伦威克。卖给其他人的，都是她挑过的——安全的、不会害死人的东西。她不会飞升。但她可能是唯一一个能在这张蜘蛛网上安然睡了二十年的人。**

## 表层身份

**地下妓女。** 不是一个固定的妈妈桑，不是一个名妓——她是那种"谁有需要就叫她"的从业者。不在任何一家固定待着，也没有自己的卖身契。那些年轻一点的新来者有些怕她，有些偷偷学她——她们发现这个老女人走进一个房间后，那个房间的客人总是隔了好久才找下一个人。

表面上，她是牛门和地下拱顶地下一层之间那些不归任何人管的灰色妓院里的熟面孔。仅此而已。

---

## 真实欲望

**存够钱，在五十岁之前离开这座城市。**

她不要飞升。她不要权力。她不要复仇。她只想要一个不叫爱丁堡的地方——她不知道"之外"是什么，地图上标明爱丁堡之外什么都没有。但她相信：如果在这座城里有些路可以通到地图上没画的地下空间，那一定有一条路可以通往外面。

她想知道的三个问题：

1. **爱丁堡之外到底有什么？** — 她从客人们的嘴里听过"海"——他们说的是利斯港的海。但如果船可以离开港口，它去了哪里？有没有一个地方，不是湿的，不是冷的，不需要你永远记住每一个人的名字才能活着？
2. **伦威克最终会处理掉她吗？** — 不是因为他坏。是因为她太有用。六年的线人关系让她知道太多。如果某一天她知道的量变成了他的风险——他会是保护她，还是把她放进一份不受欢迎的名单里？她从来不认为伦威克是个坏人。但她从不认为任何人不是"在一定条件下"的坏人。
3. **莉莉能活着出去吗？** — 那个红头发的小姑娘记得每一个人。玛格特知道这不是普通的记性好——而是某种她不能告诉莉莉的东西。如果某一天莉莉记住了不该记住的人——玛格特能在那天之前把她送走吗？

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 圆滑、不亲近不冷漠、永远在微调距离。她知道怎么在一个三分钟对话里让你觉得"这个人好像挺关注我"——但你离开房间以后发现她什么都没说。 |
| **真实一面** | 极度疲惫。二十一年的下面——生理上、心理上、所有的层面上——像一条用太久的抹布，还能用，但已经被拧干了太多次。罗文说她"像一把刀口已经平了的刃——不是为了战斗，是为了还在手里握着"。她真正的自己不是那个笑脸，也不是那个间谍，是那个女人——在深夜无人时坐在自己那间窄小的后屋里，什么东西都不做，坐半个时辰，然后站起来继续走。 |
| **弱点** | ① 莉莉——如果莉莉发生任何事，玛格特的冷静会在瞬间解体。她会做不理智的决定。这是她唯一的弱点。也可能是她最后的人性。② 对伦威克的依赖——六年的稳定让她的防护在慢慢松散。她已经不再自己判断"这条信息交给伦威克安全吗"——而开始默认"安全"。这是任何一种秘密线人关系中最致命的拐点。③ 她不会离开这座城市——不是不想，是不会。二十一年的生活模式让她习惯了"逃跑不如适应"。当真正的离开机会来临时，她可能会害怕。 |
| **道德立场** | 她不对任何势力忠诚。她的忠诚对象是"她不希望他们死的少数几个人"。她出卖过很多人——不是冷血的，是真的需要的。但她从来没有出卖过罗文。从来没有出卖过莉莉。从来没有泄露过格里格后屋的位置。 |
| **可被利用的方式** | ① 莉莉——如果你能以任何方式触动她对莉莉的保护本能 ② 提供关于伦威克的、她不知道的信息——她会用同等的信息交换 ③ 你提供了一条能够离开爱丁堡的途径——不管真假，她都会凑过来看一眼 |

## 对应剧情线的可能方向

```
主线线索：
  这座城市最不可见的人——和她手里攥着的地图上没有名字的脉络。

支线线索A（伦威克的名单）：
  玛格特知道名单上至少有五个名字。其中一个是罗文。
  如果她的名字也开始在上面——不是作为线人，是作为对象——
  她将面临这个选择：继续做线人，还是开始自己找人保护自己？
  如果玩家恰好是那个人——她的情报库将第一次全面开放。

支线线索B（莉莉的倒计时）：
  这条线——在莉莉的角色卡中——莉莉的终点可能是一场"记住了不该记住的人"的意外。
  玛格特是能够提前察觉这一点的唯一的人。
  如果她和玩家联手——她可能能在悲剧发生前敲响警钟。
  如果没人听——那她会在事后找出那个凶手。不是为了警方。是为了自己的良心。

支线线索C（地下"从未被发现"的通道）：
  玛格特手里的某些通道——连地下势力的老手都不知道。
  如果玩家需要一条完全不在任何人预判范围内的渗透路线——
  和老芬奇喝几次酒，她会在某一天晚上用指甲在地上画一条线。
  那条线通往⑮玛莉金巷的某个没人注意的入口，或者通往⑯隐秘水窖的某个废弃阀门。
  然后她不会陪你去。她会说："那条路我走了三十年。如果你死在那边……没有人会找到你。"

支线线索D（离开的路线）：
  如果玩家在探索世界边界相关的内容——
  玛格特的"爱丁堡之外有什么"的问题可能引出世界观中一个被埋没了很久的线索：
  唐纳王朝不是这个城邦的起源，而利斯港曾经有过通往外部的航线。
  那些航线的航海图被谁收走了？为什么所有的记录都在七十年前被撤销？
```
