## 背景：一个空了一半的庄园

科迪莉亚·蒙特罗斯——那时候姓法恩利——二十六岁嫁入蒙特罗斯家族。她父亲是新城西区的商人，嫁女进贵族是她父亲一生最大的成就，也是她自己一生最复杂的交易。那时候的阿奇博尔德·蒙特罗斯还不是侯爵，只是一个刚从父辈手里接过家族遗产的、说话太快走路太急的三十一岁年轻人。

他们不是政治婚姻。他们相爱过。

八年。八年的婚姻——不算特别快乐但也不算不快乐的那种。阿奇博尔德总在书房工作到很晚，科迪莉亚管着庄园的大小事务——宴请、修缮、和仆人之间的那套微妙的阶级平衡。他们生了一个孩子——叫詹姆斯——是个男孩。詹姆斯的摇篮放在蒙特罗斯庄园二楼最靠东的房间，每天早上第一缕阳光会从他窗户照进来。

**詹姆斯八岁那年死了。**

不是病。是意外——蒙特罗斯家在海滨有一栋夏天用的度假屋。那天詹姆斯和保姆在海滩上，风浪突然变大。没有人预判到。没有人需要负责。

从那以后，他们住的那栋灰色的庄园就空了——不是人走了，是人的里面走掉了。阿奇博尔德用更长的工作时间填充每一个不需要面对妻子的时刻。科迪莉亚用沉默填充每一间屋子。他们两个在各自的悲伤里，像两艘下了锚但不在同一个港口的小船。

**八年前，科迪莉亚发现阿奇博尔德开始花钱。** 不是贵族的那种花钱——不是赛马、不是酒窖、不是旅行。他的钱在流往一些她不认识的方向：没有发票、没有收据、账本上写着各种模糊的分类——"书籍采购"、"专项研究"、"私人顾问费"。开始她以为他有了另一个女人。后来她发现那个女人的名字是"密传"——她花了一年才找出了这个词。

**赤脉结社的音乐会是她的秘密出口。** 三年前，她在欢乐之地偶然走进一场赤脉结社的地下音乐会——那种用心跳节奏作为音乐节拍基底的演奏形式。她坐在后排听完，然后哭了一场——这是她在詹姆斯死后第一次不在这座庄园里哭。不是快乐的哭。是感到被理解的哭——那些节拍里的人不认识她，但那些心跳的旋律比任何语言都更接近她想说、但不知道要跟谁说的东西。

从那以后她一个月去一次——从不让佣人跟着，从不等结束，从来坐在最后一排靠门的位置。

**海伦娜替她做的胸针——一枚银质的、刻着铃兰花和小小的翅膀——是她身上除了珍珠手链之外唯一觉得还属于"科迪莉亚"这个人而不是"蒙特罗斯夫人"的东西。**

## 表层身份

**蒙特罗斯侯爵夫人。** 在新城西区的社交圈里——她是一个令人同情但不被拉拢的寡妇式存在（虽然丈夫还在。但其实不在了）。她在公开场合维持着足够得体的社交表现——参加必要的宴会、点头、微笑、在别人的谈话里填补那些不需要内容的空白。

## 真实欲望

**在死之前，用一个早上——不需要长——做一个早上"科迪莉亚"。不是蒙特罗斯夫人。不是詹姆斯的母亲。不是阿奇博尔德的妻子。**

她不要飞升。她甚至不知道飞升是什么。她想要的是：以"科迪莉亚"的名义——一个二十六岁还没有结婚的手工商人女儿——做一件她一直想做但从没允许自己做的事。她还不确定那件事是什么。也许是赤脉结社音乐会上的前排座位。也许是学一样她丈夫说"夫人不必懂"的东西。也许是离开新城——独自——在某个不冷的地方租一间小屋子。那间屋子只有她和阳光。

她想知道的三个问题：

1. **阿奇博尔德花了家里的钱在干什么？** — 她不是要追责。她是要理解——那个在她儿子死后开始和某种东西"对话"的男人，到底在找什么？如果他在找"答案"——答案能够被找到吗？如果找到了——那个人还是她的丈夫吗？
2. **她发现的那个木盒子代表什么？** — 那个干掉的动物心脏、那张写满符号的纸。她说不清这些让她想起了什么——直到最近的一次音乐会上，她听到一首曲子里有一个三段重复的节奏，和那个纸上的符号数量一样。那不只是纸上的答案。那可能是一道她丈夫在问的问题——用了她不认识的语言。
3. **詹姆斯会原谅她吗？** — 八年前的海滩上，是她提议那天的下午去海边散步。她提议的。保姆只是执行了她的建议。没有人怪她——阿奇博尔德从来没说过半句责备的话，但阿奇博尔德从那以后也不怎么和她说话了。她不确定詹姆斯的死是不是她的错。她不确定答案重要不重要。

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 冷寂的优雅——社交上的得体但不深入。说话温柔，但眼睛里没有焦点。多数时候像一个还在运转但已经没人在住的屋子。 |
| **真实一面** | 一个被悲痛冻住、但最近在慢慢融化的女人。三年前那场音乐会是裂口——从裂口里开始透进了她以为这辈子已经不会有的东西：好奇心、对音乐的真实反应、一个不是"侯爵夫人"的自己。 |
| **弱点** | ① 詹姆斯——任何关于她儿子的善意或恶意的提及都会让她在短时间内情绪失控。② 她丈夫的账本——如果有人能找到她藏起的财务记录，她会被迫回应。③ 赤脉结社——她不是成员，但她的存在如果被公开，可能会被蒙特罗斯的政敌用作攻击材料。 |
| **道德立场** | 她想保护阿奇博尔德的名誉——不是因为爱，是因为"蒙特罗斯"这个姓氏是她的责任。但如果她发现阿奇博尔德在做会伤害人的事——她的良知会和她对这个姓氏的忠诚打架。她不确定谁会赢。 |
| **可被利用的方式** | ① 以赤脉结社为切口——她对这个群体的联系是不能被公开的信息。② 让她看到阿奇博尔德在外面还有别的事——不止是花钱，是真正的行动。她不是在追查——她是在准备离开。如果被推一下，她会选。③ 那枚海伦娜做的胸针——如果有人认识它上面的符号（灯或铸相关的），那个人的出现会让科迪莉亚意识到：她已经在不自知中成为了一张更大拼图中的一块。 |

## 触发事件

**事件一：丈夫的秘密账目**

科迪莉亚在某次社交场合或通过海伦娜的引荐认识了玩家。在建立了基本的信任后，她将一份手抄的清单递给玩家——上面是阿奇博尔德近三年的"异常支出"摘要。

"你能帮我查一下这些是什么吗？我不需要证明他做错了什么。我只是需要知道。"她看着玩家的眼睛——然后补了一句："如果你查到了他不应该做的事……先告诉我。"

追踪这些账目会引导玩家发现阿奇博尔德的密传收藏——包括他已经从某些独行卖家手中购买的低阶密传物品、以及他正在为洛克哈特手稿的拍卖准备的"出价"。如果玩家告知科迪莉亚她丈夫的密传活动——她的反应取决于告知的方式：情报可以让她原谅（一种"我终于知道他在做什么了"的释然），也可以让她彻底离开（如果内容证明他正在做危险的事）。

**事件二：木盒子里的恐惧**

科迪莉亚在家中的一次私下会面中向玩家展示了她发现的那个木盒——就在阿奇博尔德书房锁坏的抽屉里。她不敢碰里面的东西，但她需要有人解释它们是什么。

如果玩家具有密传知识（或找到拥有密传知识的人帮忙鉴定），会发现：
- 烧焦的蜡烛头：蛾之仪式中使用的"注意转移蜡烛"——点燃它的人会在仪式期间不被不知情者注意到。
- 画着符号的纸：一份低阶的"铸之锻纹"——用于加固某个物品的物理结构。这是一份被剪掉一半的残片——另一半在某个人手里。
- 干涸的动物心脏：心之道仪式中的"跳动物件"——用动物的心跳作为替代品来完成某个需要"活的心脏"但人可以活着的仪式。

这些物品拼在一起——不是一种性相，是三种：蛾（隐藏）、铸（加固）、心（跳动）。阿奇博尔德不是在专攻一条路。他是在尝试把所有路连接在一起——就像洛克哈特的手稿做的那样。

科迪莉亚的恐惧不是发现丈夫加入了一个"邪教"——而是发现她的丈夫在找的这种东西，她觉得她已经见到了。见到过他书房那扇关紧的门后面不是另一个女人——而是一种比他更老的东西，在等着把他吞掉。

如果玩家选择不说出鉴定结果——或选择说一部分保留一部分——将直接影响后续科迪莉亚是否将阿奇博尔德的其他秘密信息提供给玩家。
