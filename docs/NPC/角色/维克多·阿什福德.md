## 背景：幕布后面的另一张脸

**维克多·阿什福德在十六岁那年发现了自己的天赋——不是在舞台上，是在生活中。**

他出生在牛门的一间廉价公寓里。父亲是个三流裁缝，母亲在贵族宅邸洗衣服。他在十岁时被送去一个巡游剧团当学徒——不是因为家里发现了他的天赋，而是因为少一个人吃饭。

十五岁时，他已经能在一个晚上记住三页对白、模仿四种口音、在舞台上哭出真正的眼泪。但他最重要的发现是：**台下的人比台上的人更好"演"**——一个真诚的语气、一个恰当的停顿、一次"不经意"的眼神接触，你就能让任何人对你产生好感。

十七岁那年，他在一次街头表演后被一个穿深色羊毛外套的中年男人拦住了。男人递给他一张纸条，上面只有一个地址。他去了。那个男人是奥尔德雷克·索恩。

**索恩没有教他"蛾之道"。索恩教他的是：真正的变化不是换一张脸，而是让人分不清哪张脸是真实的。**

随后的十一年里，维克多成了蛾之教会最出色的外勤特工。他擅长渗透、观察、从目标人物的社交圈里获取信息——不是通过胁迫或偷窃，而是通过被人喜欢。他在圣路德公园入口修了自己的露天剧场，一方面是舞台，另一方面——公园入口是爱丁堡各色人等的汇聚地。每一个来看戏的人，都在不经意间给他提供了信息。

**但十一年的"隐身"让他开始厌倦。**

蛾之教会的传统教义说：蛾的道路是隐秘的。变化在暗中发生。飞升——如果你能完成——应该不留下任何可以被世人所知的痕迹。

维克多不同意。他认为这种"必须隐身"的信条是一种胆怯。为什么所有的变化都要藏在暗处？蛾的飞升——如果它真的是变化——为什么不能是**华美的、可见的、让所有人记住的**？太阳下也飞着很多蝴蝶。不都是活在灯罩里的虫蛾。

**索恩听了他的这些言论——什么都没说。维克多知道。"什么都没说"在索恩的词典里，意味着"你有麻烦了"。**

他是对的。但维克多不怕。他正年轻，正在他最好的年纪，而且他比任何人都了解如何在"被人看到"的同时不被抓住。他只需要找到一条新的路——一条不在黑暗中飞升的路。

洛克哈特的手稿。跨性相飞升。如果灯+蛾可以、如果心+蛾可以——也许蛾的飞升，本来就需要另一条路径作为"亮面"。

## 表层身份

**演员兼剧作家。回声剧场的创立者。** 他的剧团在圣路德公园入口的露天舞台定期演出——改编自苏格兰民谣的短剧、讽刺贵族的喜剧、以及某些"太过聪明所以让观众不太确定自己看懂了什么"的实验性作品。

他在圈内小有名气。贵族偶尔包场，平民在周末带着凳子来看。他的谢幕总是一样：深深一躬，抬起头时——先看左边的观众，再看右边的。他在数观众席上有几个新面孔。不是出于虚荣——是出于索恩教他的习惯。

## 真实欲望

**找到一种不需要"隐身"的蛾之飞升。成为——不只是求道者中的传说，而是被这座城邦记住的人。**

他相信蛾的本质不是"躲藏"而是"变化"，而变化本身是可以被看见的——只要你变化得足够快、足够彻底、足够华美，你的"隐身"就是你的光芒。

他想知道的三个问题：

1. **蛾的飞升是否天然需要另一性相来平衡？** — 十年前那名蛾之飞升者成功了，但维克多研究了所有能找到的记录——包括索恩不让他看的那些——发现那个人的飞升仪式中，可能混入了不属于蛾的元素。蛾本身不足以飞升吗？
2. **洛克哈特的手稿中有没有蛾相关的跨性相路径？** — 他是冬之道路的修行者，但他研究了全部六种性相。如果他发现了某种"通用法则"——一种可以被任何性相借用的飞升框架——那维克多就能用它来完成自己的"可见的飞升"。
3. **索恩会阻止他吗？** — 师徒一场，十一年的情谊。但索恩是守旧派——他相信蛾的力量在于不被人知。如果维克多真的找到了那条"可见的飞升之路"，索恩会在最后一步拦下他吗？

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 迷人、健谈、让人如沐春风。你会觉得他是你见过的最有趣的人——直到你离开，然后你发现他什么都没告诉你。 |
| **真实一面** | 不安、野心勃勃、对自己选择的道路有深层的怀疑。他在舞台上拥有绝对的自信——但在索恩面前，他仍然是那个十七岁时接过纸条的街头少年。他对"被看见"的渴望，可能是因为他从未真正感觉自己"被看见"过——父亲把他送去剧团是为了省饭钱，母亲在他十五岁那年去世时他没赶上见她最后一面。 |
| **弱点** | ① 无法拒绝一个"好故事"——如果有人以一种戏剧化的方式向他求助，他会先帮忙再想后果。② 他对索恩的感情是真实的——如果有人利用索恩来威胁他，他会失去冷静。③ 他喝酒——不多，但每次演完《飞蛾之夜》后都会喝到凌晨。在那个时间段里，他不是在"表演"。 |
| **道德立场** | 模糊。他认为真诚和欺骗不是对立的——善意地说谎和真诚地表演是一样的。他在表演中投入的真实情感比大多数人在日常生活中投入的还多。他不觉得自己在"骗人"——他在"表演自己"。 |
| **可被利用的方式** | ① 关于《飞蛾之夜》剧本的建议——他会认真听，因为他自己不满意 ② 告诉他索恩在监视他（这可能会引发师徒决裂）③ 以"观众"的身份观察他足够久——他会忍不住想知道你"看懂"了什么 |

```
  如果剧本本身就是一场仪式的"剧本"——那演员和观众是什么角色？

支线线索B（师徒决裂）：
  索恩知道维克多在找什么。两人之间的张力在增加——
  尤其是在拍卖即将到来的情况下。索恩可能派人监视维克多；
  维克多可能在通过玩家反监视索恩。关键节点：谁先动？
  如果维克多先拿到手稿——索恩会怎么做？

支线线索C（莉莉的记忆）：
  莉莉·斯托克不是求道者，但维克多注意到她在脏鸭与壶端酒时
  能记住每一个客人的细节。他在一次和索恩的报告中提到了她——
  用的是"潜在资源"这个词。他不知道这句话可能意味着什么。
  如果索恩把她列为"可征用的线人"——维克多会不会后悔？
```
