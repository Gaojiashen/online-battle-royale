---

## 背景：被刃养大的孤儿

**罗文不记得父母是谁。十六岁之前的人生，她也不怎么提。**

她只说过一件事：她第一次摸到刀的时候——是在牛门后巷的一个打架现场，地上有一把被人踢开的短刀。她弯腰捡起来，刀柄的温度和她的手刚好吻合。像有人在刀柄上提前刻好了她掌心的形状。

持刀的人倒在地上，血从嘴角流出来，看着她。说了一句话：「你捡刀的动作，不像第一次。」

那人是个断刃修会的招募人。三天后，罗文进了修会。

**断刃修会不养闲人。尤其不养没来历的孤儿。** 她从最底层的杂活干起——替正式成员擦刀、清理训练场地的血、凌晨四点起来熬煮用来浸泡刀刃的特殊油膏。但她有一个任何人都无法忽视的天赋：她打架的时候不思考。不是莽撞——是她的身体在动，而大脑是旁观者。马库斯后来形容她是「天生的刃——刃之道找上她了，不是她选了刃。」

三年间，她从杂活升到见习打手，再到能在训练场上和正式成员持平。修会里有些人开始说她可能是这一代最年轻的飞升候选人。

**但她二十一岁那年离开了。**

官方说法是「拒绝服从层级」。真正的原因更简单：修会要她杀一个她不愿杀的人。一个拖欠保护费的酒馆老板——六十多岁的老头，腿瘸了，养着三个孙辈。修会说这是「刃的试炼」，要她证明忠诚。

罗文去了酒馆。坐了一晚。喝了两杯。付了钱。第二天早上回去告诉修会：「做了。」

修会发现老头还活着。罗文说：「在我心里他死了。」然后交还了修会的徽记。

**她没有完全切断联系。** 修会里几个和她一起从底层爬上来的旧识还在。马库斯——那个编外的格斗教练——公开说她是「我见过的最好的苗子」和「最蠢的决定」。他继续训练她，不收钱。说他就是想看「这把刀能走多远」。

海伦娜为她打了两把弯刃——照着她的手形，改了三次才交给她。收的材料费，工钱没要。说：「我父亲说过，好刀应该打给会用的人。」

**她现在是一个自由打手。** 不加入任何组织。不效忠任何人。接的活要么是擂台赛，要么是格里格安排的"伴随保护"——在敏感交易的暗处站着，以防万一。

她真正想要的不是钱。她想要一场能让她看清楚刃之道尽头的战斗。

---

## 表层身份

**自由打手兼地下保镖。** 牛门的人叫她"刀女"，但很少有人知道她真名。她在「屠夫女儿」酒馆有一个固定角落——后排靠墙，面朝门。如果你需要找一个能打的人，去那里，点一杯她桌上放着的波本酒，把杯子推到她面前。

不接杀人的活。这不是什么道德底线——她杀过人——而是她觉得那是刃之道最无聊的用法。

---

## 真实欲望

**在战斗中找到刃的"切口"——面对一个真正的飞升者，让刀告诉自己：刃的尽头到底是什么。**

她读过的刃之密传里说：刃的力量来自「切割」——每一次挥刀都在切割世界与世界的缝隙。她想在战斗中亲眼看一次：如果对手是已经穿过那个缝隙的人，她的刀能不能切到对方。

她想知道的三个问题：

1. **飞升者能在战斗中被打败吗？** — 不是杀死，而是让一个已经超越了人类范畴的存在在她的刀下被迫后退。那意味着刃之道真正的力量不在于结果，而在于对抗本身。
2. **马库斯找到的那本手记的作者是谁？** — 她看过马库斯在训练间隙翻那本破册子。她问他为什么不给她看，他说"你还没到读它的时候"。她想知道什么时候才到。
3. **修会当年为什么选中她？** — 那个在巷子里流血的人，真的是碰巧被她在那里捡到的吗？还是说从她出生以前，就已经有人在安排这场相遇？

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 冷淡、少话、不主动惹事但也不躲事。像一把收在鞘里的刀——危险，但不会无故出鞘。 |
| **真实一面** | 比外表想得多。她离开修会不是因为叛逆——而是因为她发现她对"忠诚"的理解和修会不一样。她忠诚的对象是刃之道本身，而不是修会的层级命令。这一点让她在夜晚偶尔辗转——她走上这条路的时候只有十六岁，没有人教过她除了服从之外的东西。马库斯教她的第一件事是："你不是武器。武器不需要做决定。你需要。" |
| **弱点** | ① 太年轻——她不怕死，但还不太懂活。长远的策略不是她的优势。② 对"强大的对手"缺乏抵抗力——如果有人对她说"我能让你看到刃的尽头"，她会跟着那个人走很远才停下来问"你是谁"。③ 两只腕骨的旧伤让她在连续长时间用刀后会抖——但她从来不在人前停手。 |
| **道德立场** | 不打不能还手的人。不杀不想参与战斗的人。她认的规则不是法律——是"擂台上的公平"。擂台之外，世界不欠她公平，她也不欠世界。 |
| **可被利用的方式** | ① 一个声称能让她和飞升者交手的机会 ② 有关于断刃修会当年招募她的内幕 ③ 马库斯受伤或被威胁的消息 ④ 一把足够好的刀——她看到好刀会不由自主地停下来 |

## 对应剧情线的可能方向

```
主线线索：
  在战斗中抵达刃之道尽头——飞升的"切口"是什么？

支线线索A（断刃修会的招募真相）：
  那个在巷子里"恰好"被她捡到刀的人——他真的是碰巧在那里吗？
  如果修会在她出生前就已经在"培养"她成为刃之人——那她的父母是谁？
  她的父母可能还活着，可能在修会内部，可能在修会的对立面。

支线线索B（马库斯的秘密）：
  马库斯不让她看那本手记的真正原因——不是因为她不够强。
  而是因为手记的作者可能是她的母系血脉。
  如果是——那马库斯找的那个人，罗文可能比任何人都更接近。

支线线索C（与海伦娜的双刃）：
  海伦娜为她打的两把弯刃——上面刻了什么？海伦娜说她"照着图纸打的"。
  那张图纸不是海伦娜自己的。是谁委托她打的？
  可能指向一个知道罗文存在、但从未现身的人。

支线线索D（伦威克的名单）：
  伦威克的监视名单上有她——为什么？
  是因为她是断刃修会的叛逃者，还是因为她身上有某种比"前成员"更危险的东西？
  如果他的人真的找上她——她会反抗，还是会和他聊一聊？
```
