## 背景：普通得不能再普通的一天

**莉莉是格拉斯市场「脏鸭与壶」酒馆的女侍。从十四岁干到现在，五年了。**

她爹在她九岁那年跟着一条商船出了海，再没回来——可能死了，可能跑了，没人知道。她妈在洗衣房给人洗衣服，一双泡烂了的手养她到十四岁，终于累垮了腰。莉莉于是辍了学，开始做工。

**她的人生没有什么宏大的叙事。** 端酒、擦桌、收钱、骂那些手脚不干净的酒鬼、偶尔替老板娘去牛门进货、下工后绕路给妈抓药。这就是全部。

但她有一项自己都没意识到的天赋：**她记得每一个人。**

不是那种"我记得你点过什么"的记得——而是更深的。她能记住一个人的步态、说话的尾音、喝酒时惯用左手还是右手。她不知道这有什么用，她只是……记得。像记住一首听过的歌一样自然。

她不知道的是，这项天赋来自她从未见过面的父亲——那个出海未归的男人——血管里流着的一丝远祖之血。不是神秘的血脉，不是什么被选中的宿命。只是隔代遗传的一点敏锐感知。在另一个时代，她的祖先靠着这种感知在荒野中活下来。在她身上，它变成了一种对"人"的记忆。

她不是求道者。她不知道自己工作的酒馆里哪些客人不是普通人。她不知道那些"说话文绉绉、喝酒很慢、小费给得大方"的客人为什么总是选角落的座位。她只是觉得——他们是好客人。有礼貌，不闹事。

## 表层身份

**酒馆女侍。** 格拉斯市场上最便宜的麦芽酒——「脏鸭与壶」——的端酒人。老板娘玛格丽特对她像对半个女儿，常唠叨她"笑得太大声，不像个淑女"。熟客们叫她"小红"（因为她的头发），偶尔有人喝多了动手动脚，不等老板娘出手，旁边的老客人就会把那只手按回桌上。

她住在灰衣修士墓园后面那条巷子的最深处——一间租金最便宜的房间，窗户对着后巷，看不到阳光。窗台上养着一盆半死不活的迷迭香，她总忘记浇水。

---

## 真实欲望

**很简单：活着，让妈过得好一点。**

- 她想过攒钱给妈看一个真正的医师（不是草药铺那种）
- 想过买一件冬天不会透风的斗篷
- 想过有一天能坐在自己端酒的位置上，喝一杯不用自己擦的杯子装的酒
- **她没有想过飞升。她不知道飞升是什么。**

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 开朗、话多、笑声响亮。脏鸭与壶的气氛制造者之一。 |
| **真实一面** | 比看起来敏感得多。她记住很多人，但很少有人真正记住她。她有时候会想——如果有一天她消失了，除了她妈和老板娘，会不会有人注意到。 |
| **优点** | 过人的记忆力（尤其是对人的记忆）、讨喜的性格、在格拉斯市场的扎实人缘 |
| **弱点** | 完全不具备自保意识——她不知道她每天接触的人中有多少"不该接触的人"。她以为世界就是她看到的样子：有好人、有坏人、好人比坏人多。 |
| **道德立场** | 朴实的是非观。不偷不抢，照顾妈妈，对善意的人友善。如果有人欺负她在意的人——她会用杯子砸对方的头。她干过一次，还会再干。 |

## 对应剧情线的可能方向

```
这条线不是"玩家从莉莉这里获取资源"——这条线是"玩家看着她走进去"。

玩家可能在脏鸭与壶和她成为朋友。她记得你的名字，记得你爱喝什么。
你可能会在某个深夜帮她挡下一个纠缠的醉汉，她会在你下次来的时候偷偷往你的杯子里多倒半指。
你可能会路过她的小巷时看到她窗台上那盆迷迭香，随口说了一句"该浇水了"，她下次见到你会笑着说"浇了"。

然后有一天——在玩家某个行动的世界事件通告中，或者在你路过格拉斯市场时——

「格拉斯市场今晨发生一起事故。脏鸭与壶的一名女侍在收工回家途中……」
「……据目击者称，她似乎是在错误的时间经过了错误的地方。」
「……遗体已移交银棺修会的殡仪馆。」

官方结论：夜间抢劫，遭遇反抗，意外致死。

真正的原因：她在那天晚上的酒馆里，记住了不该记住的人。

没有人为她追查真相。除非你为她追查。

---

这条剧情线的走向，取决于玩家是否注意到她的缺席，以及是否选择去做点什么。

如果玩家选择为她追查：
  - 可能发现她是被某个与神秘有关的人灭口的
  - 可能找到她死前留下的某个"她自己也说不清为什么觉得重要"的小物件
  - 可能——只是可能——发现她的那条远祖血脉让她在最后一刻"记住"了凶手的某种特质

如果没有人追查：
  - 格拉斯市场的熟客们会惋惜几天，然后新的女侍会接替她的位置
  - 她的母亲会被送进救济院
  - 窗台上的迷迭香会干死
  - 除了你——可能——没有人会记得莉莉·斯托克
```
