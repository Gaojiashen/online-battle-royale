## 背景：双手接住从黑暗中递出来的东西

**莫伊拉二十三岁成为助产士。在那个年代——产房里一半的婴儿活不过周岁——她接生了三百多个孩子，活下来两百多个。**

她不觉得这有什么特别的。生命本来就这样：一部分留下，一部分离开。她的工作不是让所有人活——而是**让每一个到来的生命被好好地接住**，不管它停留多久。

三十二岁那年，她在一个风雪夜接生了一个死婴。脐带绕颈三圈。她把那个冰冷的小身体洗干净、裹好、放进母亲怀里——然后走出产房，在走廊里坐了很久。不是因为悲伤。是因为她在那间房里感觉到了一种她从未接触过的东西——不是死亡，而是一种**比死亡更古老的、持续不断的搏动**。

第二天，一个赤脉结社的成员找到了她。他说："你感觉到了，不是吗？那不是幻觉。那是心——生命背后的生命。"

她花了三年研究心的基础密传。又花了五年晋升为赤脉结社的主母——不是通过权术，而是通过每一次接生、每一次演奏、每一次在别人崩溃的时候把手放在他们肩膀上。心的力量不是夺取，是**给予和循环**。

**她的音乐学院是一层掩护，但也是一座真正的学校。** 她教穷人家的孩子弹琴、唱歌、读谱——不收钱。赤脉结社的集会在地下室进行，而楼上永远有琴声：巴赫的前奏曲、苏格兰民谣、学生们自己写的小调。两种声音互不干扰。或者说——它们本来就是同一种声音。

七年前——唐纳66年——她在格拉斯市场遇到了一个女人。太瘦，灰白的头发紧束在脑后，随身挎着一只旧皮包。女人说她想找一个能睡几天的地方。莫伊拉没有问原因，给了她音乐学院楼上的杂物间。

第三天，莫伊拉在走廊里经过那间杂物间时——停住了。门缝里渗出来的不是冷气，而是**冬**。不是冬天的冬——是冬之性相的那种冬。一种极度的寂静，像是所有声音都被吸进去了一样。

她敲了门。女人开了门。莫伊拉看着她的眼睛，沉默了很久，然后说："我不能让你留在这里。不是因为你是威胁——因为心的力量是放大生命，而冬的力量是保存死亡。这两种力量在同一个屋檐下……你会不舒服，我的学生也会。"

西尔维娅没有争辩。她当晚就离开了。

**莫伊拉至今后悔。** 不是因为她的判断错了——而是因为她没有在那扇门关上之前说一句"如果以后有需要——"。

## 表层身份

**烛芯音乐学院的院长。** 她的学生大多是穷人家的孩子——工人的儿女、孤儿院送来的、街头卖唱的流浪儿。她教他们唱。真正的、从胸腔里发出的声音。

学院的声誉在小范围内很好——有些贵族偶尔会请她的学生去私人聚会上唱歌。她收的演出费比市场价低一半，条件是：**不许碰她的学生。** 口气温和，但眼神像看产床上的出血——不惊慌，但绝不退让。

## 真实欲望

**见证一种"不离开"的飞升。不逃离世界，不抛弃生命——而是带着所有你在乎的人一起往前走。**

她不认为飞升是生命或死亡的"抵消"。她认为飞升是**共鸣**——当你的心跳和这个世界的心跳完全同步的那一瞬间，你就会成为那个更大的节奏的一部分。

她想知道的三个问题：

1. **心之道能否复活死人？** — 蒙特罗斯勋爵资助了她三年，从未明说他的目的，但她知道。他死去的儿子一直在那袋金币的重量里。心可以维持生命、放大生命——但能把已经熄灭的再点燃吗？她不知道。她试过。她没有告诉蒙特罗斯结果。
2. **洛克哈特的跨性相飞升——心能和什么结合？** — 如果冬是保存死亡，蛾是变化，灯是揭示——心是生命。跨性相的组合中，心能和什么配对？如果洛克哈特的手稿里有答案——她必须读。
3. **西尔维娅·韦恩现在在哪里？** — 她听说有人在灰衣修士墓园见到过一个"太瘦的守墓女工"。她不敢去确认。但她反复梦到那扇关上的门。

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 温暖、从容、像所有人的母亲——但不过度亲近。她给的是安全感，不是溺爱。 |
| **真实一面** | 比外表复杂得多。她见过太多生命的来去，形成了自己的一套生命哲学——"不强行挽留，但绝不轻易放弃。"她在西尔维娅那件事上的后悔一直在暗中影响她的判断——她现在更倾向于"先问，再决定。" |
| **弱点** | ① 她无法拒绝一个"用心呼救"的人——如果有人在极度绝望中向她求助，她会先伸出手再考虑后果。② 蒙特罗斯的关系——她知道他在利用她，但她无法拒绝，因为他的悲伤是真实的。③ 她对"心跳停止"有生理性的应激反应——看到人倒下时会不自觉地冲过去，不顾自身安全。 |
| **道德立场** | 生命至上。但她对"生命"的定义比普通人宽——一个求道者的飞升尝试如果失败了，那仍然是一条生命的终结。她不区分"值得"和"不值得"的生命。 |
| **可被利用的方式** | ① 利用她的后悔——以西尔维娅的信息作为交换条件，可以让她做很多事 ② 利用蒙特罗斯的关系——如果告诉他"莫伊拉在骗你"，他会停掉资助 ③ 用一个濒死的人（或假装濒死的人）来触发她的本能反应 |

  心之飞升能否跨越生命的终极边界——死亡？

  她收了他的钱，但她知道他想要的是她用不了的东西——

  蒙特罗斯会不惜一切代价去竞拍。届时莫伊拉必须决定：帮他，还是阻止他。

  西尔维娅·韦恩在墓园。莫伊拉知道，或至少猜到。

  那些碎片中可能包含了莫伊拉苦寻多年而不得的心之飞升线索。

支线线索C（合唱团与赤脉结社）：
  音乐学院的学生中，有几个人已经展现出了心的天赋——
  不需要密传，没有仪式，只是天生地对"心跳"敏感。
  赤脉结社内部有人主张"招募"——把她们拉进地下。
  莫伊拉反对。但反对能持续多久？
