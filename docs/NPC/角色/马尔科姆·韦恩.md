## 背景：渡口上的守盒人

**马尔科姆·韦恩这辈子没离开过利斯港。** 他甚至没怎么进过老城——城里那些被煤油灯熏黄的巷弄、教堂投下的影子、地下不知通到哪里的走廊，他只是听那些坐他的船的人提过。他是港口人。港口人的世界是水、是沙、是缆绳、是对岸。

他有一艘小渡船——从父亲手里接过来的。船底是榆木的，船缘是松木的，每年春天要上一次桐油。这艘船是他的全部。他在利斯渡口二十年：从二十岁帮父亲划桨，到三十岁接手，到现在四十岁——他运过的乘客加在一起足够填满半个利斯港。

**但十年前——唐纳63年的一个深夜——他做了这辈子第一件违反常规的事。**

他妹妹西尔维娅——十年没回家也不写家的妹妹，那个他看着她从小进墓园后跑腿的、他给她买了她第一次银笔的、她走了以后他在母亲的葬礼上独自站在角落的妹妹——砸开了他在渡口的小屋门。

她给他一个封了蜡的盒子。牛皮纸包的，外面的线打了七圈。她的手指在发抖——不是冷，是恐惧。他从来没看过她这样。

「帮我留着。不要打开。不要告诉任何人我还在爱丁堡。」

他问：「里面是什么？」

她沉默了一下——他记得那个沉默的长度，大概和他一个潮汐的节奏差不多。「一个人的笔记。如果它还在我身上——我会被找到。」

马尔科姆接过那个盒子。他没有再问她任何关于那东西的内容。他问的是：「你还能回来吗？」

她没有回答。她只是用力抱了他一下——那个拥抱的力道和他们在父亲葬礼上分吃的那个面包一样实沉——然后消失在利斯路上的薄雾中了。

**从那以后他再也没见过她。**

他没有打开盒子。不是因为不想——十年里至少有五百次他把盒子放在桌上，坐到椅子对着它看了半个晚上。但是西尔维娅说过不要打开。她说不要打开——不是因为里面的东西不重要，而是因为它重要到她知道打开了就会牵扯他。他信她。他认识她。她还是那个小时候为了救一只掉进井里的猫、把绳子绑在自己腰上、就让自己往井下爬的小女孩。她从来不让人做她自己不做的。

**于是他等着。十年。**

他不知道盒子里的东西足够让三个势力在十年的时间里为它杀了至少六个人。他不知道他妹妹在城里逃了十年的命。他不知道这座城市有一个世界在他头顶上——在地面上那些他不常去的巷子和地下全黑的拱廊之间——用一种他从未听说过的语言在运转。

但他知道那个盒子的蜡封一天没破，她就还在。她不需要来找他。这个盒子就是她还在的信号。

**最近，有两个人在夜里找到了他的渡口小酒馆。** 一个是退休船长伊莱亚斯·斯托姆（莉莉的父亲前同事）——老船长最近在自己也说不清为什么的状态下搬到利斯路附近，说要"重新吹吹海风"。他经常坐到马尔科姆旁边，开始说一些细碎的往事，说到一半会停下来——看着自己的杯子，像在杯子里看到了某个他决定不说的名字。

另一个是邓肯·麦克莱恩——城市的点灯人。他会偶尔出现在利斯路附近的那家酒馆，坐在角落里，一个人喝一杯，然后离开。马尔科姆和他说过三两次话——都是关于天气、码头的情况、过了四十岁以后天亮的时候膝盖是不是会有感觉。他不认得邓肯的点灯棒——他只知道这个人和他一样不是本地那种来利斯路混日子的人。

**马尔科姆不知道他们两个讲的各半句话如果拼在一起，可能会凑出他妹妹此刻正在躲的人——和她一直在找的东西。**

但他心里某个地方在醒过来。某种他二十年渡船生涯从来没有唤醒过的直觉——那个盒子不是一个物件。那个盒子的分量在他手里正在慢慢改变。

## 表层身份

**利斯港渡船人。** 渡口最稳定的那个划船人——二十年没出过错。他每天划八趟，多的时候九趟。船费是一个硬币：半个金币。他收得不多，收得规律，所以在利斯路这一带没有人对他多看一眼——他就是那个渡船人。

他给船取了个名字叫「韦恩的桨」。没有漆过。没有旗帜。但它走得直。即便在有风的天气。

---

## 真实欲望

**等到西尔维娅回来拿这个盒子。然后让她解释一切。**

他不觉得妹妹死了。他只是不知道她为什么走了这么久。他没有恨——他给她的信任是一只装着牛纸皮盒子的蜡封，还没打开过的。

他想知道的三个问题：

1. **她还活着吗？** — 十年。如果不是这个盒子还在——他会去城里找。但他从来没去过。他知道如果他去了，她会知道。她想让他安全。他给了她十年不用担心的安全。但他已经快到极限了。
2. **盒子里的东西为什么值一个人的命？** — 他不知道。但他听到了那些他不应该知道的人的名字在某些晚上酒馆另一个桌子上被提到——用那种"你不会告诉别人你想找这本书"的语气。他只是渡船人。别人在他面前不设防。他知道的比他应该知道的多了那么一点点。
3. **伊莱亚斯为什么不把话说完？** — 那个老船长在喝了三杯以后会说——"我有个同事，一条船上的……他闺女跟你们家年纪差不多。"然后不说话了。他不说了。那个"同事"是谁？那个"闺女"是谁？伊莱亚斯为什么总在说到这里的时候把酒干了然后付账走人？

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 沉默、可靠、不复杂。一个你愿意把钱包掉在他船上然后第二天原封不动拿回来的好人。他不是蠢——他是相信世界的规则很简单：渡人过水，收钱，不说别人的话，不偷别人的东西。 |
| **真实一面** | 比外表想得深得多。十年替妹妹保管一个未知盒子的行为本身——如果他是一个不经过把心剖开反复看的人，他不可能做到。他每天晚上在渡口小屋里反复下着同一个判断：今天要不要打开盒子？每天他都做出了同样的决定。那不是软弱，是一种对于承诺和血脉的绝对忠诚——那种忠诚不在书里，在他骨头里。 |
| **弱点** | ① 西尔维娅——只要提到他妹妹的任何事，他会在瞬间变得不理智。他不谈判。不妥协。不给你第二次机会。② 对"盒子被盯上"的恐惧——如果有人在渡口附近打听"十年前一个女的来给渡船人过一个盒子"，他会立刻警觉。但他不知道怎样在那样的情况下保护好自己和盒子——他不是战士。③ 二十年的渡口生涯让他习惯了一种固定不变的节奏。任何需要他打破那个节奏的事——包括离开利斯——都需要比他允许自己感觉的更大的勇气。 |
| **道德立场** | 非常老派的、朴素的忠诚。对妹妹。对渡船。对还在付钱渡水的人。不对任何一个看不见的组织效忠。他不会主动伤害人，但如果有人威胁到那个盒子——或者威胁到他妹妹——他会把那艘小船停在水中央对那个人说："你游到对岸。" |
| **可被利用的方式** | ① 冒充认识西尔维娅的人——但谎话他会听出来。他听过这世界所有形式的推脱和谎言——那些人坐船时说的——他只是不说。② 让他相信盒子里的东西正在害他妹妹——他会打开它。③ 接触伊莱亚斯或邓肯，通过他们间接接触到马尔科姆——这条线更自然。 |

## 对应剧情线的可能方向

```
主线线索：
  一个普通人在十年间守护的神秘遗产——直到他决定不再等待。

支线线索A（盒子的开启）：
  那个封蜡盒子是洛克哈特跨性相理论的原始笔记。
  如果西尔维娅只拿到了冬的部分——洛克哈特把灯+蛾+刃+铸+冬+心的桥梁论证
  写在了她的那份之外。盒子里的东西包含了跨性相的过渡路径。
  开启它意味着：西尔维娅人生中最重要的秘密终于被外延。
  也意味着：马尔科姆·韦恩正式进入了爱丁堡神秘世界。

支线线索B（伊莱亚斯的半截话）：
  老船长伊莱亚斯·斯托姆——莉莉从未见过面的父亲的前同事——
  在和马尔科姆喝着第三杯的时候，说到一半的话。
  那些半截话拼凑的是：十一年前一条商船在地图外触礁的航线；
  船上有一个不是水手的人——一个"花钱买船票、全程不离开船舱"的乘客；
  事故发生后，绝大多数水手死在海里，只有伊莱亚斯和那个乘客活了下来。
  那个乘客是谁？
  如果那条航线的目的和洛克哈特的某些秘密有关——

支线线索C（邓肯的胸针与马尔科姆的盒子）：
  两个普通人——一个守护了一个盒子十年，一个还带着一枚捡到的胸针——
  没有飞升，没有性相，没有任何进入神秘世界的资格。
  但他们在同一个酒馆里喝着同一种酒，分别在两个人不同的耳朵里灌着不同的半真半假。
  如果玩家把两条线牵到一起——这两个普通人的所有物可能拼出一份不需要任何密传基础
  也能验证的结论：洛克哈特的手稿散布在这种酒馆里、这种渡船上、这种没人多看一眼的地方。
  因为他故意让它们沿着和他平行的"不属于求道者的路线"在传播。
  
  然后，那些路线正在自己交汇。

支线线索D（西尔维娅归来）：
  如果西尔维娅在十年后重新出现在利斯渡口——
  她看到的不是一个23岁的渡船少年。
  是一个花白了头发、手指更粗、话更少、但桌子还是那张桌子的四十岁男人。
  他没有多问。他把盒子从床底下拉出来——推到她面前。
  她打开它——然后两个人坐在同一个桌子上，三十年前他们一起吃面包的位置，
  现在是一个装着改变世界的东西的盒子和两个已经不年轻的人。
  
  然后他问：「你是回来打开它——还是来把它拿走？」
  她的回答决定她是否允许自己最后一次需要自己唯一的亲人。
```
