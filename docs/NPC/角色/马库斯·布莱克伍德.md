## 背景：荆棘之路

**马库斯不是天生的刃之道信徒。他是被刃选中的。**

他十六岁参军，在那个不算庞大的城邦军队里打了十二年仗——边境巡逻、贵族之间的私人械斗、偶尔镇压地下教团的骚乱。他擅长这个。擅长到上司开始把"不太好声张的活"交给他：恐吓、清场、让某个碍事的人消失。他不问为什么。他只是做。

直到某次任务——一个地下教团的据点清剿——他在一个被掀翻的祭坛下找到了一本破旧的手记。封面没有标题，翻开第一页只写着一行字：

> **「每一次挥刃，你都在切割世界。学会看那些切口。」**

他不知道为什么没有上交那本手记。他留了下来。那是他第一次接触刃之密传。

随后几年间，他陆续通过各种渠道接触了更多的密传碎片——从缴获的教团物品中、从黑市交易里、从一个被灭口的刃之道信徒的遗物里。他把它们藏在床板下面，在深夜的油灯下一页一页地抄录、烧毁。

离开军队时他二十八岁。用积攒的军饷在西弓街盘下了一间铺面，开了「黑铁俱乐部」——表面上是退役军人的格斗训练场，实际上是断刃修会的一个据点和招募站。

他没有正式加入过断刃修会。他是编外人员——为他们做一些"需要有人弄脏手"的事，换取的报酬有时是金币，有时是下一本密传。

**他想要的不是权力，不是财富。他想要知道「刃」走到尽头是什么样子。**

## 表层身份

**黑铁俱乐部老板。** 西弓街上一个退役老兵开的格斗馆，教人打拳、防身、偶尔接保镖或收债的活。街坊邻居对他的评价是"不好惹，但还算公道"。交保护费的时候从不讨价还价，这是他和地下势力之间相安无事的原因。

俱乐部后院有一间上锁的房间——里面放着那本手记、一些零散密传碎片、以及一套保养得极好的武器。他说那是"存货"。没人细问。

---

## 真实欲望

**抵达刃之道尽头，在战斗中找到飞升的切口。**

他相信飞升不是逃离——而是**穿过**。穿过暴力的表象抵达某种更本质的东西。

他想知道的三个问题：

1. **刃的飞升是什么样子的？** — 灯是光，蛾是变，刃呢？是一场永恒的厮杀？还是一种绝对的静止？
2. **十年前那两次飞升的人，现在在哪里？** — 他们真的"成功"了吗？还是变成了某种不再属于人的东西？
3. **那本手记的作者是谁？** — 他找了十一年，没有找到任何关于那个笔迹的线索。但那个人的行文方式告诉他：写下这句话的人，很可能已经飞升了。或者，已经死了。

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 沉默、冷淡、不好惹。钱货两清，不多废话。 |
| **真实一面** | 比外表谨慎得多。他见过的暴力太多了，知道真正的力量不在于先动手，而在于不动手也能赢。 |
| **弱点** | 对那本手记的作者有执念——任何人只要声称有关于那个笔迹的线索，都能轻易引起他的注意。另外，他的左手旧伤在高湿度天气时会抖（爱丁堡这种地方……几乎天天抖）。 |
| **道德立场** | 不主动伤害无辜者，但关他什么事的时候绝不多管闲事。他相信暴力和欠债一样——迟早要还，关键是谁先付。 |
| **可被利用的方式** | ① 有关于那本手记作者的线索 ② 有一场他无法拒绝的战斗（对手够强、够值得） ③ 有人威胁到俱乐部的安全 |

## 对应剧情线的可能方向

```
主线线索：
  刃之道路的终极形态——飞升的"切口"是什么？

支线线索A（手记作者）：
  那本手记的作者可能还活着——可能就在爱丁堡。
  笔迹分析显示：是中年人，女性，左撇子。
  更深的线索指向……银棺修会？

支线线索B（俱乐部暗流）：
  新来的那几个"出手阔绰的面孔"——他们是蛾之教会派来摸底的人。
  为什么蛾之教会突然对断刃修会的据点感兴趣？

支线线索C（十一年前的清剿任务）：
  他找到那本手记的任务现场——当年被清剿的那个教团据点。
  如果他回去那个地方……会发现有些痕迹不该还在那里。
  那个祭坛下面，可能不只是有一本手记。
```
