## 背景：风里的耳朵

**鬼火在蛾之教会的位置极其特殊——她不是间谍，不是密探，不是特工。她是信使。是这个城市地下情报网络中最不起眼、最快速、最好用的传递系统。**

没有人知道她的真名——她签在教会名册上的名字是"鬼火"，注册日期是"大约在十年以前"。教会的档案里关于她的条目只有短短五行字：现任情报总长（索恩）专属信使；经手死信箱十二个；无违纪记录；备注：可信。但不要问她任何关于她自己的问题。

**她不回答不是因为她有秘密要藏——而是因为她不认为自己的历史是"信息"。** 她在爱丁堡街头长大——不是孤儿，但差不多：母亲在牛门后巷替人洗衣服，父亲是个偶尔出现的码头工人。她十二岁开始为街头小贩跑腿——从东头送到西头，一个铜板一趟。十四岁开始跑更大的——从格拉斯市场送到布市，从布市送到高街，从不看自己送的是什么。

十九岁的某一天，她递送的一封信在半路上被人截了——两个男人把她堵在一条后巷里，要她交出信来。她没有。她咬了第一个人的手，从第二个人两腿之间钻过去，跑了两条街，把信准时送到了。信是索恩的。索恩在香料行后门等她——看着她满嘴血、膝盖擦破、信完好无损——说了一句话："以后只跑我的。"那是十二年前。

**她的工作是简单的：接收死信箱中的信息，转送到下一个死信箱或直接交给索恩。** 她每天跑八个死信箱——那些位置是分散在全城的：格拉斯市场的一颗松动的墙砖后、高街一间书店的第三个书架底层、牛门地下酒馆的吧台下方、灰衣修士墓园的某块没有名字的墓碑后面。她从不停留超过三十秒。她几乎从不和人交谈——除非交信息时需要一句切口。

**她从来不读自己送的信。** 这是索恩定的规矩——也是她自己选择的原则。她说：一个好的信使不知道自己在送什么——因为如果你不知道自己在送什么，你不能在被人抓住时说漏嘴。

但最近她开始破例。不是刻意窥探——是上一封信被雨水浸湿了，封口自己松开了。她在重新封口之前看到了纸上的字："……手稿拍卖日期确认。受邀者名单共七人。第三位是——"剩下的字花了。她把信封好，送到了。但她记住了"七人"——和那张名单被水泡烂的墨迹形状。如果她能看到完整的名单，她一定能认出那些被遮掉的名字。

**她不知道那个丢了的死信箱是被谁拿走了。** 但她在每个经过的角落都留了一个暗号——一缕布丝系在门框上、一块小石子放在窗台特定的位置。她在等那个拿走皮管的人联系她。在她看来——能发现死信箱并从里面取走东西而不被她的暗号线发现的人——值得谈一谈。

## 表层身份

**高街上的杂活跑腿人。** 如果你在高街上看到一个瘦小的灰斗篷身影以稳定的节奏穿过人群——那就是鬼火。街坊邻居觉得她是一个"给商行跑腿的勤快姑娘"。她没有固定住址。她睡在不同的地方：有时是索恩香料行的后间（索恩给她的——她一个月只用两三次），有时是地下拱顶的某个干爽角落，有时就坐在高街某间关了门的店铺门口——眯着眼等到天亮。

没有人问她叫什么名字。她也不说。

---

## 真实欲望

**找到一种不依附于任何人的活法——不是为了背叛索恩，而是为了有一天如果"不再做信使"，她还能知道自己是谁。**

她做了十二年的信使。她不知道自己在传递的东西中有多少条信息决定了某人的生死——她不想知道。但她最近开始意识到：如果有一天索恩不再需要她——或者更糟，索恩死了——她的全部身份就没了。她不追求飞升。她不追求知识。她追求的是一种更基本的东西：一个不用向任何人汇报的名字。

她想知道的三个问题：

1. **那张名单上的其他人是谁？** — 她被浸湿的信上写着"受邀者名单共七人"。她知道索恩在看——但其他六个人是谁？为什么只有七个人？如果她能凑齐名单——她可能不需要通过索恩就能知道拍卖的真相。
2. **那个拿走皮管的人想干什么？** — 她的死信箱被动了。那个人没有留下任何可以被追踪的痕迹——这本身就是一个信号：对方是专业的。但她放出去的暗号还没有被回应。她想知道：那个人是在躲她，还是在等她？
3. **索恩会让她离开吗？** — 她从来没有问过。但蛾之教会的信使通常不会超过十五年——不是在教会里退休，是被其他信使取代。十二年了。她还有几年？

## 性格与行为倾向

| 方面 | 描述 |
|:-----|:------|
| **对外姿态** | 不搭理人、不闲聊、不社交。在街上是透明的——不是隐身，而是"不值得注意"。如果有人特意找她说话——她的第一反应是评估你的步幅和手的位置：你是来找茬的还是来交信息的。 |
| **真实一面** | 厌倦了。十二年跑同一条路线，十二年不读自己送的信，十二年不知道自己在帮谁传递什么。她的疲惫是一种缓慢的累积——不在她的脸上，在她的脚上：最近她开始绕更多的弯路到达目的地。不是因为有追踪——是因为直线走太多次之后，她觉得自己的鞋子已经在地上刻出了同一条凹陷。 |
| **弱点** | ① 对"跑腿的孩子"有本能的保护欲——如果她看到一个孩子在做她小时候做的事，她会不自觉地提供帮助。② 她的左手虎口旧伤在湿冷天气会疼——爱丁堡几乎天天湿冷。疼的时候不能爬墙，不能从高处翻身——她的行动半径会缩小一半。③ 对索恩的"欠债感"——十二年前索恩给了她一份工作，让她不再饿肚子。这种感激至今未消。如果有人以"帮索恩"的名义让她做某件事——她会做，然后事后才想"这真的是帮吗？" |
| **道德立场** | 街头人的实用道德——不偷穷人、不打比自己小的人、不传递自己知道会直接杀人（不是间接——间接她管不了）的信息。她很清楚自己的职业在道德上是一团灰色的雾——她选择不看雾的形状。 |
| **可被利用的方式** | ① 告诉她"索恩需要这份信息立刻送到"——她会送，然后发现你骗了她（届时你会多一个敌人）② 利用她丢失的死信箱——提出用皮管里的内容交换她的配合 ③ 在"跑腿的孩子"被欺负时出手帮忙——她会记你一辈子 |

## 触发事件

### 事件一：丢失的死信箱

玩家在格拉斯市场或者高街上发现了一只小皮管——被塞在某个不自然的位置（墙缝太浅、台阶下太容易被发现）。打开皮管，里面是一张卷起来的纸，写着某个玩家的名字、近期的日程路线和一句标注："需要进一步确认性相等级。"

如果玩家带着皮管在高街上反复徘徊——鬼火会注意到。她不会主动接近——但会开始跟踪玩家。如果玩家将皮管放回原位——鬼火会在当晚取走，并在原位留一颗小石子：意思是"我来了。我知道你看到了。"

**触发条件**：玩家在格拉斯市场或高街附近触发了寻找隐藏物品的检定。

### 事件二：以信息换信息

鬼火在跟踪玩家几日后，选择在一个无法被其他人看到的地点现身——一条窄巷的拐角、地下拱顶的某个死路尽头、或者深夜无人的圣路德公园。她站在十步以外，说："那封信你看了。我需要知道你有没有给别人看。作为交换——我可以告诉你你名单上另一个人最近在做什么。"

如果玩家同意交易——鬼火会提供一份关于另一个NPC的近期监视报告（内容涉及该NPC的密传活动）。如果玩家拒绝——她会消失在拐角，并且接下来一周内你不会再在任何地方看到她。

**触发条件**：事件一已触发 + 玩家没有把皮管内容张扬出去。

### 事件三：信使之死？

鬼火在某次常规递送中遭遇了埋伏——不是针对她，是针对她携带的信息。有人在牛门地下酒馆附近拦截她——两个刃之道路的打手。她受了伤但逃脱了——信息被毁了（她吞掉了纸条）。索恩会知道：有人在截断他的情报线路。鬼火需要找一个安全的地方养伤——她不会去找索恩（暴露香料行），不会去找维克多（她不相信他）。如果玩家在事件二中建立了信任——她会在玩家的藏身处出现，肩上带着一道还在渗血的刀伤。

**触发条件**：事件一已触发 + 事件二已完成 + 玩家的某个敌对阵营的NPC（与索恩对立）近期的活跃度达到一定值。或者DM在关键剧情节点上判定伏击发生。
