# 环境卡（3张）

> 战斗开始前，由法官/双方协商确定环境。环境改变可用行动。

---

## 🏠 室内 · 狭窄空间

```
┌───────────────────────────┐
│                           │
│      🏠  室  内            │
│                           │
│  如：酒馆后厅、墓穴通道、   │
│      图书馆书架间、地窖     │
│                           │
│  ═══════════════════════  │
│                           │
│  受限：                   │
│  ❌ 闪避 不可用            │
│  ❌ 撤步 不可用            │
│                           │
│  增益：                   │
│  ✅ 逼近 无需消耗一步      │
│     （默认近身位）         │
│  ✅ 格挡 +1 减免           │
│                           │
│  战术含义：               │
│  无处可逃。近身肉搏。       │
│  蛾之道在此极度不利。       │
│                           │
└───────────────────────────┘
```

---

## 🌳 室外 · 开阔空间

```
┌───────────────────────────┐
│                           │
│      🌳  室  外            │
│                           │
│  如：市场广场、墓园、       │
│      利斯港码头、街道       │
│                           │
│  ═══════════════════════  │
│                           │
│  标准：                   │
│  ✅ 所有行动可用           │
│  ✅ 默认中距离             │
│                           │
│  增益：                   │
│  ✅ 撤步获得额外效果：      │
│     下步对手命中 -1        │
│  ✅ 闪避成功率 +1          │
│                           │
│  战术含义：               │
│  空间充裕。机动自由。       │
│  蛾和灯在此如鱼得水。       │
│                           │
└───────────────────────────┘
```

## 🌑 黑暗 · 无光环境

```
┌───────────────────────────┐
│                           │
│      🌑  黑  暗            │
│                           │
│  如：地下墓穴、废弃矿井、   │
│      深夜小巷、下水道       │
│                           │
│  ═══════════════════════  │
│                           │
│  受限：                   │
│  ❌ 佯攻 命中 -1（看不见   │
│     虚晃）               │
│  ❌ 抢占先手 不可用        │
│     （看不清，无法预判）    │
│                           │
│  增益：                   │
│  ✅ 灯·性相加护：          │
│     额外揭示对手上一步      │
│     具体行动名             │
│  ✅ 蛾·性相加护：          │
│     闪避后自动反制         │
│     （造成1伤害）          │
│  ✅ 冬·性相加护：          │
│     寒意叠加 +1             │
│                           │
│  战术含义：               │
│  灯、蛾、冬的主场。         │
│  刃和铸在此如同盲人。       │
│                           │
└───────────────────────────┘
```
