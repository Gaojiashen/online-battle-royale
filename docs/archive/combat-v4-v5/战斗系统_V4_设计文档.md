> 2026-07-03 · 从 V3.1 全面重构

## 一、V4 核心变更总览

| # | 变更 | V3.1 | V4 |
|---|------|------|----|
| 1 | 提交模式 | 双方编程5步→逐步结算 | **双方同时提交5步→逐步揭示→逐步结算**（无预公布） |
| 2 | 反应槽 | 条件→替换当前步 | **仅能对已揭示的历史步做出反应**（如"若对手第2步打了进攻→我第3步换成XX"） |
| 3 | 伤害类型 | 有无视防御(Piercing) | **取消无视防御，当前阶段不做伤害类型** |
| 4 | 远近身位 | 近身/远程/中距离，独立系统 | **取消，统一为状态buff/debuff** |
| 5 | 先后手 | 先手/绝对先手机制 | **取消先后手，同优先级同时结算** |
| 6 | 同时死亡 | 平局 | **欠血少者胜**（HP负数绝对值小者赢） |
| 7 | 战斗HP | 战斗HP=游戏HP | **战斗HP独立于游戏HP**（如双方100战HP→败方扣10游戏HP） |

| 9 | 禁止出牌 | 存在"不让对方用X"的卡 | **不允许禁止对方出牌的设计** |

## 二、结算流程 V4

### 2.1 提交阶段（双方同时、秘密进行）

```
双方各自编排 5 步序列 → 秘密提交给法官
法官收到双方序列后 → 进入揭示阶段
```

### 2.2 揭示与结算（逐步）

```
For step N = 1 to 5:
  1. 双方同时揭示第 N 步的卡牌
  2. 检查反应槽（基于第 1..N-1 步已揭示的历史）
  3. 若反应槽触发 → 替换当前步卡牌
  4. 按优先级排序：打断(P1)=佯攻(P1) > 进攻(P2) > 防御(P3) > 状态(P4)
  5. 同优先级同时结算
  6. 应用伤害/效果
  7. 检查战斗是否结束（HP≤0）
  8. 若未结束 → 进入 Step N+1

全部5步结算完毕 → HP高者胜
双方HP均≤0 → 欠血少者胜（|负HP|小者赢）
双方HP>0且相等 → 平局
```

### 2.3 欠血判定

```
A: HP=5, 受8伤害 → -3 (欠3血)
B: HP=3, 受5伤害 → -2 (欠2血)
→ B胜（欠血2 < 欠血3）
```

## 三、核心规则变更

### 3.1 防御不能连续使用

- 若第 N 步使用了「防御」类卡牌 → 第 N+1 步不能使用任何防御卡
- 适用于所有防御卡：格挡、闪避、锻体、寂灭之息、不屈、蛾群幻象
- 此限制在编排序列时即生效（不能提交非法序列）
- 反应槽替换不受此限（替换后的卡如果是防御→仍然适用，即替换后的下一步不能防御）

### 3.2 禁止出牌设计

**不允许任何卡牌阻止对方使用某类/某张卡。** 受影响需重做的卡：
- ~~寂灭之息(W03): "双方均无法使用进攻"~~ → 重新设计

### 3.3 战斗HP与游戏HP分离

- **战斗HP**：仅存在于单场战斗中的临时HP（如双方均100）
- **游戏HP**：玩家的持久HP，战斗外生效
- 胜方：游戏HP不变（或微量回复）
- 败方：扣除游戏HP（基础10点，可因卡牌/性相等调整）
- "击杀后回满血"类效果取消 → 改为"回复战斗HP至满"（不影响游戏HP）
- 斩界(B05)的"击杀回满HP"移除

### 3.4 取消远近身位

- 移除：逼近(C04)、撤步(C05)
- 移除所有"近身位+2伤害""远程位解锁闪避"等位置规则
- 移除环境自动近身规则
- 位置相关效果统一为状态buff：
  - 原"近身位"→ 状态"压制"：下步进攻+2伤害
  - 原"远程位"→ 状态"拉开距离"：下步可闪避
- 室内闪避限制相应调整

### 3.5 取消先后手

- 移除"绝对先手"概念
- 所有同优先级行动同时结算
- 反应槽替换在揭示后、优先级排序前执行
- 灯的先读(L03)完全重做

## 四、RPS 克制关系（保留）

| 类别 | 优先级 | 效果 | 克制关系 |
|------|:--:|------|------|
| 进攻 | P2 | 造成伤害 | 被防御克制（对方获得破绽） |
| 防御 | P3 | 减免进攻伤害→获得破绽 | 被佯攻克制（绕过，无破绽） |
| 佯攻 | P1 | 绕过防御+1伤害 | — |
| 打断 | P1 | 取消状态 | — |
| 状态 | P4 | 增益/回复/信息 | 被打断克制（取消） |

进攻**不**中断状态。此RPS关系不变。

## 五、破绽机制（微调）

- 防御成功应对进攻→获得1层破绽（若减免至0则不获得）
- 消耗1层破绽→下步进攻伤害×2
- **最多持有2层**（不变）
- 佯攻绕过防御→防御方不获得破绽
- 闪避成功→必定破绽+1（伤害完全规避≠减免至0）
- **新增**：破绽在获得后的下步才能使用（不可同歩获得+消耗）

## 六、连击机制（Multi-Hit）

### 设计回答：8 vs 4+4

连击的价值在于**防御穿透特性**：

- **防御减免仅对每步第一个伤害实例生效**，同一行动内的后续伤害实例全额穿透
  - 8 vs 格挡(减免2) → 8-2 = **6**
  - 4+4 vs 格挡(减免2) → (4-2)+4 = **6**（第二击不受减免）
  - 3+3+3 vs 格挡(减免2) → (3-2)+3+3 = **7**（更多击→穿透更多）

- **每击独立触发命中效果**：寒意、回血、破绽消耗选择等
- **对护盾更高效**：第一击破盾，后续击打HP
- **对幻象更稳定**：50%×N次 → 至少部分命中的概率更高

结论：连击的价值是**穿透防御 + 多次触发**。击数越多，防御越难完全阻挡。

## 七、反应槽 V4

### 规则

- 每场1个反应槽
- 条件格式：「若对手第 [已揭示步号] 使用了 [行动类别/具体卡名] → 我第 [后续步号] 改为 [卡名]」
- **只能引用已揭示的历史步**，不能引用当前步或未来步
- 反应槽在序列提交时预先声明
- 结算时条件满足 → 自动替换 → 新卡按正常优先级参与排序

### 示例

```
反应槽：「若对手 Step 2 打出「进攻」→ 我 Step 3 改为「格挡」」

Step 1: 双方揭示，对手用佯攻 → 不触发
Step 2: 双方揭示，对手用劈斩(进攻) → 条件满足，标记触发
Step 3: 你的原始卡被替换为格挡，揭示并结算
```

## 八、灯性相 V4 重新设计

**核心主题：** 观察与适应（Observer & Adapter）。灯不预知——灯看穿。

### 新卡牌设计

| 编号 | 卡名 | 类别 | 门槛 | 效果 |
|------|------|------|------|------|
| L01 | 明见 | 状态 | Lv2+ | 查看对手**反应槽的条件**（不消耗反应槽，纯粹信息）。Lv6：同时查看对手剩余步中是否有打断卡。Lv10：同时查看对手剩余步中打断卡的具体步号。Lv15：查看对手全部隐藏信息。 |
| L02 | 驱散 | 打断 | Lv2+ | 取消对手本步的状态。若成功取消→你获得对手本步被取消的那张卡的信息（卡名+完整效果文本）。每场2次。 |
| L03 | 洞悉 | 佯攻 | Lv6+ | 造成1伤害。若对手本步使用的行动类别与上一步**相同**→本卡伤害+2且对手无法获得破绽。 |
| L04 | 全视 | 状态 | Lv10+ | 选择对手一个**尚未揭示**的步号→查看该步的卡牌。然后你可以将自己对应步号的卡替换为手牌中任意一张（仅限该步）。每场1次。 |
| L05 | 揭示一切 | 状态 | Lv15 | 查看对手全部尚未揭示的步的卡牌。选择其中一步→将其卡牌**变为你指定的一张通用基础卡**（挥击/佯攻/格挡/打断中任选）。每场1次。 |

### 灯被动 V4

| 等级 | 被动 | 效果 |
|:--:|------|------|
| Lv2 | 启蒙 | 开战前得知对手的**主性相**和**性相等级范围**（低0-5/中6-9/高10-14/飞升15） |
| Lv6 | 注视 | 每场1次：对手揭示一步后，你可立即免费查看该步卡牌的完整效果文本 |
| Lv10 | 适应 | 你的反应槽可额外触发1次（即每场最多2次反应槽触发） |
| Lv15 | 无蔽 | 迷雾/黑暗/任何信息遮蔽效果对你无效。你免疫"信息不完整"类效果。 |

### 设计意图

- 灯不再"预知"——改为"看穿隐藏信息"（反应槽条件、未揭示步骤、卡牌详情）
- Lv15是灯唯一可以**干涉**对手序列的卡（将对手一张卡变为基础卡）——这是"揭示"的终极形态：不仅看到，还能改写
- 被动强化信息的深度和质量，而非获取时机
- 洞悉(L03)惩罚重复出牌——和防御不能连续用形成交叉压力

## 十一、关键词汇表 V4

| 词条 | 说明 |
|------|------|
| 进攻 | 造成伤害的行动。优先级2。 |
| 防御 | 减免/规避伤害。优先级3。**不能连续两步使用。** |
| 佯攻 | 快速低伤。优先级1。克制防御（绕过，不产生破绽）。 |
| 打断 | 取消状态。优先级1。不造成伤害（除非卡牌另有说明）。 |
| 状态 | 增益/回复/信息。优先级4。唯一克星是打断。 |
| 破绽 | 防御成功应对进攻→+1层（减免至0则不获得）。消耗1层→下步进攻伤害×2。最多2层。佯攻绕过→无破绽。 |
| 寒意 | 冬专属debuff。每层使目标进攻伤害-1。最多3层。3层触发处决。 |
| 蓄力 | 铸专属buff。每层使下步进攻伤害+1。攻击出手后清零。最多3层。 |
| 幻象 | 蛾专属。对手行动50%落空。落空→破绽+1+反刺。 |
| 护盾 | 心专属。额外HP缓冲，优先消耗。 |
| 反应槽 | 每场1次（灯Lv10=2次）。基于已揭示步的条件→替换后续步。 |
| 终局压力 | Step4-5：所有进攻伤害+1。 |
| 连击 | 同一行动内的多次攻击。防御减免仅对第一击生效。 |
| 欠血 | HP降至负数时的绝对值。同时死亡时欠血少者胜。 |
| 战斗HP | 单场战斗的临时HP，与游戏HP独立。 |
