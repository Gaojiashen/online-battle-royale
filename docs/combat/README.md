# 战斗系统演练牌组

本目录包含可打印的战斗系统演练牌组，用于手动测试"行动序列预编程"战斗模式。

## 版本

| 版本 | 文件 | 说明 |
|:--:|------|------|
| **V2（当前）** | `牌组/行动卡_v2.md` | 性相等级决定卡池+效果缩放+被动 |
| V1（旧） | `牌组/行动卡.md` | 能量池方案，已搁置 |

## 文件说明

| 文件 | 用途 |
|------|------|
| `牌组/行动卡_v2.md` | 30张行动卡（含等级门槛和缩放公式） |
| `牌组/性相卡.md` | 6张性相参考卡 |
| `牌组/环境卡.md` | 3张环境卡——室内/室外/黑暗 |
| `牌组/战斗记录表.md` | 战斗记录表单——可打印 |
| `牌组/示例对战_v2.md` | 3场V2示例对战——逐步拆解 |
| `牌组/规则速查_v2.md` | V2规则单页速查 |

## 快速开始

1. 选定你的**性相和等级**（如：刃Lv6）
2. 查看 `规则速查_v2.md` ——确认你的卡池和被动能力
3. 查看 `行动卡_v2.md` ——确认每张卡在你当前等级的效果
4. **打印** `战斗记录表.md`（每场战斗一张）
5. 找一位朋友：各自确定性相等级，秘密编排5步序列，同时揭示，按步结算
6. 参考 `示例对战_v2.md` 看完整对战流程

## 核心规则速览（V2）

- 固定 **5步**，分三阶段：**开局(1) → 中盘(2-3) → 终局(4-5)**
- **性相等级** 决定你能用什么牌 + 同一张牌打出的效果
- 卡牌分为四级准入门槛：Lv0+（通用）→ Lv2+ → Lv6+ → Lv10+ → Lv15
- **性相被动** 在 Lv2/6/10/15 自动获得
- 每步双方各执行1个行动，按**优先级**结算
- 每场 **1个反应槽**（可选，预先声明触发条件）
- 终局阶段（Step 4-5）所有攻击伤害 **+1**
