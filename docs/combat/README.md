## V6 核心规则

- **48 张卡牌**：6 通用 + 42 性相专属（7 张/性相 × 6 性相）
- **六性相独占资源**：锋芒(刃)、幻影(蛾)、蓄力(铸)、寒意(冬)、脉动(心)、洞悉(灯)
- **RPS 结算**：进攻 → 防御 → 佯攻 → 打断 → 进攻（状态独立于 RPS 环外）
- **战斗 HP = 20**，战前从可用牌中选 8 张为本场牌库
- **双方盲编 5 步序列，逐步揭示，同时结算**
- **无先后手、无视防御（V6）、远近身位已移除**
- 等级门槛：Lv0 / Lv2 / Lv6 / Lv10 / Lv15

## 快速开始

1. 确定双方性相等级（灯/蛾/铸/冬/心/刃，各 Lv0-15）
2. 运行 `python generate_cards_v6.py` 生成最新卡牌 Excel
3. 查看 `牌组/战斗牌组_v6.xlsx` 了解全部 48 张卡牌
4. 法官面板：访问 `/judge` 发起对战
5. API 文档：访问 `/docs`（FastAPI 自动生成）
