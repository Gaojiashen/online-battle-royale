# 战斗系统 V5 设计文档

> 2026-07-03 · V4→V5 精简重构

---

## V5 核心变更（相对于V4）

| # | V4 | V5 | 原因 |
|---|-----|-----|------|
| 1 | 有优先级(P1-P4) | **无优先级，同时结算，RPS决定交互结果** | 简化结算 |
| 2 | 破绽(Opening) | **看破**（效果不变：防御成功→下次进攻伤害×2） | 更直观的命名 |
| 3 | 词条说明含"典型卡牌"列 | **去掉** | 用户要求 |
| 4 | 防御减免至0则不获得破绽 | **防御成功必定获得看破** | 简化判定 |
| 5 | 看破可主动消耗（选择哪一击×2） | **看破自动触发：下次进攻自动×2** | 看破不可预测，不能设计"主动利用" |
| 6 | 蛾闪避/幻象落空→反刺 | **落空只积累看破，不反刺** | 简化蛾之道 |
| 7 | 心护盾溢出伤害穿透到HP | **溢出伤害不结算**（10(2)受3→10(0)） | 护盾更强 |
| 8 | 阶段限制（中盘/终局） | **全部取消** | 无存在必要 |
| 9 | 每场限制由设计规定 | **留白** | 用户自行设计 |
| 10 | 基础打断每场1次 | **无限制** | 简化 |
| 11 | 连斩可选择哪一击消耗破绽 | **看破自动翻倍全部击数** | 玩家无法预判看破何时出现 |
| 12 | 卡牌列表有破绽机制列 | **去掉** | 看破是通用机制，不必每张卡列出 |
| 13 | 方括号⌊⌋用于取整 | **统一用[]** | 更简洁 |
| 14 | 万刃主动消耗看破 | **重做：高伤+自限** | 不能主动利用看破 |
| 15 | 斩界击杀回满血 | **去掉回血** | 战斗HP≠游戏HP，击杀后回血无意义 |
| 16 | 终焉无自伤只翻倍 | **重做：自伤+寒意+处决翻倍** | 没伤害翻倍无意义 |
| 17 | 心脏共鸣双方回血 | **对方回HP，己方回护盾** | 用户设计意图 |
| 18 | 灯看隐藏/改出牌 | **全部重做：模式惩罚+战斗外信息** | 盲提交下信息效果无效/耍流氓 |

## 结算流程 V5

```
1. 双方同时盲编5步序列→提交法官
2. Step 1→5 逐步：
   a. 双方同时揭示当前步卡牌
   b. 检查反应槽（仅基于已揭示的历史步）
   c. 双方行动同时结算，RPS决定交互
   d. 应用伤害/效果
   e. 检查战斗结束（HP≤0）
3. 5步结束→HP高者胜
   双方HP均≤0→欠血少者胜
   双方HP>0且相等→平局
```

## RPS 克制关系

所有行动同时结算，RPS决定结果：

| A \ B | 进攻 | 防御 | 佯攻 | 打断 | 状态 |
|--------|:--:|:--:|:--:|:--:|:--:|
| **进攻** | 双方互伤 | A被减免+B获看破 | 双方互伤 | 各自生效 | 各自生效 |
| **防御** | A减免+A获看破 | 双方无效步 | A被绕过+无看破 | 各自生效 | 各自生效 |
| **佯攻** | 双方互伤 | A绕过B防御 | 双方互伤1 | 各自生效 | 各自生效 |
| **打断** | 各自生效 | 各自生效 | 各自生效 | 双方浪费 | B状态取消 |
| **状态** | 各自生效 | 各自生效 | 各自生效 | A状态取消 | 双方生效 |

要点：
- 进攻**不中断**状态
- 佯攻绕过防御→防御方不减免、不获得看破
- 打断唯一目标是取消状态
- 防御不能连续两步使用

---

## 看破机制（原破绽）

- 防御成功应对进攻→**必定获得1层看破**（无论伤害减免多少）
- 看破自动触发：**下次进攻伤害自动×2**
- 最多持有2层看破→×4
- 佯攻绕过防御→防御方不获得看破
- 玩家**不能主动选择**何时消耗看破——它是被动的、自动的
- 闪避成功→必定获得看破

---

## 连击机制

- 防御减免仅对每步**第一个伤害实例**生效，后续击穿透
- 看破翻倍→**全部击数翻倍**（因为看破="下次进攻"，整个行动是"一次进攻"）
- 每击独立触发命中效果（寒意施加等）

---

## 寒意机制（不变）

- 每层使目标进攻伤害-1，最多3层
- 叠满3层→下次受伤害时触发处决（伤害翻倍）
- W06凛冬保持原设计
- W07终焉重做：自带伤害+寒意+立即处决

---

## 心·护盾

- 护盾优先消耗
- **溢出伤害不结算到HP**：10(2)受3伤害→10(0)，多余1点伤害消失
- 护盾消耗后不自动回复

## 关键词汇表 V5

| 词条 | 说明 |
|------|------|
| 进攻 | 造成伤害的行动。 |
| 防御 | 减免/规避伤害。**不能连续两步使用。** |
| 佯攻 | 快速低伤。克制防御（绕过，不产生看破）。 |
| 打断 | 取消状态。不造成伤害（除非卡牌另有说明）。 |
| 状态 | 增益/回复/信息。唯一克星是打断。 |
| 看破 | 防御成功应对进攻→必定+1层。自动触发：下次进攻伤害×2。最多2层。佯攻绕过→无看破。 |
| 寒意 | 冬专属debuff。每层使目标进攻伤害-1。最多3层。叠满3层→下次受伤害触发处决（翻倍）。 |
| 蓄力 | 铸专属buff。每层使下步进攻伤害+1。攻击出手后清零。最多3层。 |
| 幻象 | 蛾专属。对手行动50%落空。落空→看破+1。无反刺。 |
| 护盾 | 心专属。额外HP缓冲，优先消耗。**溢出伤害不穿透**（3伤害打10(2)→10(0)）。 |
| 反应槽 | 基于已揭示步的条件→替换后续步。 |
| 连击 | 同一行动多次攻击。防御减免仅对第一击生效。看破翻倍全部击数。 |
| 欠血 | HP降至负数时的绝对值。同时死亡时欠血少者胜。 |
| 战斗HP | 单场战斗临时HP，与游戏HP独立。败方扣除游戏HP。 |
