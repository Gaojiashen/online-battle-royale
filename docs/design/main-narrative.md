## 一、游戏本质

本项目不是传统大逃杀。玩家之间的战斗是手段，而非目的。

游戏本质是：

> 在一个注定于 Day 20 迎来毁灭的世界中，通过探索、调查、成长与飞升，寻找活下去的方法。

核心体验：

```
探索世界 → 获得情报 → 发现真相 → 接触神秘学 → 争夺飞升机会 → 决定自身命运
```

---

## 二、世界观基础

### 毁灭事件

**Day 20，爱丁堡将被毁灭。**

毁灭者不是一个来自外部的高维存在，而是一个**人类**——蛾教会的一名高层成员。他在进行蛾飞升仪式时，额外注入了一件灯蛾双性相的遗物。飞升成功了，但他承载了人类心灵无法承载的融合力量，**失控屠城**。

对爱丁堡的普通人来说，他和高维存在无异——无法抵抗。无论财富、权力、军队或势力，都无法在毁灭中存活。

### 反派概要

| 维度 | 内容 |
|------|------|
| **身份** | 蛾教会高层成员 |
| **性相** | 蛾（主），暗中追求灯蛾双性相的力量 |
| **动机** | 野心——他想要前所未有的力量，相信自己能驾驭两个性相 |
| **在城时间** | 全程在爱丁堡。Day 1 他已在城内，在20天内为飞升做准备 |
| **核心物品** | 一件十年前灯蛾双飞升留下的遗物，同时含有灯和蛾的性相，由蛾教会秘密保存 |
| **所知风险** | **他知道**这件遗物含有灯性相。注入它是他主动的选择，不是被蒙蔽，不是别无选择——是傲慢 |
| **结局** | Day 20 完成飞升 → 灯蛾双性相力量超出控制 → 失去理智 → 屠城 |

### 核心物品：双性相遗物

十年前（唐纳63年），第一届灯之飞升和蛾之飞升在同一晚完成。那晚产生了一件同时含有灯与蛾性相的遗物。蛾教会在事后获得了它，秘密保存至今。

这件遗物的具体形态和性质**尚未确定**（待后续讨论）。

## 三、时间回溯

所有玩家都经历过一次毁灭。

- Day 20：反派飞升失控，世界毁灭，所有玩家死亡。
- 随后时间回溯至 Day 1。
- **玩家保留前一次轮回的记忆。**

时间回溯是游戏规则层面的设定，不需要进一步解释其来源或原理。重点不是"为什么回溯"，而是"玩家如何利用记忆活下去"。

### 当前设定：第二次轮回

本次游戏是玩家的**第二次轮回**。玩家拥有一次完整的 Day 1-20 记忆。在之前的轮回中，可能有某个玩家在 Day 20 接近了真相——甚至伤到了反派——这为本次轮回创造了新的可能性。

（是否有多于两次轮回、NPC 是否残留记忆碎片——**尚未确定**。）

## 四、玩家目标

| 阶段 | 目标 |
|:----:|------|
| **开局** | 活下去 |
| **中期** | 调查毁灭原因——发现异常事件、追踪反派的踪迹、理解双性相遗物的真相 |
| **后期** | 寻找飞升方法——飞升是唯一能跨越 Day 20 的手段 |
| **最终** | 获得跨越毁灭的资格——以飞升者身份在 Day 20 存活 |

## 五、飞升

飞升是世界中的真实超凡道路。

随着调查深入，玩家会逐渐发现：普通手段无法抵抗 Day 20 的毁灭。飞升不是力量追求，而是**生存需求**。

> 飞升是活下去的唯一方法。

### 蛾飞升仪式：「蜕」（已设计框架）

六性相各有独立的飞升仪式。蛾飞升的框架已初步确定，其他性相待设计。

#### 仪式总览

```
阶段一：隐匿（锚点设置 1h→2h→3h）
    ↓
阶段二：剥离（仪式战斗 · 与旧我之战）
    ↓
阶段三：重生（沉默24h → 飞升完成）
```

#### 阶段一：隐匿

| 项目 | 内容 |
|------|------|
| **持续时间** | 3天 |
| **前置条件** | 蛾性相 Lv14 |
| **仪式内容** | 在爱丁堡地图上选择3个区域作为锚点，在每个锚点执行一个设置仪式 |
| **锚点耗时** | 第一锚点 1h → 第二锚点 2h → 第三锚点 3h（递增） |
| **区域条件** | 仪式执行期间，该区域不能有任何其他玩家存在或经过 |
| **材料消耗** | 每个锚点消耗：蛾密传 + 蛾性相物品 |
| **被经过的后果** | 仪式不被破坏，但需额外消耗蛾性相物品（不额外消耗密传）补充后继续 |
| **设计意图** | 耗时递增 + 材料成本递增；玩家需要侦查其他玩家的位置来执行仪式；被经过不导致完全失败，但提高成本；为其他玩家提供反制窗口（怀疑某人在准备飞升→分散巡逻增加其成本） |

#### 阶段二：剥离 ——「与旧我之战」

| 项目 | 内容 |
|------|------|
| **触发条件** | 阶段一完成 |
| **仪式时间** | 深夜（22:00-8:00），在任一锚点区域 |
| **战斗类型** | 仪式战斗（PvE，非 PvP） |
| **对手** | 「旧我」镜像 |
| **镜像牌组** | 从玩家**所有获得过的卡牌**中随机抽取指定数量，构成镜像牌组 |
| **出牌控制** | 由法官操控镜像的出牌顺序 |
| **胜利条件** | 击败旧我镜像 |
| **失败后果** | 损失游戏HP上限（非战斗HP） |
| **设计意图** | 旧我由"你走过的每一条路"组成——包括你放弃的卡。战斗是叙事性的：你在杀死过去的自己 |

#### 阶段三：重生

| 项目 | 内容 |
|------|------|
| **持续时间** | 24小时 |
| **效果** | 战斗胜利后，你在该锚点"消失"——无法被任何玩家或 NPC 定位，不能主动发起任何行动 |
| **完成后** | 24小时后以新形态重新出现在世界某处。飞升完成。你不再是原来的你 |

### 反派的特殊飞升

反派使用同一套蛾飞升仪式框架。区别在于：他在阶段二中**额外注入双性相遗物**——他不满足于单纯的蛾飞升，要把灯的那部分也吞下去。

飞升成功了。但灯蛾融合的力量超出了人类心灵的承载能力。他在飞升后失控。

## 六、结局层级

### 第一层：生存结局

成功飞升。在 Day 20 存活。这是最基础的胜利条件。

### 第二层：超越结局

成功飞升。在 Day 20 拥有与失控飞升者对抗的资格。参与最终决战。

### 第三层：救世结局

多个飞升者共同努力。在反派失控前阻止他，或在失控后击败他。阻止爱丁堡的毁灭。改变世界既定命运。达成最高级结局。

## 七、剧情设计原则

### 1. 先设计毁灭，再设计飞升

先明确：世界为何毁灭、毁灭如何发生、毁灭有哪些征兆。然后设计：飞升路线、超凡体系、飞升仪式。飞升是解决方案。毁灭才是问题本身。

### 2. 所有剧情围绕 Day 20 展开

整个世界都应受到毁灭事件影响。包括：NPC、势力、报纸、事件、传闻、调查线。它们都应该与主线产生联系。

### 3. NPC 是世界参与者

NPC 不是任务发布器。NPC 拥有自己的目标与命运。例如：调查异常、试图阻止灾难、被灭口、接触神秘力量、成为飞升候选者。NPC 会随着时间推进发生变化。

### 4. 世界应逐渐异常化

- Day 1：世界看似正常。
- 随着时间推进：异常事件不断增加。
- 玩家逐步意识到：世界正在走向毁灭。
- 而不是在 Day 20 突然宣布世界毁灭。
- 玩家应能够感受到倒计时的存在。

### 5. 仪式是核心行动类型

仪式是游戏中的重要行动类型，消耗材料和时间，获得确定的结果。飞升仪式、教团内部仪式、反派在 Day 20 的最终仪式——都共用同一套仪式行动框架。（仪式通用模板待后续设计。）
