## 一、总览

### 世界定位

城邦世界。一座与世隔绝的爱丁堡，包含老城、城外乡村、港口（利斯港）与贵族庄园。除此之外再无可知的远方。

这是一个**阴冷潮湿的哥特式城邦**——湿漉漉的鹅卵石街道、不散的薄雾、煤油灯的黄晕在巷弄间摇曳、教堂钟声在石砌建筑间回荡。历史感的厚重与神秘诡谲的气息交织在一起。

### 纪年

**唐纳王朝（Donnell Dynasty）· 治世第 73 年**

王朝历史不作详考。知道这一点就够了：这不是初代王朝，也不是摇摇欲坠的末路。唐纳家族坐在王座上已有三代人的时间，足够让一切看起来"从来如此"。

### 主流教会

两大教会各自独立，**井水不犯河水**——但在具体事务上，冲突或合作都可能发生，并非非黑即白。

#### 灯之教会（Lantern Church）

- **核心追求**：洞彻、知识、揭示
- **世俗对应**：学者、图书管理员、法官、天文学家
- **影响力方式**：通过教育、司法、文书系统渗透权力结构——为贵族子弟授课、担任市政厅书记官、管理档案馆
- **对超自然的态度**：研究、记录、尝试理解。灯教会追求"知道"，但不一定追求"使用"。
- **标志色**：暖金 / 象牙白

#### 蛾之教会（Moth Church）

- **核心追求**：隐秘、变化、诱惑
- **世俗对应**：盗贼、走私者、情报贩子、演员
- **影响力方式**：通过人情网络、秘密交易、信息差影响决策——他们知道谁和谁在交易、谁欠谁人情、哪封密信里写了什么
- **对超自然的态度**：利用、保护秘密、有选择地释放信息。蛾教会相信"该让人知道的才让人知道"。
- **标志色**：暗紫 / 银灰

### 地下教团

四个独立的秘密教团，各自信奉一种性相。**表面敌对**（被官方发现信众在接触，不会牵扯到其他教团），**暗中合作或共存**。

当局（市政厅/贵族）知道它们存在，**睁一只眼闭一只眼，必要时打压**。

| 性相 | 教团名 | 公开伪装 |
|:----:|:-------|:---------|
| **刃** | 断刃修会 | 退役军人的格斗俱乐部 |
| **铸** | 熔心会 | 工匠行会 / 炼金术士公会 |
| **冬** | 银棺修会 | 殡仪馆 / 档案管理 |
| **心** | 赤脉结社 | 乐师、诗人、舞者的娱乐社团 |

四个教团共同相信：**所有路径都可以通往飞升**。但哪条路径最容易、最安全、代价最小——它们各有各的答案。

### 其他势力

| 势力 | 说明 |
|:-----|:------|
| **市政厅** | 行政管理机构。与贵族阶层深度勾结。高层可能有几人是教会的棋子，但不构成系统性控制 |
| **贵族阶层** | 掌握土地与财富。对性相的态度因人而异——有人当学问钻研，有人当工具使用 |
| **大学** | 受多方势力渗透的灰色地带。教师与学生中既有教会的人、教团的人，也有单纯的求知者，还有什么都不懂的普通人 |
| **地下势力** | 黑帮、走私者、地头蛇——本质是高层养的狗。偶尔被用来打压不听话的教团成员，偶尔被教团反渗透利用 |

## 三、飞升体系

### 六性相

飞升有六条路径，对应六种性相（Aspects）：

| 性相 | 象征 | 主流/地下 | 十年前成功飞升 |
|:----:|:-----|:---------:|:--------------:|
| 灯 | 洞察、知识、揭示 | 🔆 主流 | ✅ 灯之飞升（被掩盖为"书店火灾"） |
| 蛾 | 变化、诱惑、隐秘 | 🔆 主流 | ✅ 蛾之飞升（被掩盖为"飞蛾骚乱"） |
| 刃 | 冲突、毁灭、锋锐 | ⛓️ 地下 | ❌ |
| 铸 | 锻造、转化、创造 | ⛓️ 地下 | ❌ |
| 冬 | 寂灭、记忆、坚韧 | ⛓️ 地下 | ❌ |
| 心 | 生命、守护、激情 | ⛓️ 地下 | ❌ |

### 性相等级

性相等级以**偶数**推进：

```
0 → 2 → 4 → 6 → 8 → 10 → 12 → 14 → 15
```

**关键节点：**
- Lv2 → 解锁技能①
- Lv6 → 解锁技能②
- Lv10 → 解锁技能③
- Lv15 → 解锁技能④（需通过关键仪式达成）
- Lv15 = 飞升条件之一

提升性相等级的主要途径：**学习密传** + **特定仪式**。

### 密传（Lore）

**密传是书。** 记录了对应性相的隐秘知识、历史片段、仪式线索。

**密传等级体系（偶数为级）：**

```
等级: L2  L4  L6  L8  L10  L12  L14
```

**合成规则：**

```
2 份同级密传 → 合成 → 1 份高一级密传
例：2 × 灯·L2 + 2 × 灯·L2 → 1 × 灯·L4
```

**学习规则：**
- 学习密传后，该密传**销毁**
- 性相等级决定你能读什么等级的书：
  - Lv2 时 → 可读 L4 密传
  - Lv4 时 → 可读 L6 密传
  - 以此类推，**性相等级 < 密传等级 − 2 时无法阅读**
- 各性相的密传**各自独立**，不能跨性相合成

**密传获取途径：**
- 探索区域→发现
- NPC 处购买 / 交易 / 受赠
- 从其他玩家处抢夺
- 通过仪式获得
- 合成

### 技能

| 性相等级 | 解锁 | 说明 |
|:--------:|:----:|:------|
| Lv2 | 技能① | 可被动可主动，具体效果后续细化 |
| Lv6 | 技能② | |
| Lv10 | 技能③ | |
| Lv15 | 技能④ | |

技能不是用来"说服法官"的道具——**裁定依赖规则，而非叙事理由**。技能的效果体现在规则层面的硬性修正或能力解锁。

### 飞升的三个条件

1. **对应性相达到 Lv15**
2. **关键仪式**（含所需材料）
3. **承担失败风险**（反噬、泄露、不可逆后果）

飞升完成后，该玩家在游戏中的旅程即告结束。**只有第一个飞升的人获胜。**

## 四、时间与世界运行

### 昼夜循环

| 时段 | 时间 | 说明 |
|:----:|:----:|:------|
| ☀️ 白天 | 8:00—20:00 | 玩家可提交行动，法官正常裁定 |
| 🌙 夜晚 | 20:00—22:00 | 部分行动/仪式的条件可能特定于夜晚；玩家仍可行动 |
| 💤 深夜 | 22:00—次日8:00 | 日切时段。世界状态推进（势力行动、NPC 行动、环境变化等） |

### 行动系统（V1 — 按轮）

每轮每位玩家最多 **3 个行动位**：
- 其中最多 1 个 L3 行动（仪式 / 攻击）
- 行动按提交顺序执行
- 玩家之间**不存在自由交流渠道**——所有玩家间沟通必须通过游戏内行动

## 五、经济

金币。没有更细的辅币体系。

### 用途

- 购买密传
- 购买情报
- 雇佣人手
- 购买仪式材料

## 六、玩家

### 基础设定

玩家扮演的是**爱丁堡本地人**，且是**奉身神秘的人**（devotees of the occult）。原因各不相同——有人渴求力量，有人渴望知识，有人只想逃离平庸的生活——但他们都在这条路上走出了第一步。

### 身份系统

每位玩家在进入游戏时获得一个**身份**，决定：
- **开局包裹**（初始金币 + 可能的基础物品/密传）
- **出生区域**

身份举例：
- 报社老大
- 酒馆服务生
- 更多身份待设计……

### 玩家间关系

- 玩家之间**互不相识**（就角色而言）
- 但知道自己之外还有别的求道者在活动——这是压力的来源
- 本届规则中**不给予玩家间直接沟通的渠道**
- 所有玩家间的交互必须通过游戏内行动传递

## 九、叙事基调

**阴冷潮湿的哥特式爱丁堡。**

街道永远是湿的——即使没有下雨，石缝里也渗着潮气。煤油灯的光在薄雾中晕开成一团模糊的黄。有些巷子太窄，两个人侧身都过不去。教堂的钟声每整点响起，但在某些区域，钟声听起来比实际更远。

普通人的生活照常运转——市场讨价还价、酒馆里吹牛、学徒被师傅骂。但如果你知道该在什么时候、看向什么地方，你会发现：

- 有人在一扇没有门牌的门前停顿了三秒，然后推门进去
- 某本旧书的页边距里写着看不懂的符号
- 棺材铺的老板半夜还在亮着灯
- 地下的流水声里夹杂着不像水的声音

知道得太多不一定是好事。但你已经打开了第一本书。
