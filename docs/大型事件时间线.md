# 大型事件时间线

> **法官操作手册。**
>
> 本文档描述20-30游戏日内发生的5个全城级别大型事件。
> 每个事件列出：触发条件、受影响NPC、可能结果、法官操作步骤。
> 事件按游戏日推进——日切时（每日22:00-08:00）检查当日是否有事件触发。

---

## 事件概览

```
Day 1-3   ─ 邀请函抵达（自动触发）
    |
Day 4-6   ─ 备战期（NPC各自行动，玩家驱动）
    |
Day 7     ─ ★ 玛莉金巷拍卖（核心汇合事件）
    |
Day 8-13  ─ 争夺期（手稿分册易手，多势力追猎）
    |
Day 14-17 ─ 征兆之夜（天体排列触发跨性相现象）
    |
Day 18-19 ─ 动荡期（势力重新评估，新联盟/敌对）
    |
Day 20-25 ─ 清剿威胁（条件触发：丹比获得足够证据）
    |
Day 26-30 ─ 飞升窗口（终局收束）
```

## 事件一：邀请函抵达（Day 1-3）

### 触发
自动触发。游戏开始时同步发生。

### 事件描述
中间人的地下拍卖邀请函通过不同渠道送达全城每一个势力。没有正式名称——只有时间（七天后）、地点（玛莉金巷⑮）和一个冬之符纹的封蜡印记。

### 收到邀请函的NPC（12人）

| NPC | 接收方式 | 反应 |
|:----|:--------|:-----|
| **奥尔德雷克·索恩**（蛾） | 一只蛾停在香料铺窗台上，翅膀上有暗码 | 已经知道。蛾之教会早收到风声 |
| **莫伊拉·烛芯**（心） | 夹在音乐学院新进的乐谱里 | 警觉——冬之手稿对她而言是威胁 |
| **阿奇博尔德·蒙特罗斯**（普通） | 放在他亡子的墓碑上 | 认为这是针对他的信息——"有人知道我在找什么" |
| **艾拉拉·弗林特**（铸） | 熔炉火焰突然变色，在灰烬中形成文字 | 激动——容器材料信息可能在手稿中 |
| **乔纳斯·惠特菲尔德**（冬） | 直接送到殡仪馆——银棺修会本身就是被邀请方 | 恐惧——他知道这意味洛克哈特的秘密要被公开了 |
| **马库斯·布莱克伍德**（刃）| 借由蛾之教会在俱乐部外监视时"意外"泄露 | 警觉但不意外——他一直在追踪的线索终于浮现 |
| **西尔维娅·韦恩**（冬） | 没收到——但她通过墓园的异常活动推断出来了 | 必须在银棺修会之前找到手稿 |
| **埃德蒙·里德**（灯） | 夹在他正在编目的一本旧书里 | 意识到自己二十年的暗记追踪指向了同一个方向 |
| **格里格·哈尔斯特德**（心） | 通过地下渠道——"有人要卖东西" | 商人本能——组织安保和资金 |
| **罗文·刀**（刃） | 格里格雇她做拍卖安保 | 无所谓——有钱赚就行 |
| **维克多·阿什福德**（蛾） | 索恩告诉他 | 开始策划自己的计划——背叛导师 |
| **托马斯·伦威克**（普通） | 没收到——但玛格特从地下渠道听到了风声 | 开始调查"地下非法集会" |

### 法官操作
- Day 1日切前，在公开通告中加入暗示性描述：
  > *「近日城中出现了一则隐秘的传言——据说一件'不属于任何教团'的密传手稿将在近期被挂牌出售。邀请函已送达各方。」*
- 已锁定NPC的玩家可能在节点对话中听到NPC提及拍卖
- 此事件不强制任何NPC的行动——仅设定背景

## 事件二：玛莉金巷拍卖（Day 7）★核心汇合事件

### 触发
自动触发（Day 7 午夜0:00）。若有玩家已进入相关NPC的拍卖相关节点（西尔维娅④、海伦娜⑥、埃德蒙⑤等），则玩家可参与拍卖场景。

### 地点
⑮玛莉金巷 —— 地下拍卖场。冷光照亮中央石台。在场者约15-20人。

### 参席NPC（按势力和立场）

| 势力 | NPC | 目的 |
|:-----|:----|:-----|
| 拍卖主持 | **中间人**（?） | 主持拍卖。四册手稿分册依次上拍 |
| 蛾之教会 | **索恩**（蛾）| 竞拍跨性相理论分册。派维克多监视其他买家 |
| 蛾之教会 | **维克多**（蛾）| 表面协助索恩，暗中等待机会偷拍 |
| 赤脉结社 | **莫伊拉**（心）| 不出价——观察。如果冬相关分册落入银棺修会，她将干预 |
| 熔心会 | **艾拉拉**（铸）| 竞拍仪式要件分册（铸之盐来源） |
| 银棺修会 | **莫蒂默·阿什**（冬）| 竞拍全部冬之论述分册——"这是修会财产" |
| 贵族 | **蒙特罗斯**（普通）| 竞拍他听说能"与死者对话"的任何分册 |
| 独立 | **格里格**（心）| 不竞拍——在幕后为多方提供安保和资金 |
| 独立 | **罗文**（刃）| 格里格雇的安保。观察在场所有人的战斗能力 |
| 独立 | **西尔维娅**（冬）| 未受邀但可能在第四分册拍卖时现身（取决于玩家选择） |
| 城防军 | **伦威克**（普通）| 未在场——但接到了线报。在玛莉金巷外布控 |

### 四册分册

| 分册 | 内容 | 起拍价 | 主要竞拍方 |
|:-----|:------|:------|:----------|
| **第一册：冬之论述** | Lockhart的冬之飞升理论研究 | 高 | 银棺修会（莫蒂默）、蒙特罗斯 |
| **第二册：跨性相理论** | 六性相交汇的理论框架 | 最高 | 蛾之教会（索恩）、埃德蒙（若玩家帮）。**索恩志在必得** |
| **第三册：仪式要件** | 六种飞升各自需要的稀有材料来源 | 中 | 熔心会（艾拉拉）、海伦娜（若玩家帮） |
| **第四册：飞升见证记录** | Lockhart对63年双飞升的旁观记录和分析 | 中 | 多方感兴趣的暗标——最终价未公开 |

### 可能结果（取决于玩家介入）

| 玩家行动 | 结果 |
|:---------|:-----|
| 帮索恩 → 索恩获得跨性相理论原版，埃德蒙获删节抄录 | 蛾之教会掌控核心理论 |
| 帮埃德蒙 → 埃德蒙/玩家获得跨性相理论原版 | 灯之教会（个人）掌握核心理论 |
| 帮海伦娜 → 仪式要件分册的铸部分被海伦娜抄录 | 铸容器完整配方可用 |
| 帮西尔维娅 → 西尔维娅在拍卖中现身被莫蒂默认出 | 银棺修会确认她活着，追猎升级 |
| 不介入 → 默认：索恩获得第二册，莫蒂默获得第一册，艾拉拉获得第三册 | 势力间脆弱平衡 |
| 维克多成功偷拍 → 第二册在混乱中被盗 | 各方追猎维克多 |

### 拍卖后NPC状态（Day 7结束）

| NPC | 状态变化 |
|:----|:--------|
| 索恩 | 获得跨性相理论（或失去），情报优势增加 |
| 维克多 | 视玩家选择：仍在蛾之教会 / 叛逃中 / 已被发现 |
| 莫蒂默 | 获得冬之论述，银棺修会对西尔维娅的追猎升级 |
| 艾拉拉 | 获得/失去仪式要件铸部分，与海伦娜关系变化 |
| 西尔维娅 | 出现（或未出现）。若出现，银棺修会开始主动追猎 |
| 中间人 | 宣布下一场拍卖——三日后，仅拍跨性相理论（如果首次未成交） |

### 法官操作
- Day 6晚间：公开通告 *「地下暗流涌动——多方势力正在聚集。」*
- Day 7日切：根据玩家在拍卖相关节点中的选择，逐个裁定各玩家在拍卖中的结果
- Day 7日切后：更新所有受影响NPC的状态。发布世界通告：
  > *「玛莉金巷地下拍卖于昨夜举行。四册神秘的冬之密传手稿分册已落入不同买家手中。拍卖结束后，至少两起未遂劫案被目击。」*

## 事件三：征兆之夜（Day 14-17）

### 触发
自动触发（Day 14-17间某夜——确切日期由法官根据"哪个夜晚能让最多NPC和玩家同时被影响"来选择）。现实对应：一个罕见的天体排列。

### 事件描述
罕见的星体排列导致全城跨性相现象同时爆发。每个性相的显现被大幅度增强——不只是求道者能感受到，普通人也能。

### 跨性相现象

| 性相 | 现象 | 影响范围 |
|:----:|:------|:--------|
| **灯** | 全城灯焰在午夜同时脉动三次——包括熄灭的路灯。圣吉尔斯彩色玻璃在无光的情况下自发发光一小时 | ③④⑤ |
| **蛾** | 全城蛾群在布市②上空聚集成几何图案——持续七分钟。所有被蛾之教会监视的人同夜做了同一个巢穴梦 | ②⑧⑩ |
| **刃** | 全城刀剑、军械同时发出可听见的嗡鸣。黑铁俱乐部沙袋在没有外力的情况下晃动 | ⑨⑩⑦ |
| **铸** | 全城金属物体表面出现短暂的霜花纹路——随后消失。海伦娜的炉火在一分钟内烧出所有已知颜色 | ⑧⑭ |
| **冬** | 灰衣修士墓园中十三具十年内下葬的遗体同时出现抽搐——持续约两分钟。惠特菲尔德在地下室记录 | ⑪ |
| **心** | ⑬欢乐之地所有在场者的心跳同步了整整三分钟——任何人只要在音乐学院三十步以内都能感受到同一个脉搏 | ⑬⑱ |

### NPC反应表

| NPC | 反应 |
|:----|:------|
| 埃德蒙 | 连夜记录所有现象——这是跨性相理论的活证据 |
| 索恩 | 意识到蛾之教会无法解释这个现象——首次对自己的情报垄断产生动摇 |
| 莫伊拉 | 确认了心之道的力量。给蒙特罗斯写信："你儿子不在这些心跳中。" |
| 马库斯 | 沙袋无风自动时他正在后屋——他蹲下来盯着它看了很久。然后笑了 |
| 西尔维娅 | 在墓园亲眼看到三具尸体抽搐——她认出其中一个墓穴是洛克哈特的 |
| 伦威克 | 接到17份来自市民的异常报告——他正式向上级申请立案调查 |
| 丹比议员 | 在市政厅紧急会议中提议"全面清查超自然活动" |
| 蒙特罗斯 | 确信征兆之夜是他儿子"在那边"的信号——追加全部家产到寻找手稿上 |

### 法官操作
- 征兆之夜当天：世界通告中不发预先警告
- 征兆之夜日切后：发布一条长世界通告描述各种现象。每个玩家私信中描述其所在区域具体的现象
- 如果玩家在室外活动：可能直接目击对应区域的性相现象
- 此事件后：丹比的清剿提案加速。NPC对跨性相飞升的"真实性"不再怀疑

## 事件四：清剿威胁（Day 20-25 · 条件触发）

### 触发条件
**以下任一条件满足即触发：**
1. 丹比议员在Day 20前获得了足够证据（取决于玩家是否破坏他的调查）
2. 蛾之教会主动向市政厅提供了目标名单（取决于索恩的策略——他可能在拍卖失利后选择"清理场外对手"）
3. 某次公开的超自然事件被普通市民大规模目击（如征兆之夜影响被市民广泛谈论）

**如果三个条件都未满足** → 清剿不触发。丹比的提案在市政厅被驳回。

### 事件描述
城防军对已知的超自然活动据点发动协同突袭。行动代号"净灯"——名义上是清查无证经营和非法集会。

### 突袭目标（6-8处）

| 目标 | 受影响NPC | 风险 |
|:-----|:---------|:-----|
| 黑铁俱乐部⑨ | 马库斯、克兰中士 | 被搜查——断刃修会的刃之密传可能被发现 |
| 熔炉与月光工坊⑧ | 海伦娜 | 炉火和仪式材料可能被没收 |
| 蛾之教会香料铺② | 索恩、米拉贝尔 | 索恩预先得到了风声——他转移了关键文件 |
| 音乐学院⑬ | 莫伊拉、皮普、艾琳 | 搜查——但莫伊拉提前三小时接到匿名警告 |
| 地下拱顶实验室⑭ | 艾拉拉 | 实验室入口被发现——艾拉拉启动熔炉自毁装置 |
| 银棺修会殡仪馆⑪ | 惠特菲尔德、莫蒂默 | 搜查——但殡仪馆有合法的丧葬执照 |
| 灰衣修士墓园工具间⑪ | 西尔维娅（若她在） | 她的躲藏地点暴露 |
| 圣吉尔斯图书馆③ | 埃德蒙、爱丽丝 | 闭架区被要求开放检查 |

### NPC预警来源

| NPC | 预警来源 | 提前时间 |
|:----|:--------|:--------|
| 索恩 | 自己的情报网——蛾之教会渗透了市政厅 | 24h——已转移 |
| 莫伊拉 | 匿名警告——实际上是科迪莉亚·阿什比嬷嬷发给她的 | 3h |
| 马库斯 | 伦威克（他是马库斯怀疑的对象，但他不想看到城防军代替修会执行正义） | 1h |
| 海伦娜 | 格里格——他从市政厅书记员邓恩那里买到了突袭名单 | 2h |
| 艾拉拉 | 没收到——她在实验室里，不接地面信息 | 0h |
| 西尔维娅 | 惠特菲尔德——在突袭队出发前十分钟通知了她 | 10min |

### 可能结果

| 结果 | 条件 |
|:------|:-----|
| 突袭成功——多处被查 | 多数NPC未获预警 / 玩家未介入 |
| 突袭被提前泄露——各势力转移 | 玩家通过格里格/伦威克/索恩获取并散发了突袭名单 |
| 突袭引发暴力对抗 | 某NPC选择抵抗（马库斯/艾拉拉最可能） |
| 突袭取消 | 玩家破坏了丹比的证据基础 / 蒙特罗斯用政治影响力撤销了提案 |

### 法官操作
- Day 19：如果清剿条件满足，世界通告中隐约暗示 *「市政厅近期加强了对无证集会场所的巡查。」*
- Day 20：日切后发布全城通告描述突袭结果
- 每个受影响NPC的锁定玩家：在私信中告知突袭对其NPC的影响
- 如果某个NPC在突袭中死亡/被捕 → 🔓锁释放

## 事件五：飞升窗口（Day 26-30）

### 触发
自动触发——当手稿知识被至少一个势力/玩家获得后，飞升成为可能。

### 事件描述
洛克哈特的手稿——无论分册还是完整版——已进入爱丁堡至少三周。获得知识的NPC（或玩家）现在有了尝试飞升的理论基础。窗口期取决于天体条件——正如征兆之夜预示的，最佳飞升时机在Day 26-30之间。

### 谁可能尝试飞升（取决于游戏状态）

| 候选人 | 飞升类型 | 条件 | 成功概率 |
|:------|:--------|:-----|:--------|
| **西尔维娅** | 冬之飞升 | 获得完整冬之论述+跨性相理论+仪式要件+玩家协助 | 40% |
| **艾拉拉** | 铸之飞升 | 获得容器+仪式要件 | 35% |
| **维克多** | 蛾之飞升 | 获得跨性相理论（蛾+灯结合）+玩家协助 | 25% |
| **马库斯** | 刃之飞升 | 他在终局节点中尝试（独立于手稿） | 50%（已有剧情线） |
| **玩家** | 任意 | 玩家自行满足飞升条件 | 取决于玩家准备 |
| **无人尝试** | — | 窗口期过去，天体排列消散 | 飞升窗口关闭——下一次在数年之后 |

### NPC对飞升窗口的反应

| NPC | 行动 |
|:----|:-----|
| 索恩 | 如果蛾之教会没有掌握飞升关键信息——他会在飞升窗口期试图阻止任何人飞升。"该让人知道的才让人知道。" |
| 莫伊拉 | 试图阻止冬之飞升——如果她认为冬飞升会熄灭心之道。但如果西尔维娅证明跨性相兼容——莫伊拉可能转而支持 |
| 莫蒂默·阿什 | 如果银棺修会掌握了手稿——他会在窗口期试图执行银棺修会自己的飞升。"教义的正统飞升者" |
| 埃德蒙 | 不做尝试——但记录一切。如果跨性相飞升成功，他将发表完整的学术报告 |
| 伦威克 | 如果他在此之前已相信超自然存在——他会试图在飞升仪式现场建立城防军封锁线，控制"后果" |
| 蒙特罗斯 | 如果他的儿子"复活"无望，他可能在最后时刻试图闯入仪式场——"我再试最后一次" |

### 飞升成功/失败的全局影响

| 结果 | 全局影响 |
|:------|:--------|
| **成功（任一NPC或玩家）** | 世界通告：*「今夜，爱丁堡上空出现了一道不属于任何已知光源的光——持续了整整一分钟。在那一分钟里，所有灯焰都变成了金色。六个教团的高层同时沉默。有人穿过去了。」* 跨性相飞升的存在被证实。所有教团的教义根基动摇。 |
| **失败（任一尝试者死亡）** | 世界通告：*「[区域]发生了一起无法解释的能量释放事件。附近居民报告了短暂的耳鸣和视觉异常。城防军封锁了事发区域。」* 反噬效果可能影响邻近区域的NPC和玩家。 |
| **无人尝试** | 窗口期平静度过。飞升的秘密继续封存。各势力回到日常——但所有人都知道，下一次窗口在数年之后。而手稿还在某人手中。 |

### 法官操作
- Day 25：向已满足飞升条件的玩家/NPC推进终局节点
- Day 26-30：每日日切后检查飞升尝试状态
- 如果有人尝试：裁定飞升结果（骰子 + 玩家准备 + NPC协助）
- Day 30日切：飞升窗口关闭。发布最终世界通告，描述这场游戏的结局

## 世界通告模板

### 邀请函（Day 1-3）
> *「一股不安的气氛正在爱丁堡的地下蔓延。据传一件被尘封了十年的冬之密传手稿即将易主——至于落入谁手，全城的暗处都在赌。」*

### 拍卖后（Day 7）
> *「昨夜，在城中某个未在地图上标注的角落，一场地下拍卖落下帷幕。四册手稿分册各归其主。至少两名竞拍者离开后没有安全回到住所。城防军今晨在牛门发现了一片被冷光灼烧过的石面。」*

### 征兆之夜（Day 14-17）
> *「昨夜，爱丁堡经历了一场无法解释的全城现象。灯焰在午夜同时脉动、圣吉尔斯教堂的彩色玻璃在没有月光的情况下发光、城南墓园传出了多起无法解释的响动。城防军接到了十七份市民报告。市政厅今晨召开了紧急闭门会议。」*

### 清剿（Day 20-25，若触发）
> *「今晨，城防军对城中多处场所进行了突击搜查。官方声明称这是对'无证经营和非法集会'的例行清查。至少四处场所被查封。三名市民在冲突中受伤。霍勒斯·丹比议员在市政厅发表了'净化爱丁堡'的演讲。」*

### 飞升（Day 26-30，若有人尝试并成功）
> *「今夜，[区域]上空出现了一道短暂的光——目击者描述其为'不属于任何已知光源的颜色'。全城的灯焰在那一分钟内改变了颜色。当灯光恢复时，[区域]的地面上留下了一个烧灼的符号——它的形状不属于任何已知的性相体系。六个教团的高层同时保持了沉默。有人穿过去了。」*

### 游戏结束（Day 30）
> *「唐纳73年的冬天将近。天体排列已过。下一次飞升窗口——据洛克哈特手稿的推算——在数年之后。手稿的各分册仍然分散在爱丁堡的某个角落。有些秘密，还未到被揭开的时候。」*
