# 核心设计原则

## 这不是一款社交推理游戏

玩家之间**不存在自由交流渠道**。没有群聊闲谈，没有私聊社交。

所有玩家→玩家的互动行为，都在法官的视野内。传递消息本身消耗玩家宝贵的行动资源。

## 为什么要这样做

根据上一把游戏的经验，如果玩家之间可以自由交流，天然就会：

- 形成小圈子
- 抱团取暖
- PY 交易（情报互换、资源互换、行动协同）
- 排挤独狼玩家

这背离了大逃杀的初衷，也对独狼玩家极不友好。

独狼玩家在这个游戏中**应该有生存空间**，甚至在某些方面拥有隐性优势。

## 这个约束如何落地

| 原问题 | 解决方案 |
|--------|---------|
| 玩家随意聊天、拉帮结派 | **无群聊/私聊渠道**。所有玩家间沟通必须通过 `传讯` 行动 |
| PY 交易 | 每笔联络都消耗行动位——花行动联络别人，就少一个行动推进自己。机会成本很高 |
| 抱团取暖 | 你无法确定对方是否可信。消息可能被拦截，对方可能背叛。信任是高风险的 |
| 独狼劣势 | 独狼不需要花费行动去社交/传讯，把所有行动都用于推进自己的路线——**这是隐性的优势** |
| 信息垄断 | `潜伏` 让独狼可以观测他人的行动而不被发现。信息差可以靠自己获取，不必依赖线人 |
| 大联盟碾压 | `攻击` 和 `仪式` 都会产生泄露/暴露。联盟的集体活动会留下大量痕迹，成为全图靶子 |

## 核心经济逻辑

> **⚠️ V1 — 按轮模型**
>
> 以下为当前 V1 设计，按"每轮 3 行动位"计算。
> V2 方向：[结算不再等所有人](./灵感/结算不再等所有人.md) 将改为按天推进、流式结算。
> 届时经济模型会从"每轮 3 行动"调整为"每天行动次数上限"或"精力值"系统，但社交 vs 独狼的生态位平衡原则不变。

```
每轮 3 行动位  (V1)
每天 N 行动上限  (V2 — 待定)

V1 社交型玩家：传讯(1) + 社交(1) + 推进(1) = 每轮只推进 1 次
V1 独狼型玩家：推进(1) + 推进(1) + 潜伏(1) = 每轮推进 2 次 + 获取情报

→ 独狼天然有进度优势，社交型有信息/协作优势
→ 两者在生态位上平衡，而不是社交型碾压独狼型
```

> **V2 下该平衡是否仍然成立？**
>
> 如果改为按天结算且行动次数上限提升，社交型玩家"用行动换信息"的机会成本降低，独狼的隐性优势可能被稀释。
> 需要在 V2 设计时重新审视行动上限数值或引入精力值等约束来维持该平衡。

## 沟通渠道架构

**群聊的用途（仅限法官发布）：**

- 世界状态更新
- 公共事件通告
- 公共区域能观察到的现象（所有人都能看到的）
- 游戏开始/结束宣告

**私聊的用途（仅限玩家→法官）：**

- 提交行动
- 提交 OOC 疑问

**玩家→玩家：无直接聊天渠道。** 必须通过游戏行动 `传讯`。
