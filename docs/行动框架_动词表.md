# 行动框架 V1.1：动词表（精简版）

## 核心设计

**动词 + 目标 + 可选方式 = 裁定**

标准化行动格式：

```
[行动动词] [目标] [可选:方式/条件]
```

## 动词总览

| 动词 | 类别 | 状态 | 一句话描述 | 等级 |
|------|------|:----:|-----------|:----:|
| 探索 | 探索 | ✅ | 扫描当前区域，获取线索（物品、路径、情报） | L1 |
| 移动 | 行动 | ✅ | 沿已发现路径前往相邻区域 | L1 |
| 社交 | 行动 | ✅ | 与 NPC 互动（交谈/交易/说服等） | L2 |
| 攻击 | 冲突 | 🔧 | 直接对抗（设计中） | L3 |
| 仪式 | 神秘 | ✅ | 推进飞升的关键行动 | L3 |

### 搁置 / 移除

| 动词 | 状态 | 说明 |
|------|:----:|------|

| 潜伏 | ⏸️ 搁置 | 后续版本再考虑 |
| 研究 | ⏸️ 搁置 | 后续版本再考虑 |

## 性相系统（框架）

引入六种性相（Aspects）作为玩家的核心成长维度：

| 性相 | 象征 |
|:----:|------|
| 灯 | 洞察、知识、揭示 |
| 蛾 | 变化、诱惑、隐秘 |
| 刃 | 冲突、毁灭、锋锐 |
| 铸 | 锻造、转化、创造 |
| 冬 | 寂灭、记忆、坚韧 |
| 心 | 生命、激情、守护 |

**核心规则：**

- 每项性相等级达到 **2 / 6 / 10 / 15** 时，分别解锁一个对应技能（每性相 4 技能）
- 性相等级是**硬门槛**——不到等级无法解锁或使用该技能
- 性相等级影响探索判定（骰值修正或线索产出量）
- 仪式是提升性相等级的主要途径之一
- 技能具体设计——后续阶段展开

## 动词详细定义

### ① 探索 (Explore) — L1

了解当前所在区域，获取线索。

**线索类型：**
- **路径**：发现通往相邻区域的通道（解锁移动目标）
- **物品**：发现可拾取的特殊物品——发现时自动获知其基本用途和名称（不消耗额外行动）
- **情报**：区域事件痕迹、NPC 动向、异常现象等

**骰值与性相的关联：**

```
基础判定 = 1d10 + 探索者相关性相等级修正
难度 = 由区域特征 + 目标隐蔽程度决定

成功 → 获得主要线索（1-2 条）
大成功 → 额外获得一条隐藏线索
失败 → 获得模糊信息或无收获
大失败 → 触发负面事件（暴露位置、触发陷阱等）
```

玩家可以指定搜索方向来影响线索类型：
> `探索 废弃教堂` → 默认扫描全貌
> `探索 废弃教堂 方式:寻找暗门` → 偏重路径类线索
> `探索 废弃教堂 方式:搜索圣坛` → 偏重物品类线索

**示例：**

> 玩家：`探索 废弃教堂`
>
> 裁定：你穿过坍塌的围栏进入教堂中殿。彩色玻璃已经破碎，但东侧的圣坛完好无损。圣坛上有一本被灰尘覆盖的皮革书——《教区记录》。你随手翻了几页，注意到其中一些用暗语写成的注释。出口：东侧墓园、北侧地下墓穴入口。
>
> *（玩家同时获得物品「教区记录」的基本信息：一本19世纪末的教会日志，边缘有神秘暗语注释）*

### ② 移动 (Move) — L1

前往一个已发现路径连接的相邻区域。

**规则：**
- 必须已通过「探索」发现了通往目标的路径
- 某些区域需要特殊条件才能进入（特定性相等级、道具、知识）
- 移动过程中可能触发随机事件

> `移动 东区-废弃教堂`

---

### ③ 社交 (Socialize) — L2

与一个已知 NPC 互动。**不能以此行动与其他玩家互动。**

**可选方式：**

| 方式 | 说明 |
|------|------|
| 交谈 | 获取信息、闲聊、套话 |
| 说服 | 让 NPC 做某事 |
| 威胁 | 强迫 NPC 配合 |
| 交易 | 以物易物或情报交换 |
| 招募 | 尝试将 NPC 纳入麾下 |

**设计约束：** NPC 的反应以预设模板 + 性相/身份修正为主，不做自由对话式的复杂交互——降低设计压力的同时保留策略深度。

> `社交 酒保 方式:交谈 关于东区最近的黑帮活动`
>
> `社交 老收藏家 方式:交易 用蛇纹青铜钥匙交换地下墓穴的地图`

### ⑤ 仪式 (Ritual) — L3

推进飞升的核心行动。

**前置条件：** 特定知识 + 特定材料 + 特定地点 + 时机

**仪式成功的结果：**
- 提升某项性相的等级
- 获得特殊物品
- 召唤某种存在 / 开启新路径

**平衡机制：**
- 仪式无论如何都会产生**泄露（Leakage）**——世界事件通告向所有玩家广播异常
- 失败可能产生严重负面后果（受伤、精神污染、吸引注意）

> `仪式 破晓洞察 材料:水晶头骨 地点:废弃教堂圣坛 知识:《光之阶梯》第一章`

## 行动提交格式

```
第X轮 行动

行动1: [动词] [目标] [可选:方式/条件]
行动2: [动词] [目标] [可选:方式/条件]
行动3: [动词] [目标] [可选:方式/条件]
附言: [OOC 问题或备注，可选]
```

**规则（V1 — 按轮）：**
- 每轮每名玩家最多 **3 个行动**
- 其中最多 **1 个 L3 行动**（仪式或攻击）
- 行动按顺序执行（如果行动1导致死亡或失去行动能力，行动2/3 作废）
- 截止时间由法官指定

> **⚠️ V2 方向：按天结算**
>
> 灵感 [结算不再等所有人](./灵感/结算不再等所有人.md) 提出将"按轮"改为"按天"推进，玩家每天可多次提交行动，法官即时结算。
> 届时本表的"每轮 3 行动"将调整为"每天行动次数上限"或"精力值系统"。
> 详细设计待 V2 阶段展开。

## 信息差来源（V1.1）

玩家间信息不对称通过以下渠道自然产生：

| 来源 | 说明 |
|------|------|
| 探索收获 | 不同区域有不同线索，各自的探索方向和结果不同 |
| NPC 交互 | 不同 NPC 掌握不同情报，且对玩家的态度因身份/性相而异 |
| 世界剧情 | 区域事件和势力推演动态发生，玩家不在场就看不到 |
| 性相/技能 | 部分技能可能提供独特的感知或解读能力（后续展开） |
