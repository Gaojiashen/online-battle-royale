# 裁定流程 V1：法官工作流

> **⚠️ 即将引入 V2 思路：流式结算**
>
> 本 V1 版本采用经典的"批量收集-统一处理"模式。
> 灵感 [结算不再等所有人](./灵感/结算不再等所有人.md) 提出了另一种模式——**按天推进，即时结算**。
>
> 两者并非互斥：V1 适合手动/低频率运营阶段；V2 流式结算适合自动化程度更高、法官人力充足或系统辅助完善的阶段。
> 当前文档保留 V1 作为 baseline，V2 设计将在后续版本补充。

## 三大阶段

```
[阶段一] 收集与分类 ──→ [阶段二] 处理与裁定 ──→ [阶段三] 反馈与世界推进
```

---

## 阶段一：收集与分类（预计耗时：5-10min）

**步骤：**

1. 收集所有玩家在私聊中提交的行动
2. 按**区域**分组——同区域的事件需要处理先后顺序和互动
3. 标注行动等级（L1 / L2 / L3）
4. 检查依赖性冲突（A 攻击 B、B 在另一区域 → A 的攻击无效或需要修正）

**产出物：一张裁定工作表**

```
第3轮 裁定工作表
──────────────
【东区-废弃教堂】
  P001: 调查 圣坛暗格 (L2)
  P003: 潜伏 废弃教堂 (L1)  ← 潜伏可能观察到P001

【东区-码头】
  P002: 攻击 P005 伏击 (L3)
  P005: 交易 老渔民 (L2)    ← 注意：与P002攻击冲突

【南区-报社】
  P004: 研究 神秘信件 (L2)
──────────────
```

## 阶段二：处理与裁定（预计耗时：30-60min）

### 处理顺序规则

1. **同区域：** `潜伏` → `移动` → `探索/调查/社交/研究/传讯` → `仪式` → `攻击`
2. **跨区域：** 先处理所有 `移动`，再按新区域分组
3. **冲突行动（A攻击B）：** 按顺序执行，先到的先处理

### 逐行动：四步裁定法

#### Step 1 — 合法性检查（Yes/No）

- 玩家是否在正确的区域？
- 目标是否存在 / 是否可达？
- 拥有的资源和前置条件是否满足？
- 该行动位是否还可用？（未因之前的行动结果而失效）

> 有任一否 → 行动失败，告知原因，不消耗资源。

#### Step 2 — 难度评估（数值化）

```
基础难度（由目标决定）
  + 环境修正（时间、天气、区域危险度）
  - 身份优势（记者调查线索 +1）
  - 道具加成（使用古董放大镜 +1）
  - 知识加成（已掌握相关情报 +1）
  = 最终难度值
```

#### Step 3 — 判定（骰子）

```
1d10 + 修正值 ≥ 最终难度 → 成功
否则 → 失败（可能部分成功 / 大失败）
```

#### Step 4 — 结果记录

- 写出 1-3 句结果叙事
- 更新世界状态（玩家状态、物品、区域状态、NPC 状态）
- 如果涉及秘密/隐藏信息，写在玩家私密反馈中（不公开）

### 各行动等级参考处理时间

| 等级 | 处理时间 | 说明 |
|:----:|:--------:|------|
| L1 | 20-30s/条 | 探索、移动、潜伏、传讯。规则直接判定，简写结果 |
| L2 | 1-2min/条 | 调查、社交、研究。规则 + 少量自由裁定 |
| L3 | 3-5min/条 | 攻击、仪式。需要详细裁定和叙事 |

## 阶段三：反馈与世界推进（预计耗时：10-15min）

### ① 私密反馈（私聊发给各玩家）

反馈格式示例：

```
第3轮 结果
────────
你的行动结果：
行动1 [调查 圣坛暗格] → 成功。你在圣坛背面的砖缝中发现了一个暗格，里面有一枚青铜钥匙，上面刻着蛇形纹路。
行动2 [研究 暗语注释] → 部分成功。你破译了大部分内容，但有一段关于"地下的眼睛"的指代你暂时无法理解。
行动3 [传讯 玩家B] → 消息已发出。信童已出发，预计下一轮初抵达。

世界状态变化：
- 你感觉这个区域还有其他人在活动（潜伏判定：你注意到附近有轻微的脚步声）

泄露值：0 → 无变化
```

### ② 公共通告（群聊发布）

仅发布所有人都能观察到的事件：

```
【第3轮 世界事件通告】
────────
东区：废弃教堂方向在午夜时分出现了一次短暂的奇异光芒。附近居民有所察觉。
东区：码头发生了一起暴力事件——具体细节仍在调查。
南区：无异常事件报告。

下一轮截止时间：明晚 22:00
请于截止前提交第4轮的行动。
```

### ③ 世界状态更新（法官内部记录）

```
【世界状态变更】（仅法官维护）
P001: 获得物品「蛇纹青铜钥匙」
P001: 飞升线索 +1 (知识之道相关)
P002: 重伤状态 (HP 2/8)
P005: 死亡/失去意识（取决于裁定）
东区-废弃教堂: 圣坛暗格已空
NPC-老渔民: 目睹码头事件，畏惧状态
```

### ④ 势力推演（简洁版）

使用骰子判定势力本轮自动行动：

```
本轮势力自动行动（掷骰判定）：
鱼头帮 (扩张判定 1d6=4 → 财富+1)
血手帮 (冲突判定 1d6=2 → 无行动)
警察局 (事件响应判定 1d6=5 → 开始调查码头事件)
```

## 附：效率优化要点

| 策略 | 说明 |
|------|------|
| 模板库 | 为常用行动准备结果模板，法官填空即可 |
| 批量处理 | 固定截止时间（如每晚 22:00），一次性处理所有行动 |
| 行动分流 | L1 快速过表，L2 适度发挥，L3 重点裁定 |
| 结构化记录 | 所有状态变更使用固定格式（JSON/YAML 思路），为未来自动化准备 |
