# NPC 剧情线作为玩家入口

## 想法

对玩家来说，最直观的参与方式就是**参与剧情**。

玩家出生后遇到不同的 NPC，自然进入不同的剧情线。每一条剧情线由一个核心 NPC 驱动，NPC 有自己的欲望、秘密、日程和关系网。

## NPC 与剧情线的映射关系

```
┌─────────────────────────────────────────────┐
│  一个 NPC = 一条剧情线                      │
│                                             │
│  不同剧情线可能在节点上交叉：               │
│  NPC-A 线的重要线索，藏在 NPC-B 的剧情线里  │
│  NPC-A 和 NPC-B 都想争夺同一件物品          │
│  NPC-A 的秘密一旦暴露，会影响 NPC-B 的命运  │
└─────────────────────────────────────────────┘
```

## 设计思路

- 玩家不需要读规则书来理解"飞升"——只要跟着感兴趣的 NPC 走，自然接触到更深层的秘密
- 多条剧情线为玩家提供了**方向感**："我在追这条线"，而不是"我该干什么？"
- 剧情线之间的交叉自然产生竞争——两个玩家各自跟不同 NPC 互动，却因为两条线交汇而撞上
- NPC 的欲望是动态的，剧情线不是固定剧本，而是由 NPC 的目标驱动的——如果 NPC 的目标被玩家改变了，剧情线转向
- 这甚至可能承载飞升路径——最终的飞升方式，隐藏在某条剧情线的终点

## 需要进一步讨论的问题

- 身份与剧情线的关系（身份影响你能接触到哪些 NPC？还是所有玩家都能接触所有 NPC？）
- 如果 NPC 被杀了/剧情线断了怎么办
- 剧情线是"有限条"还是"动态生成"的
